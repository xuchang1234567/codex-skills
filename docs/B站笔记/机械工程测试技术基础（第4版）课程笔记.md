## 学完你应该获得什么

- 理解“测量、试验、测试、测试技术”的区别与联系。
- 能从测试系统、被测对象和结果评价三个层次理解工程测试。
- 掌握信号的时域、频域描述，以及周期信号与非周期信号的频谱区别。
- 理解傅里叶级数、傅里叶变换、频谱密度和窗函数的基本用途。
- 能把一个测试问题拆成测量对象、信号、系统、采样和结果解释几个环节。

## 一句话总论

机械工程测试不是“读出一个数”，而是围绕被测量建立测量与试验系统，用合适的信号表示和分析方法获得可解释、可验证的工程信息。[1][2]

## 第一章：从测量到测试技术

### 1. 测量、试验与测试

测量更具体，关注某个对象或某个物理量的数值获取，例如工作台位置、汽车动力参数。试验则强调为验证性能而搭建方法、系统和工况，通过有组织的过程得到结果。测试是测量与试验技术的结合：不仅要“得到数值”，还要设计如何得到、在什么条件下得到，以及如何判断结果是否满足要求。[2][3]

数控机床的闭环控制是典型例子。编码器或其他检测装置获得工作台位置，控制器把反馈值与目标值比较，再修正伺服电机输入。传感器装在最终工作部件上属于闭环，装在电机或丝杠等中间位置则属于半闭环。区别不在于有没有传感器，而在反馈是否真正代表最终输出。[4]

汽车动力性能试验则更完整：先建立模拟路况的试验台，再采集通过过程中的动力响应，最后分析峰值、波谷和波动是否满足法规或设计标准。这里结果来自“系统化试验”，不是孤立的一次测量。[5]

### 2. 测试系统的基本思路

建立测试问题时至少要明确：

1. **被测对象**：结构、设备、过程或材料。
2. **被测量**：位移、速度、加速度、力、温度、压力或某种性能指标。
3. **测量链路**：传感器、信号调理、采集、记录和分析环节。
4. **工况与参考**：输入如何施加、输出与什么比较、是否需要反馈。
5. **判据**：结果用于控制、验收、诊断、设计优化还是科学解释。

测试技术的难点往往不是某个仪器的读数，而是测量链是否真正代表被测对象。传感器位置、安装方式、测量带宽和反馈点的选择，都可能改变结论。

## 第二章：信号及其描述

### 1. 为什么要同时看时域和频域

时域回答“信号随时间怎么变化”，适合观察冲击、衰减、峰值和瞬态过程；频域回答“信号由哪些频率成分组成”，适合识别周期、共振、谐波和噪声。工程测试通常需要两种视角互相校验，而不是把频谱当成时域波形的另一种画法。[6]

### 2. 周期信号与离散谱

周期信号可表示为基频及其整数倍谐波的叠加。傅里叶级数把信号写成离散频率分量，每条谱线对应一个确定的频率点及其幅值/相位。周期越短，基频间隔越大；周期越长，谱线越密。[7]

这里要区分“频率点上的幅值”和“单位频宽内的谱密度”。周期信号的傅里叶级数系数有明确的离散频率点含义；非周期信号的连续频谱则需要用频率区间来理解，不能把连续谱函数直接当成离散谱线幅值。[8]

### 3. 非周期信号与傅里叶变换

可以把非周期信号看作周期趋于无穷大的极限。此时基频间隔趋于零，原来的离散谱线变成连续谱。傅里叶变换把时域信号 `x(t)` 变成频率函数 `X(omega)`，逆变换再由频域恢复时域信号。工程上它是从时间波形进入频率分析的桥梁。[9]

角频率与普通频率的关系为 `omega = 2 pi f`。使用公式时必须统一变量和单位，否则频率轴、幅值和积分结果都会产生数量级错误。[10]

连续频谱函数的量纲也不同于离散傅里叶级数系数。连续谱函数更准确地理解为单位频宽内的幅值密度；只有在一个有限频带上积分，才得到该频带的总贡献。这个区别对频谱比较、能量估计和不同采样长度之间的结果解释很重要。[8][10]

### 4. 窗函数与有限观测

真实测试不可能观察无限长时间，只能截取一段记录。把有限观测等价为原信号与窗函数相乘，时域相乘会在频域产生卷积，从而使理想谱线扩展为主瓣和旁瓣。主瓣反映主要能量集中区域，旁瓣则可能造成邻近频率泄漏。[11]

因此窗函数不是“让波形好看”的装饰，而是有限观测的数学模型。选择窗函数时要在频率分辨率、幅值准确性和旁瓣抑制之间权衡：窄主瓣有利于分辨相近频率，低旁瓣有利于观察弱信号，但二者通常不能同时达到最优。后续做频谱分析时，应记录采样长度、窗类型、重叠方式和幅值修正方法。

## 可复用的测试工作流

1. 把工程目标改写成可测量的物理量和判定指标。
2. 选择反馈点或传感器安装点，确认它代表最终输出而不是中间环节。
3. 判断信号属于周期、瞬态还是随机过程，决定采用时域、频域或两者结合。
4. 设定采样频率、记录长度和抗混叠措施，保证目标频带被可靠覆盖。
5. 对有限记录选择窗函数，并保留单位、幅值标定和频率分辨率信息。
6. 用时域峰值、均方根、衰减趋势与频域峰值、谐波、带宽互相验证。
7. 将结果放回实际工况，检查测量链、传感器位置和判据是否支持最终结论。

## 常见误区

- **把测量当成测试。** 读数只是测试链的一环，缺少工况、方法和判据就不能称为完整测试。
- **只看频谱峰值。** 频谱峰值可能来自结构响应、传感器安装、窗函数泄漏或电噪声，必须结合时域和工况解释。
- **忽略反馈位置。** 中间位置的反馈不能自动代表最终执行部件的真实运动。
- **混淆离散谱和连续谱。** 连续频谱是单位频宽意义下的密度，不能直接按离散谱线幅值读取。
- **忘记有限观测效应。** 不加窗或随意加窗都会影响泄漏、幅值和频率分辨率。

## 自测题

1. 测量与试验的核心区别是什么？答：测量强调具体量的获取，试验强调为验证目标而组织系统、方法和工况。
2. 为什么最终反馈点通常比电机端反馈更能形成闭环？答：它直接反映最终工作部件的输出。
3. 周期信号的频谱为什么是离散的？答：周期信号由基频及整数倍谐波组成。
4. 非周期信号为什么对应连续谱？答：把周期趋于无穷大后，基频间隔趋于零。
5. `omega` 与 `f` 的关系是什么？答：`omega = 2 pi f`。
6. 窗函数解决什么问题？答：描述有限时间观测，并控制频谱泄漏与分辨率之间的权衡。

## 课程级知识框架

### A. 测试问题的四层结构

任何机械工程测试都可以拆成四层：

1. **测量原理**：被测物理量如何转换成可观测信号，例如位移、速度、加速度或力如何产生相应的电量变化。
2. **测量方法**：采用直接还是间接测量，静态还是动态测量，接触还是非接触测量，模拟还是数字测量。
3. **测试系统**：传感器、调理电路、采集设备、显示记录和分析软件如何连接。
4. **数据处理**：对信号进行滤波、放大、采样、频谱分析、相关分析或其他运算，并给出工程结论。[12]

这四层不能互相替代。传感器选得再好，如果测量原理不适合被测量，结果仍然不可信；信号采集完整，如果没有明确判据，数据也不能自动变成结论。

### B. 测试系统的信号链

典型链路为：

`被测对象 -> 传感器 -> 信号调理 -> 采样/记录 -> 信号处理 -> 显示与判断`

传感器的作用有两层：首先获取被测对象的物理变化，其次把不便处理的非电量转换成电量。传感器输出往往很微弱，调理环节负责放大、滤波、整流、隔离和匹配；处理环节再把模拟信号数字化，计算频谱、功率谱、相关函数或统计指标。[13]

### C. 测量方法的选择矩阵

| 判断维度 | 常见选项 | 选择依据 |
|---|---|---|
| 是否直接得到目标量 | 直接 / 间接 | 间接测量需要可靠的模型或换算关系 |
| 变化速度 | 静态 / 动态 | 动态测试必须考虑带宽、响应和相位 |
| 信号形态 | 模拟 / 数字 | 数字系统需考虑采样、量化和混叠 |
| 传感方式 | 接触 / 非接触 | 接触可能改变被测对象，非接触可能受距离和环境影响 |
| 参考关系 | 绝对 / 比较 | 比较测量依赖标准量或校准基准 |

静态测量关注不随时间变化或变化很慢的量；动态测量关注随时间快速变化的过程，重点从“测准某一时刻的值”转向“保留变化规律”。选择方法之前，先估计信号的幅值范围、频率范围、变化速度和允许误差。[14]

## 信号分析方法卡

### 1. 周期信号

满足 `x(t)=x(t+T)` 的信号具有周期 `T`，基频为 `f0=1/T`，角频率为 `omega0=2 pi/T`。周期信号可以分解为直流分量、基频和整数倍谐波。频谱表现为离散谱线，谱线间隔由基频决定。测试中看到谐波峰值，常可追溯到周期激励、非线性、齿轮啮合、转子不平衡或结构边界。

### 2. 非周期信号

冲击、脉冲、衰减振动和有限持续时间事件通常不是严格周期信号。把周期趋于无穷大，离散谱线间距趋于零，便得到连续频谱。傅里叶变换定义为：

`X(omega)=integral[x(t) exp(-j omega t)]dt`

逆变换为：

`x(t)=(1/2 pi) integral[X(omega) exp(j omega t)]d omega`

工程解释上，`X(omega)`描述单位角频率宽度内的幅值与相位信息；不要把它和周期信号中某一根离散谱线的系数混为一谈。频谱幅值、功率谱密度和能量谱密度也应根据定义区分单位。[15]

### 3. 时域指标与频域指标

时域常用峰值、峰峰值、均值、均方根、方差、峭度、衰减率和过零率。频域常用主频、谐波幅值、带宽、谱峰、能量分布和功率谱。一个完整判断通常是：先用时域确认事件发生，再用频域判断事件由哪些频率成分构成，最后回到工况判断峰值是否具有物理意义。

### 4. 窗函数

有限时长记录相当于将无限信号乘以一个时间窗。时域相乘在频域表现为卷积，于是原本的谱线会展宽并出现旁瓣。主瓣越窄，频率分辨率通常越高；旁瓣越低，弱信号越不容易被强信号泄漏掩盖。没有“最好”的窗，只有与任务匹配的窗。

使用窗函数时应记录：采样频率、记录长度、窗类型、有效带宽、是否重叠平均和幅值修正。比较两次频谱时，如果窗、记录长度或归一化方式不同，峰值变化未必来自被测对象本身。

## 从工程问题到数据结论

### 步骤 1：定义问题

先写清“要测什么、为什么测、结果用于什么决策”。例如“测发动机振动”过于宽泛；应改成“测某转速区间内机座垂向加速度 RMS，并判断是否存在异常谐波”。问题越具体，传感器、采样率和分析方法越容易确定。

### 步骤 2：估计被测信号

根据结构、激励和运行工况估计幅值、频率、持续时间和可能的瞬态。没有先验时，可以先用低风险、宽带宽的预实验获得量级，再收窄正式测试参数。对动态信号尤其要避免传感器量程太小导致削顶，也要避免量程过大造成有效分辨率不足。

### 步骤 3：选择测量原理与方法

位移可通过直接位移传感器获得，也可由速度或加速度积分间接得到；力可由力传感器直接测量，也可由应变、刚度和几何关系间接估计。直接法链路短但不一定适合安装条件；间接法灵活但会引入模型误差。选择时要比较误差来源，而不是只比较仪器价格。[16]

### 步骤 4：设计测试系统

明确传感器安装位置、方向、固定方式、供电、信号调理、采集通道和同步时钟。传感器轴向必须与目标量一致；安装刚度、质量附加和电缆布置可能改变被测对象。对多通道测试，还要校核通道相位一致性和触发关系。

### 步骤 5：确定采样与记录参数

采样频率必须高于目标最高频率的两倍，并在实际工程中留出抗混叠滤波和过渡带余量。记录长度决定频率分辨率；记录太短，频率间隔过大，难以区分相近峰值；记录太长，则可能混入工况漂移。对非平稳过程，应采用分段、短时分析或同步平均，而不是把整个过程直接做一次傅里叶变换。

### 步骤 6：调理、采集与校准

信号调理通常包含放大、滤波、隔离、整流、偏置和阻抗匹配。调理器的截止频率、增益和噪声底要与传感器输出匹配。采集前用已知输入或标准源检查灵敏度、零点、线性度和通道一致性；采集后保存原始数据、校准系数、单位和测试工况，不能只保存一张处理后的图片。

### 步骤 7：分析与判断

先检查原始波形是否削顶、漂移、丢点或饱和，再进行去趋势、滤波、窗函数和频谱计算。频谱峰值需要与转速、激励频率、结构固有频率、采样设置和传感器安装位置对照。最后报告测量不确定度、重复性和结论适用范围。

## 测试系统的误差与可信度

测试结果不是被测量本身，而是被测量经过测量链后的估计。分析结果时至少要区分：被测对象自身波动；传感器灵敏度、零漂、迟滞、非线性与温漂；安装方向、松动、附加质量和接触刚度；调理与采集的增益、噪声、饱和、量化和通道同步；以及滤波、窗函数、平均和频率分辨率带来的算法误差。

一个可复核的测试报告应说明仪器型号和校准状态、传感器安装位置、采样参数、原始数据保存方式、处理步骤、重复测量结果和结论适用范围。只给出“峰值是多少”而没有测量链信息，无法判断结果是否可复现。

## 动态测试的系统观点

动态测试不能只问“测得准不准”，还要问系统是否能跟上输入。传感器和测试系统具有自身的频率响应：输入为某一频率的正弦信号时，输出可能发生幅值衰减、放大和相位滞后。若目标频率接近系统固有频率，输出可能共振；若目标频率超过带宽，信号会被衰减或失真。

因此应保证目标最高频率位于有效带宽内，传感器安装不会显著改变被测结构，调理器和采集卡带宽不低于测试需求，多通道系统的相位延迟已经校准。对冲击和瞬态信号，还要关注上升时间和过载恢复时间；对慢变化信号，零漂和低频噪声可能比高频带宽更重要。

## 采样、混叠与数字化

连续信号进入计算机前，需要经过采样和量化。若采样频率为 `fs`，目标最高频率为 `fmax`，理论上应满足 `fs > 2 fmax`；工程上还要为抗混叠滤波器过渡带和相位精度留余量。

高于 `fs/2` 的频率成分若没有被模拟抗混叠滤波器抑制，会折叠到低频区，形成无法通过数字后处理恢复的假频率。混叠峰值可能很稳定，却不是被测对象真实响应。量化位数越高，幅值步长越小，但提高位数不能修复传感器噪声、安装问题或前端饱和。

有限记录长度 `T` 决定频率分辨率，基本关系为 `Delta f = 1/T`。采样率主要决定可观察的最高频率，记录长度主要决定频率网格的细密程度；想区分相近频率，不能只提高采样率，还要延长有效记录。对非平稳过程，应采用分段或短时分析，而不是把整个过程直接做一次傅里叶变换。

## 频谱计算与处理顺序

推荐顺序是：

1. 查看原始波形，确认没有削顶、跳点、漂移、断线或工况切换。
2. 依据目标频带设计滤波，记录滤波器类型、阶数和截止频率。
3. 去除直流分量或趋势项，但不能误删真实低频信息。
4. 选择窗函数和分段长度，估计频率分辨率与泄漏影响。
5. 计算 FFT、幅值谱、功率谱或时频结果。
6. 结合转速、载荷、激励频率和结构固有频率解释峰值。
7. 用重复测量、不同传感器或不同工况复核结论。

窗函数不是让波形好看的装饰，而是有限观测的数学模型。窄主瓣有利于分辨相近频率，低旁瓣有利于观察弱信号；二者通常需要权衡。比较两次频谱时，应保持窗、记录长度、归一化方式和平均策略一致。

## 第二章公式卡与理解要点

### 周期量的基本参数

对周期信号 `x(t)=x(t+T)`：周期 `T` 表示重复一次所需的时间，频率 `f=1/T`，角频率 `omega=2 pi f`。基频是最低的非零频率，谐波是基频的整数倍。若信号含直流偏置，频谱在零频处会有直流分量；若波形具有对称性，某些谐波会因对称性消失。

周期信号可以写成正弦、余弦叠加，也可以写成复指数形式：`x(t)=sum C_n exp(j n omega_0 t)`。`C_n` 的模表示分量幅值关系，辐角表示相位。实信号正、负频率通常成共轭对称，因此单边谱需要重新归一化，不能直接把双边谱的一半当作单边谱。

### 连续谱的密度含义

非周期信号的 `X(omega)`不是某一根离散谱线的幅值，而是单位角频率宽度上的复幅值密度。取有限频带 `[omega_1,omega_2]` 时，总贡献需要在该区间上积分。频带越窄，单个频率点的数值越不能脱离频宽解释；这也是不同 FFT 长度和不同频率分辨率的结果不能直接比较峰值的原因。

傅里叶变换的工程互换包括：延时引入频域相位变化但不改变幅值谱；时间压缩使频谱展开；微分对应乘以 `j omega`，积分对应除以 `j omega`，所以积分容易放大低频漂移，微分容易放大高频噪声；时域卷积对应频域相乘。

## 频谱诊断流程

如果出现一个主峰，先检查它是否与转速、激励频率或已知固有频率一致；若随转速线性移动，可能是转频或阶次；若固定在某频率且接近该频率时响应放大，可能是结构模态。整数倍峰值可能来自非正弦激励、齿轮啮合、周期冲击、间隙碰撞或非线性，不能只凭“有二倍频”就断言故障类型。

若频谱底噪升高，先检查接地、屏蔽、电源、量程和安装，再检查采样频率与滤波器。若峰值展宽或漂移，可能是工况不稳定、结构参数变化、记录过长、分辨率不足、窗泄漏或转速波动，应尝试分段频谱、阶次跟踪、同步平均或时频分析。

## 第一章概念详解

测量是将被测量与预定标准进行定量比较，得到数值、单位和不确定度；检验是依据规范或阈值判断是否合格；试验是为验证性能、假设或设计而组织对象、工况、设备、方法和数据处理。测试技术把测量和试验结合起来，既关心数据如何获得，也关心数据如何支持工程判断。

测试对象可以是机器、零部件、材料、环境或运行过程。被测量是对象中希望量化的物理量，例如位移、速度、加速度、力、压力、温度、噪声和转速。信号是承载这些信息的时间函数或数据序列，可能表现为电压、电流、频率、脉宽、数字码或光学量。传感器输出不是被测量的原样复制，而是经过灵敏度、带宽、噪声和安装条件改变后的结果。

## 测试系统的六个环节

1. 激励或工况单元提供力、位移、转速、温度等输入。
2. 传感器感受被测量并转换为可传输信号。
3. 信号调理完成供电、放大、滤波、隔离、整流和匹配。
4. 采集与记录完成采样、量化、同步、存储和时间标记。
5. 分析与显示完成时域、频域、统计和相关计算。
6. 校准与参考单元提供标准输入、零点检查和灵敏度修正。

系统设计时要同时关注信息流和能量流。传感器可以输出有用信息，但安装也可能向被测对象引入附加质量、刚度或阻尼。测试系统不是越复杂越好，而是要在信号质量、侵入性、成本和可维护性之间平衡。

## 测量分类与选择

直接测量由仪器直接给出目标量；间接测量先测相关量，再依据模型换算。间接法便于安装或扩大量程，但会引入模型、积分漂移和参数不确定度。

静态测量关注变化很慢的对象，重点是灵敏度、零漂、重复性、线性度和分辨率；动态测量还要考虑带宽、频率响应、相位、上升时间、阻尼和采样。接触测量可能改变对象质量和刚度，非接触测量则更敏感于距离、表面状态、光照和电磁环境。数字测量便于存储和编程，但采样、量化、抗混叠滤波和时间同步会成为新的误差来源。

## 复习题

1. 测量和试验有什么区别？答：测量是获取某个量，试验是为验证目的组织系统、工况和方法。
2. 如何判断反馈是闭环还是半闭环？答：看测量点是否直接代表最终受控输出。
3. 间接测量的主要风险是什么？答：换算模型、参数误差和累积误差。
4. 为什么动态测量比静态测量多考虑带宽和相位？答：动态系统会随频率改变幅值和相位。
5. 为什么采样前要抗混叠滤波？答：防止高频成分折叠到低频形成伪信号。
6. 为什么提高采样率不能代替延长记录？答：采样率决定最高可观察频率，记录长度决定频率分辨率。
7. 为什么窗函数影响频谱峰值？答：有限记录相当于时域乘窗，频域卷积会造成主瓣展宽和旁瓣泄漏。
8. 一份可复现测试结果至少应包含什么？答：对象与工况、传感器和安装、采样参数、校准状态、处理方法、原始数据和不确定度。

## 工程案例与排查

以发动机振动测试为例，先定义机座三向加速度、转速和负载，再选择传感器方向、安装方式、采样率和抗混叠滤波。数据处理先看波形，再计算分段频谱或阶次谱；转速变化时不能对整段记录直接做一次 FFT，否则移动的峰值会被涂宽。结果要同时对照 1X 转频、倍频、结构固有频率和工况变化，不能只凭一个峰值判断故障。

闭环工作台定位中，电机轴端编码器只反映中间环节，丝杠间隙、热变形和负载引起的误差可能没有被反馈消除。若传感器装在最终工作台，反馈更接近真实输出，但安装和环境保护更困难。应同时记录目标位置、编码器位置、工作台位置和控制器修正量，才能区分控制、传动和测量误差。

| 异常现象 | 优先排查 |
|---|---|
| 波形平顶 | 量程、增益、前端饱和 |
| 整体漂移 | 零漂、温度、安装、去趋势 |
| 高频噪声增加 | 接地、屏蔽、线缆、采样卡 |
| 低频假峰 | 混叠、趋势项、积分漂移 |
| 峰值随记录长度变化 | 频率分辨率、泄漏、窗函数 |
| 多通道相位不一致 | 时钟、触发和通道延迟 |

4. 用“观测事实—可能解释—验证证据—适用限制”写一个测试结论，不要把未经验证的峰值直接命名为故障。

## 关键方法总结

这套课程最值得迁移的不是某个孤立公式，而是一种测试思维：先定义问题，再选择测量原理；先估计信号，再设计传感器、带宽和采样；先检查原始数据，再处理和画图；先将峰值与工况对应，再提出工程解释；最后用重复测试、校准和不确定度约束结论。任何跳过其中一步的“漂亮频谱”，都可能只是处理链的产物。

## 术语表

| 术语 | 含义 | 使用提醒 |
|---|---|---|
| 被测量 | 希望获得的物理量 | 先明确对象、位置、方向和单位 |
| 传感器 | 感受被测量并转换信号的装置 | 关注灵敏度、量程、带宽和安装 |
| 灵敏度 | 输出变化与输入变化的比例 | 需结合校准和单位使用 |
| 量程 | 允许输入的范围 | 超量程会削顶或损坏 |
| 带宽 | 能可靠传递的频率范围 | 不等于采样频率 |
| 采样频率 | 每秒采集样本数 | 决定最高可观察频率 |
| 频率分辨率 | 相邻频率网格间隔 | 主要由记录长度决定 |
| 频谱泄漏 | 有限记录导致能量扩散 | 与记录长度、窗函数有关 |
| 主瓣 | 窗函数频谱的主要能量区 | 越窄通常越利于分辨近频 |
| 旁瓣 | 主瓣两侧的泄漏成分 | 越低越利于观察弱信号 |
| 混叠 | 高频成分折叠为低频假成分 | 采样前必须抗混叠滤波 |
| RMS | 均方根值 | 常用于振动和交流信号能量比较 |
| 功率谱密度 | 单位频宽上的功率分布 | 注意单位和归一化方式 |

## 公式与单位核对

1. `f=1/T`：周期用秒时，频率单位为 Hz。
2. `omega=2 pi f`：角频率单位为 rad/s，不能直接把 Hz 代入以 `omega` 为变量的公式。
3. `Delta f=1/T_record`：记录长度越长，频率网格越细。
4. `fs > 2 fmax`：这是理想采样下限，工程上还要留滤波器过渡带。
5. 傅里叶变换中的积分变量和频率变量会影响量纲，连续谱值不能脱离频宽解释。
6. 单边谱通常需要对非直流、非 Nyquist 分量做能量折算，不能把双边谱直接截半。

## 数据处理决策树

**原始波形是否正常？**

- 否：先排查传感器、线缆、量程、供电、接地和同步；不要直接滤波掩盖问题。
- 是：继续判断信号是否平稳。

**信号是否平稳？**

- 平稳：可使用整段 FFT、分段平均或功率谱估计。
- 非平稳：使用短时傅里叶、时频分析、阶次跟踪或分工况处理。

**是否关心瞬态峰值？**

- 是：保留原始峰值，谨慎去趋势和滤波，检查滤波器相位。
- 否：可用 RMS、功率谱或频带能量进行统计比较。

**频谱是否出现异常峰？**

- 先核对采样和混叠。
- 再核对转速、激励、结构模态和传感器位置。
- 最后通过改变工况或测点进行复核。

## 综合自测

1. 若要测量一台电机的振动，为什么不能只买一个量程足够大的传感器？
2. 若两次频谱的峰值不同，哪些处理参数必须先确认一致？
3. 为什么相同的物理信号在不同记录长度下会显示不同频率分辨率？
4. 为什么一个频谱峰值不能单独证明存在故障？
5. 采样频率提高后仍可能发生混叠吗？答：如果没有有效抗混叠滤波，仍可能有超过 Nyquist 频率的成分折叠。
6. 窗函数主瓣和旁瓣各自影响什么？答：主瓣影响频率分辨率，旁瓣影响弱信号被强信号泄漏掩盖的程度。
7. 什么时候应使用间接测量？答：目标量难以直接安装传感器，但可由稳定可靠的模型换算时。
8. 为什么动态测试必须记录工况？答：同一对象的响应会随转速、载荷、温度和边界改变，不记录工况就无法解释差异。
9. 测试报告为什么要保留原始数据？答：便于复核处理步骤、发现误差和在新问题下重新分析。
10. 测试技术的最终输出是什么？答：经过校准、处理、验证并限定适用条件的工程判断，而不是单纯一组数字。

## 误差预算与校准流程

测试前可建立简化误差预算，把不确定度分为传感器、安装、调理、采集、算法和重复性六类。先判断哪一类是主导项，再决定投入校准、改进安装或增加重复次数。不要平均地优化所有环节：若安装误差远大于量化误差，提高采集卡位数不会改善结果。

推荐的校准顺序是：空载检查零点；使用标准输入检查灵敏度；在多个幅值点检查线性；在目标频带检查幅频和相频响应；最后用已知信号走完整测量链，验证软件单位、增益和显示值。校准记录应包含日期、标准源、环境条件、修正系数和有效期。

## 频谱参数对照

| 参数 | 主要决定 | 常见误解 |
|---|---|---|
| 采样率 `fs` | 最高可观察频率 | 不能决定频率分辨率 |
| 记录长度 `T` | `Delta f=1/T` | 不能直接提高最高频率 |
| FFT 点数 | 频率网格显示密度 | 补零不会增加真实分辨率 |
| 窗函数 | 主瓣、旁瓣和幅值修正 | 没有绝对最优窗 |
| 平均次数 | 稳定随机谱估计 | 会削弱瞬态信息 |
| 抗混叠滤波 | 防止高频折叠 | 数字滤波无法恢复已混叠成分 |

## 一套最小可行测试方案

当时间和设备有限时，可采用以下最小方案：明确一个被测量和一个判据；选一只量程和带宽匹配的传感器；记录原始波形和工况；采样率留出目标最高频率两倍以上的余量；先做一次短记录确认信号量级，再做正式记录；保存原始数据和全部处理参数；用至少一次重复测试检查稳定性。即使不能完成完整不确定度分析，也必须说明主要误差来源和结论限制。

## 课程总结

第一章建立测试技术的工程边界：测试是测量与试验的结合，包含原理、方法、系统和数据处理。第二章建立信号语言：时域说明什么时候发生，频域说明由哪些频率组成；周期信号产生离散谱，非周期信号对应连续谱；傅里叶变换连接两种描述；窗函数和有限记录决定频谱泄漏与分辨率。

把两章合起来，可以得到一个可迁移的判断链：

`明确目标 -> 定义被测量 -> 选择原理 -> 选择方法 -> 搭建系统 -> 校准 -> 采样 -> 处理 -> 解释 -> 验证`

后续学习传感器、测试系统特性和数字信号处理时，始终沿着这条链检查。它可以帮助你识别“仪器读数正常但测试结论错误”的情况，也能把课本公式转化为真实机械系统中的可执行方案。

## 后续章节的学习接口

### 测试系统的静态特性

后续学习应先掌握灵敏度、线性度、迟滞、重复性、分辨率、漂移和死区。灵敏度决定输入变化能否在输出中被观察；线性度决定校准曲线能否用简单比例关系表示；重复性反映相同条件下多次测量的一致程度；漂移则说明零点或灵敏度会随时间和环境变化。选择传感器时，不应只看量程和价格，还要看这些指标与判据是否匹配。

### 测试系统的动态特性

动态特性通常用阶跃响应、频率响应、幅频特性和相频特性描述。阶跃响应可以观察上升时间、超调、稳定时间和阻尼；频率响应可以判断某个频带内的幅值衰减和相位滞后。被测对象、传感器、调理器和采集系统会串联形成总传递特性，因此某个峰值可能来自测试系统自身，而不是被测结构。

实验中可用已知正弦扫频、标准冲击或校准激励检查动态响应。若系统带宽不足，直接提高软件采样率没有意义；若系统出现共振，应改变安装、阻尼或测点，而不是简单在结果中删除峰值。

### 传感器选型的四问

1. 被测量是什么，方向和作用点是否明确？
2. 幅值范围、频率范围和瞬态峰值是多少？
3. 传感器安装是否会改变结构或工况？
4. 输出接口、供电、调理、环境适应性和校准方式是否匹配？

例如测高频冲击，优先关注固有频率、上升时间和过载恢复；测低频位移，优先关注零漂、分辨率、积分误差和环境温度。没有脱离任务的“最好传感器”，只有在误差预算和安装条件下更合适的方案。

### 复现与版本管理

测试数据应与脚本、参数、校准文件、传感器方向、设备连接图和工况记录一同保存。处理后的频谱应能由原始数据重新生成；若更换窗函数、滤波器或平均策略，应保留版本号和变更说明。对长期监测项目，还要记录传感器更换、重新校准、设备维修和环境变化，否则跨时间比较可能把设备变化误认为对象变化。

## 最终学习清单

- 能用一句话定义测试目标和判定指标。
- 能画出完整测试系统并指出各环节误差。
- 能根据频率范围、幅值范围和环境选择传感器。
- 能区分采样率、记录长度、FFT 点数和频率分辨率。
- 能解释周期谱、连续谱、频谱密度和窗函数。
- 能从波形、频谱和工况三者联合解释现象。
- 能设计校准、重复性和异常排查方案。
- 能写出带单位、处理参数、不确定度和适用条件的测试结论。

## 一页式复习提纲

### 第一层：概念

测量是与标准进行定量比较；试验是为验证目的组织方法、系统和工况；测试是测量与试验技术的结合。测试的对象不是孤立数字，而是对象、信号、系统、处理和判据组成的证据链。

### 第二层：系统

测试系统可概括为“激励/工况—传感器—调理—采集—处理—显示/判定”。传感器负责感受和转换，调理负责把弱信号变成可用信号，采集负责数字化，处理负责提取指标。任何环节的带宽、噪声、相位和校准都会影响最终结果。

### 第三层：信号

周期信号由基频和谐波构成，频谱是离散谱；非周期信号可看成周期趋于无穷的极限，频谱变为连续谱。傅里叶变换连接时域和频域；频谱函数要按单位频宽解释。角频率满足 `omega=2 pi f`，记录长度决定 `Delta f=1/T`，采样率决定最高可观察频率。

### 第四层：数据

有限记录会产生泄漏，窗函数在主瓣宽度和旁瓣高度之间权衡。采样前要抗混叠滤波，采样后要检查削顶、漂移、丢点和同步。频谱峰值必须与转速、激励、模态、测点和工况对应，不能凭图形直接下结论。

### 第五层：工程判断

结论应包含观测事实、可能解释、验证证据和适用限制。一个可复现的结果必须能够回到原始数据、传感器安装、校准参数和处理脚本。测试技术的目标是支持决策，而不是制造一张看起来复杂的图。

## 重点对照表

| 问题 | 应检查的核心量 | 典型错误 |
|---|---|---|
| 测什么 | 被测量、位置、方向、单位 | 目标不清，只说“测振动” |
| 怎么测 | 原理、方法、传感器 | 直接套用现有仪器 |
| 采多久 | 记录长度、分辨率 | 只提高采样率 |
| 采多快 | 最高频率、抗混叠 | 忽略 Nyquist 条件 |
| 怎么处理 | 去趋势、滤波、加窗、平均 | 处理参数不记录 |
| 怎么解释 | 峰值与工况对应 | 把所有峰值当故障 |
| 如何证明 | 校准、重复性、交叉验证 | 只保留处理后图片 |

## 考试与实验快速检查卡

遇到概念题，先回答“定义—用途—边界”：先说明术语是什么，再说明它解决什么问题，最后指出不能如何滥用。遇到计算题，先统一单位和变量，再列出已知量、未知量、公式和结果单位；涉及频率时先判断题目使用 `f` 还是 `omega`。

遇到频谱题，按“信号类型—频率轴—幅值含义—窗与分辨率—工程解释”的顺序作答。先判断是周期、非周期、随机还是非平稳信号；再说明离散谱或连续谱；然后核对幅值谱、RMS 谱或功率谱密度的单位；最后结合采样率、记录长度、窗函数和工况解释峰值。

遇到测试系统设计题，先画方框图，再逐块写功能和误差。传感器不只是一个“测量端”，还可能改变结构；调理器不只是放大器，还承担滤波、隔离、匹配和保护；采集卡不只是存数据，还决定采样、量化、同步和混叠风险。

实验结束前检查五件事：原始数据是否保存；单位和校准系数是否记录；传感器方向和安装是否可复现；处理参数是否写清；结论是否包含工况和限制。缺少其中任意一项，后续复核都会困难。

## 结论质量标准

高质量结论应能回答四个问题：看到了什么、可能为什么、凭什么这么判断、在哪些条件下成立。比如“频谱有峰值”只是观察；“峰值位于转频附近”是更具体的观察；“峰值随转速线性移动且更换测点仍存在”才提供了支持转频来源的证据；“仍需改变负载或做扫频验证结构响应”则说明了边界。

本课程的最终能力不是记忆设备名称，而是把一个模糊的工程问题转成可测量、可采样、可处理、可验证的测试方案，并知道每一步可能引入什么误差。
