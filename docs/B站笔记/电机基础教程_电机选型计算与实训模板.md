## 负载计算

分别计算：

- 稳态摩擦/阻力；
- 加速力或加速转矩；
- 重力分量；
- 外部工艺力/切削力；
- 启动、急停和最大负载；
- 传动效率和安全系数。

不要把稳态转矩当作加速转矩，也不要用峰值转矩代替连续热能力。

## 电机类型判断

| 条件 | 倾向选择 |
|---|---|
| 低速定位、成本敏感、允许失步风险可控 | 步进电机 |
| 需要反馈、高动态和高精度 | 伺服电机 |
| 连续旋转、成本和维护优先 | 有刷/无刷驱动电机 |
| 取消传动、低回差、高响应 | 直驱电机 |

最终选择还要核对驱动器、编码器、制动、电源、热和机械安装。

## 验证实验

1. 空载确认方向、速度和反馈。
2. 低速加载，观察启动和定位。
3. 分阶段提高速度，记录转矩/推力和跟随误差。
4. 测试加速、减速、急停和反向。
5. 按最大连续工况运行，记录温升。
6. 改变负载惯量或外部扰动，检查稳定性。
7. 对照选型计算，分析实际与理论差异。

## 失败模式

- 步进电机高速失步：检查动态转矩、加速度、驱动电流和惯量。
- 伺服振荡：检查增益、惯量匹配、机械刚度、反馈和共振。
- 驱动电机过热：检查负载、连续电流、散热和驱动器匹配。
- 直驱低速抖动：检查编码器、控制带宽、机械刚度和负载扰动。
- 丝杆速度不够：检查导程、电机转速上限和传动效率。
- 输送机启动困难：检查启动阻力、加速转矩和减速机构。

## 复盘问题

1. 这次选型的最坏工况是什么？
2. 连续能力和峰值能力分别由什么约束？
3. 负载惯量折算后，电机是否仍能满足动态要求？
4. 如果改用减速机/直驱，哪些计算必须重做？
5. 实测温升、速度和跟随误差是否支持选型结论？
