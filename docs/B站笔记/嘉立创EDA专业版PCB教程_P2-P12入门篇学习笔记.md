## P2-P3：课程与 PCB 技术背景

入门阶段先建立 PCB 的整体认识：PCB 不只是把导线画在板子上，而是把电路连接、器件安装、机械结构、信号完整性、散热和制造约束统一到一个可生产实体中。学习软件之前，应先理解原理图、封装、网络、层、板框和生产文件之间的关系。

PCB 技术的发展推动了从手工连线到高密度、多层、高速和自动化制造的变化。对初学者来说，最重要的不是记历史，而是理解为什么需要规范的原理图、封装库、层叠、规则检查和生产输出。

## P4-P5：基本电子元件

电阻、电容、电感是电路分析和 PCB 设计的基础。电阻主要用于限流、分压、偏置和终端；电容用于储能、去耦、滤波和耦合；电感用于储能、滤波和电源转换。二极管具有方向性，三极管和场效应管可用于放大、开关和功率控制。

做 PCB 之前要关注的不只是符号，还包括器件封装、极性、耐压、电流、功率、散热和焊接方式。极性器件和三极管/MOS 引脚顺序一旦封装错误，原理图检查可能通过，但实物会直接失效。

## P6：元件数据手册

数据手册是器件选型和电路设计的主要依据。初学者应重点查找：绝对最大额定值、推荐工作条件、引脚定义、封装尺寸、典型应用电路、电气特性、时序参数、热参数和布局建议。

绝对最大额定值不是推荐工作点，不能把它当作长期运行目标。对电源、MOS、稳压器和接口芯片，还要检查启动、短路、温升、输入输出范围和保护行为。

## P7：电路定理

电路定理用于把复杂电路拆成可分析的局部问题。学习时应掌握电压、电流、节点、回路、串并联、分压分流和等效思想，并能把器件参数转化为网络约束。

实际 PCB 设计中，电路分析直接影响电阻取值、去耦位置、反馈网络、电源电流和信号参考。先完成纸面计算，再进入 EDA 绘图，可以减少“图画对了但数值错了”的问题。

## P8：读懂原理图

读原理图建议按电源、核心器件、输入、输出、时钟、复位、通信和调试接口分块。先找电源入口和地，再沿信号方向追踪网络；对每个网络明确来源、去向、电平和功能。

原理图中的网络标签、总线、分页连接和电源符号必须保持一致。不要只看线是否相连，还要检查网络名称、引脚类型、未连接标记和输入输出方向。

## P9-P10：PCB 结构与叠层

PCB 由板材、铜层、介质层、阻焊、丝印、焊盘、过孔和机械边界构成。单层、双层和多层板在布线空间、回流路径、成本和制造复杂度上不同。

叠层设计应服务于电源完整性、信号参考、阻抗、散热和制造能力。高速或敏感信号需要连续参考平面；功率路径和地回流不能只靠视觉上“铺满铜”解决。

## P11：元件符号与封装

原理图符号表达电气功能，PCB 封装表达物理焊接和机械尺寸，两者通过引脚编号映射。符号引脚 1、2、3 与封装焊盘 1、2、3 不一致，是 PCB 设计中最危险的错误之一。

绘制或调用封装时，应核对数据手册的焊盘尺寸、间距、方向、丝印、阻焊、装配高度和极性标志。连接器、芯片、二极管、电解电容和 MOS 尤其需要人工复核。

## P12：PCB 设计流程

标准流程是：需求分析 -> 元件选型 -> 原理图 -> ERC/DRC 前检查 -> 封装绑定 -> 导入 PCB -> 板框和机械约束 -> 布局 -> 布线 -> 铺铜 -> DRC -> 丝印检查 -> 生产文件导出 -> 首板测试。

不要在没有确定原理图和封装的情况下直接画 PCB。PCB 是电路逻辑、器件尺寸、机械结构和制造规则的综合结果，后期修改封装或接口方向往往会引起连锁返工。
