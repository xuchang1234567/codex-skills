# 电机基础教程：电机系统故障排查

## 通用顺序

1. 断电确认接线、极性、机械卡滞和安装。
2. 核对电机、驱动器、电源和反馈型号。
3. 空载低速运行，确认方向和反馈。
4. 逐步增加速度、加速度和负载。
5. 记录电流、温升、振动、噪声、误差和报警。
6. 一次只修改一个参数，并保留原始设置。

## 步进电机失步

检查目标速度下动态转矩是否足够、加速度是否过大、负载惯量是否匹配、驱动器电流和电源电压是否正确。还要检查共振、细分设置、机械卡滞和脉冲信号质量。

保持转矩足够不代表高速不失步；降低加速度、增加合理减速比、改善驱动和避开共振区可能比盲目增大电机更有效。

## 伺服振荡或跟随误差大

检查位置/速度/转矩环参数、采样和反馈方向、惯量比、机械刚度、联轴器间隙、负载变化和共振。增益过高会振荡，过低会响应慢；积分过强可能造成超调和恢复慢。

先低增益确认方向和反馈正确，再逐步增加响应。不要在机械松动或惯量严重不匹配时靠 PID 参数掩盖问题。

## 有刷直流电机火花、噪声或过热

检查电刷/换向器状态、负载电流、启动和堵转、驱动器 PWM、续流/吸收回路和散热。火花可能与换向、过流、机械负载或电刷磨损有关。

## 无刷电机不转或抖动

检查换向顺序、霍尔/位置反馈、相线、电源、驱动器使能和启动策略。相序错误可能表现为抖动、反转、过流或启动失败。

## 驱动电机过热

区分电机本体、驱动器、电缆和机械轴承的热源。检查连续电流、负载转矩、散热、供电压降、堵转、制动和驱动器限流。额定功率不是允许任何工况长期运行的保证。

## 直驱电机低速抖动

检查编码器分辨率、反馈噪声、控制带宽、机械刚度、负载惯量、结构共振和电机齿槽转矩。直驱取消了传动环节，也取消了传动机构对扰动和惯量的部分隔离。

## 线性机构运行异常

检查导轨摩擦、丝杆预紧、皮带张力、齿轮间隙、偏载、垂直重力和制动。速度/转矩计算正确但机构安装不正确，仍会出现过流、定位误差和振动。

## 选型复核

出现故障时重新核对：

- 负载侧实际力/转矩；
- 速度和加速度；
- 连续和峰值工况；
- 负载惯量折算；
- 传动效率和安全系数；
- 驱动器和电源；
- 热、机械刚度和反馈；
- 启动、急停、堵转和断电状态。
