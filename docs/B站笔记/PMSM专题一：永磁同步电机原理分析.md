# PMSM专题一：永磁同步电机原理分析

## 学完你应该获得什么

- 能解释永磁同步电机（PMSM）的“永磁”和“同步”分别指什么。
- 能区分转子永磁体、定子三相绕组、位置传感器和逆变器的作用。
- 能理解旋转磁场如何产生电磁转矩并带动转子同步旋转。
- 能建立 PMSM 的磁路、电路和机械运动三类基本方程。
- 能看懂后续矢量控制、坐标变换和 Simulink 建模所依赖的变量。

## 一句话总论

PMSM 用转子永磁体提供稳定磁场，用定子三相电流产生可控旋转磁场；两者相互作用形成电磁转矩。要实现精确控制，必须同时描述磁链、电压电流和机械运动，并区分电角速度与机械角速度。[1][8]

## 知识地图

`电机结构 -> 磁场相互作用 -> 同步运行 -> 磁路方程 -> 定子电压方程 -> 机械运动方程 -> 后续控制与仿真`。

视频是一门控制系列课程的第一讲，因此主要完成“建立物理图像和数学入口”，并未展开 PID、滑模、预测控制、无位置传感器控制或驱动硬件的完整设计。[1]

## PMSM 的结构与分类

普通电机由定子和转子组成。异步电机的典型鼠笼转子由导条和端环构成，定子三相绕组产生旋转磁场，转子因感应电流产生磁场；转子速度始终略低于定子旋转磁场速度，所以称为异步。[2][3]

PMSM 的转子不再依靠感应导条，而是在转子铁芯中安装稀土永磁体；定子仍是三相绕组。所谓“永磁”，强调转子磁场由永磁体提供；所谓“同步”，强调转子磁场与定子旋转磁场保持同步速度。[4]

PMSM 通常还配有位置反馈接口。霍尔传感器成本低、精度有限；光电编码器精度较高；旋变更适合对可靠性和环境适应性有要求的场景。位置的意义是告诉控制器转子磁极在哪里，从而决定定子电流磁链的方向。[5]

按永磁体安装方式，可分为表贴式和内埋式。表贴式结构简单、转动惯量小，常近似 `L_d = L_q`；内埋式磁路不对称，通常 `L_d != L_q`，因而除了永磁转矩外还可能产生磁阻转矩。这个差异会直接影响后续 d-q 轴模型、弱磁控制和最大转矩电流比控制。[6]

## PMSM 如何产生转矩

电机能量转换可以用三步理解：电能进入定子绕组，绕组电流按安培定律产生磁场，定子磁场与转子永磁场相互作用，最终转化为机械能。[7]

单个磁极的直观规律是异性相吸、同性相斥。若定子磁场方向固定，转子永磁体会转到更有利的相对位置；两磁场在合适的夹角下产生切向力，切向力乘以力臂就是电磁转矩。三相绕组的价值在于通过时序变化的电流合成连续旋转磁场，而不是让转子只在几个固定位置之间跳动。[7]

控制器给逆变器电压指令，电压建立定子电流；电流产生定子旋转磁链；定子磁链再与转子永磁磁链作用。要让电机加速、减速或保持某个角度，本质上都是在调节定子磁链的幅值、方向和相位。[7][8]

当定子旋转磁场速度与转子磁场速度一致时，PMSM 进入同步运行。多对极电机需要特别区分电角速度与机械角速度：

`omega_e = p * omega_m`

其中 `p` 是极对数，`omega_e` 是电角速度，`omega_m` 是转子机械角速度。极对数越多，同一机械转速对应的电角变化越快。[9]

## 三组基本数学关系

### 1. 磁路与磁链

定子磁链由定子电流产生的磁链和转子永磁体产生的磁链叠加。定子磁链是随电角位置旋转的量；转子永磁磁链的幅值近似恒定，但方向随转子机械位置旋转。总磁链决定了定子绕组感应电压和电磁转矩。[8][9]

在后续 d-q 坐标系中，通常把坐标轴与转子磁场对齐，从而把交流量转换成便于控制的直流量。表贴式和内埋式的 `L_d`、`L_q` 差异，会在这个坐标系下表现为不同的电磁转矩组成。[6]

### 2. 定子电压方程

定子电压的物理结构是：电阻压降 + 电感电压 + 磁链变化引起的反电动势。一般可写成向量形式：

`u_s = R_s i_s + d(psi_s)/dt`

展开磁链后，电感项反映电流变化，永磁磁链项反映转子磁场对定子的作用。这里的电压、电流和磁链通常先在三相自然坐标系中描述，之后再通过 Clarke/Park 变换进入静止或同步旋转坐标系。[10][11]

不要把反电动势理解成额外串联的“电源”。它来自磁链随时间变化，是电机运动状态与电路状态耦合的结果；转速越高，反电动势通常越明显，也会限制逆变器可提供的有效电压裕量。[10]

### 3. 机械运动方程

机械侧的核心是转矩平衡：电磁转矩需要克服负载转矩、摩擦/风阻等阻尼转矩，并用于改变转子转速。可写成：

`T_e = T_L + B omega_m + J d(omega_m)/dt`

其中 `T_e` 是电磁转矩，`T_L` 是负载转矩，`B omega_m` 是与速度相关的阻尼转矩，`J d(omega_m)/dt` 是加速转矩。若电磁转矩大于负载与阻尼之和，转子加速；若小于它们，转子减速。[12]

## 面向控制的建模流程

1. 明确电机类型、极对数、绕组接法、`R_s`、`L_d/L_q`、永磁磁链和转动惯量等参数。
2. 先建立三相自然坐标系中的电压方程和磁链关系，检查符号、单位和正方向。
3. 根据转子位置建立电角度关系，确认 `theta_e = p theta_m`，避免把机械角直接当电角使用。
4. 将电路方程与机械方程连接：电压决定电流，电流决定磁链和电磁转矩，转矩再决定转速与位置。
5. 再进入 d-q 坐标系和控制器设计，逐项验证坐标变换、反电动势、转矩方向和负载响应。
6. 在 Simulink 中先做开环或简单给定验证，再增加 PI/PID、矢量控制和位置反馈，不要一开始就堆叠全部控制模块。[1][13]

## 常见误区

- “永磁”不等于不需要电流：永磁体提供转子磁场，但定子电流仍负责建立旋转磁场和控制转矩。
- “同步”不等于启动时天然同步：启动和失步风险取决于控制策略、位置估计、负载与电流能力。
- 不区分表贴式和内埋式：`L_d = L_q` 不能对所有 PMSM 通用。
- 混用机械角速度与电角速度：多对极时会造成频率、反电动势和控制相位全线错误。
- 只写电压方程不写机械方程：这样只能描述电路，无法解释加速、稳态转速和负载扰动。

## 自测题

1. PMSM 的“永磁”和“同步”分别指什么？答：转子磁场由永磁体提供；转子与定子旋转磁场同步运行。
2. 为什么需要转子位置反馈？答：用于确定磁极方向，正确设置定子电流磁链相位。
3. 表贴式和内埋式的关键电感差异是什么？答：前者常近似 `L_d=L_q`，后者通常 `L_d!=L_q`。
4. 电角速度与机械角速度有什么关系？答：`omega_e=p omega_m`。
5. 机械运动方程中的惯性项表示什么？答：改变转速所需克服的转动惯量效应。
