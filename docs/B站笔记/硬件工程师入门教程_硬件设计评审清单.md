# 硬件工程师入门教程：硬件设计评审清单

## 需求与边界

- [ ] 输入电压范围、额定值和瞬态范围已定义。
- [ ] 输出电压、电流、纹波、响应速度和精度已定义。
- [ ] 工作温度、环境光、机械尺寸和接口约束已记录。
- [ ] 正常、启动、关机、堵转、短路、反接和过温状态已考虑。
- [ ] 测试方法和通过标准已写清楚。

## 原理图

- [ ] 电源入口有极性、保险、限流或防反接设计。
- [ ] 每个 IC 的电源、地、使能和配置脚都有明确状态。
- [ ] 输入悬空不会产生危险误动作。
- [ ] 去耦电容、滤波器、上拉/下拉和测试点齐全。
- [ ] 感性负载有续流、钳位或吸收回路。
- [ ] 采样点、反馈点和功率回路连接正确。
- [ ] 每个器件的耐压、电流、功率和温升有计算依据。

## 器件与 BOM

- [ ] 型号、封装、引脚顺序和数据表版本已核对。
- [ ] MOS 按实际 `Vgs` 下的 `Rds_on` 选择。
- [ ] 二极管检查正向电流、反向耐压和恢复时间。
- [ ] 电感检查饱和电流、额定电流、DCR 和温升。
- [ ] 电容检查耐压、容量、ESR、纹波电流和极性。
- [ ] 运放/比较器检查共模范围、输出摆幅、带宽和稳定性。
- [ ] 关键器件有替代型号和供应风险说明。

## 电源与热设计

- [ ] 最大输入、最大输出和效率已计算。
- [ ] 线性稳压热耗散和散热路径满足要求。
- [ ] 开关电源峰值电流和开关节点耐压满足要求。
- [ ] 启动、空载、满载、短路和输入突变已计划测试。
- [ ] 功率地与信号地回流路径已明确。
- [ ] 高 di/dt 回路面积尽量小。

## PCB 布局

- [ ] 电源入口、功率器件、电感和输出电容布局合理。
- [ ] 去耦电容靠近对应引脚。
- [ ] 采样线采用独立、短、对称或 Kelvin 连接。
- [ ] 模拟信号远离开关节点和电机大电流线。
- [ ] 线宽、铜厚、过孔和间距满足电流/电压要求。
- [ ] 散热铜皮和热过孔已规划。
- [ ] 接插件、安装孔、板框和器件高度满足机械结构。
- [ ] 极性和方向丝印清晰，测试点可探测。

## 设计验证

- [ ] 断电测量电源与地之间无异常低阻。
- [ ] 限流上电测空载电流。
- [ ] 分模块验证电源、逻辑、驱动、传感和反馈。
- [ ] 示波器测量启动、关断、负载突变和开关节点。
- [ ] 记录器件温升和最热点位置。
- [ ] 测试异常状态，不只测试正常运行。
- [ ] 结果包含输入条件、仪器设置、波形和结论。

## 生产资料

- [ ] 原理图、PCB、BOM、坐标文件和生产图纸版本一致。
- [ ] Gerber、钻孔、阻焊和丝印文件已检查。
- [ ] DRC/ERC 结果已保存并审查例外项。
- [ ] 关键器件替代料不会改变引脚和电气特性。
- [ ] 首板修改记录和版本号可追溯。
