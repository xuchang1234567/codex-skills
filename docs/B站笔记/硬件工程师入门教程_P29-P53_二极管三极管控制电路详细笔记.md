# 硬件工程师入门教程 P29-P53：二极管、三极管与基础控制电路

## P29-P37：二极管

二极管的核心作用是利用单向导电特性控制电流方向。实际器件不是理想开关，需要考虑正向压降、反向漏电、最大正向电流、最大反向耐压和功耗。选择二极管时，不能只看“能不能导通”，还要检查反向状态是否会被击穿。

### 整流

半波整流只利用一个半周期，全波整流利用两个半周期，输出仍然包含纹波，需要电容、稳压或后级滤波进一步处理。峰值检波利用二极管给电容充电并在输入下降时保持峰值附近电压，设计时要同时考虑负载放电速度和二极管压降。

### 钳位与方向控制

钳位电路把信号限制在某个电压范围，保护后级输入或改变信号直流基准。二极管控制电路的电流方向时，应画出正向和反向两种工作状态，检查每个节点的电压和电流路径。

### 稳压二极管

稳压二极管利用反向击穿区附近的特性提供近似稳定电压，但必须配合限流电阻，保证工作电流处于允许范围。输入变化、负载变化和器件功耗都会改变稳定效果；它适合作为简单参考或小电流稳压，不应无条件替代完整稳压器。

## P38-P46：三极管开关

三极管可以作为放大器，也可以作为电子开关。作为开关时，需要根据基极驱动、电流增益、饱和压降、负载电流和关断状态设计基极电阻与下拉/上拉路径。不能只按“基极有电流就导通”判断，还要检查是否真正进入所需工作区。

### P 型与 N 型

P 型、N 型器件在高端/低端开关中的连接方式和控制电平不同。选择时要看负载位置、控制信号电压、参考地和是否需要电平转换。基极下拉或上拉电阻用于定义默认状态，防止控制端悬空导致误动作。

### 耐压与功耗

三极管最大耐压不能只按额定电源电压选择，开关瞬态、感性负载和布局寄生也可能造成更高尖峰。设计时应预留安全裕量，并为继电器、电机等感性负载提供续流或钳位路径。

## P47-P53：从器件到系统案例

自动出水水龙头、温控风扇、人体感应灯和光控灯把传感器、比较器/阈值、三极管开关和负载组合起来。分析这类电路时可按以下顺序：

1. 传感器输出随环境如何变化；
2. 阈值或比较器如何判断触发；
3. 三极管处于关断、放大还是饱和开关状态；
4. 负载电流从哪里来，是否需要续流或限流；
5. 上电、断电和输入悬空时默认状态是什么。

比较器上拉电阻不能省略时，通常与开漏/开集电极输出有关；输出晶体管只能主动拉低，释放时需要外部电阻把信号拉高。上拉阻值要在上升速度、功耗、输入漏电和多器件并联之间折中。

## 设计检查清单

- 二极管反向耐压是否大于可能出现的峰值？
- 正向电流和平均功耗是否满足？
- 整流后的纹波是否满足后级需求？
- 稳压二极管是否有可靠限流？
- 三极管基极默认电平是否明确？
- P/N 型器件的高端或低端位置是否正确？
- 感性负载是否有续流/钳位路径？
- 比较器开漏/开集输出是否配置上拉？
- 传感器阈值变化时，电路是否会颤振？

## 自测

1. 半波整流和全波整流的输出纹波与利用率有什么差异？
2. 为什么稳压二极管必须配限流电阻？
3. 三极管作为开关时，怎样判断它是否进入饱和？
4. 为什么输入悬空会导致三极管误导通？
5. 感性负载断电时，二极管的续流路径如何保护开关管？
6. 开漏/开集输出为什么需要上拉电阻？
