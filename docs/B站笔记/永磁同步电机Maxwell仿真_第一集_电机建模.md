## 这集应当解决什么问题

电机仿真的第一关不是立刻点击求解，而是建立一个“物理上合理、几何上闭合、命名上清楚、后续可赋材料和激励”的模型。对永磁同步电机，至少要明确：定子、转子、气隙、永磁体、槽型、绕组区域，以及内转子/外转子和二维/三维建模方式。

建模完成后，后续 Maxwell 流程通常还要继续完成材料、绕组和电路激励、运动设置、边界条件、网格、求解设置和结果后处理。因此，“模型画出来”不等于“模型已经可以可信地计算”。

## 建模时的关键检查点

### 1. 先固定结构和参数

在软件中输入或绘制前，先整理极数/槽数、定转子尺寸、气隙长度、磁钢尺寸与充磁方向、绕组匝数和电流等参数。参数必须有来源：设计指标、图纸、测量值或明确的假设。评论区有新手追问定子/转子 `hs`、`b` 等参数如何取得，说明参数含义和来源是学习中的主要门槛之一。[C271181105322]

### 2. 保证几何关系自洽

重点核对定子外径、内径、转子外径、磁钢位置和气隙之间的对应关系，避免出现参数定义不了、`dy` 与 `dg` 对不上、尺寸相互覆盖等问题。几何体应有清晰的独立区域，特别是气隙、永磁体和绕组区域，不能只看二维轮廓是否“像电机”。[C228542441888][C257995952384]

### 3. 选择合适的建模入口

如果目标电机能被 RMxprt 的模板描述，可先用 RMxprt 建立参数化模型，再导入 Maxwell 2D；评论者认为这样能减少手动画图工作。若库中没有目标结构，或是外转子、特殊槽型等结构，则可能需要自行绘制并用布尔操作整理几何。[C260136525713][C259291829825]

### 4. 绕组和命名要为后续求解服务

绕组不是“填一个名称”就结束：需要确认槽内分相、线圈方向、串并联关系、匝数和电流正负。永磁体也应使用稳定、可读的名称，方便材料赋值、运动区域设置和结果检查。评论区反复询问绕组参数、自动分相、永磁体命名，说明这些设置很容易成为后续报错或结果异常的来源。[C299530619758][C285707051008]

## 可执行的复现清单

1. 建立项目并统一单位，确认使用 Maxwell 2D 还是 3D。
2. 记录槽数、极数、定转子主要尺寸、气隙和磁钢参数。
3. 用参数化模板或草图建立定子、转子、磁钢、绕组和气隙区域。
4. 检查所有区域闭合、无重叠，确认磁钢充磁方向和气隙尺寸。
5. 对几何体和绕组区域统一命名，保存一份参数表。
6. 不急于求解，先验证模型能否顺利进入材料、绕组、运动和网格设置阶段。

## 评论区反馈能提供的额外提醒

评论者提到外转子模型的设置可能不同、RMxprt 模型库未必覆盖目标电机、求解过程可能因配置或设置不当而很慢，也有人询问如何计算空载转速。[C259291829825][C299180185920][C260136525713]这些问题更适合作为后续章节的检查清单，而不是本集已被视频正文证明的结论。
