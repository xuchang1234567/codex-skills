# 硬件工程师入门教程：核心自测卡片

## 基础电路

**问：分压公式什么时候会失效？**  
后级负载不能忽略时，必须把负载并入下臂重新计算。

**问：为什么电阻需要看功率？**  
阻值决定电流与压降，功率决定发热和可靠性；超过额定功率可能漂移、损坏或引发安全问题。

**问：电容和电感各自不能瞬间改变什么？**  
电容电压不能瞬间突变，电感电流不能瞬间突变。

## 二极管和开关

**问：稳压二极管为什么需要限流？**  
反向击穿区必须限制电流，否则会超过功耗和热设计能力。

**问：感性负载断电为什么会产生高压？**  
电感试图维持原有电流，回路被突然切断时会通过高电压寻找电流路径。

**问：MOS 阈值电压能否作为导通电压？**  
不能。阈值只代表小电流开始导通，功率设计要看指定 `Vgs` 下的 `Rds_on`。

**问：高端 NMOS 为什么难驱动？**  
源极电压随负载变化，栅极必须保持比源极高出足够的 `Vgs`。

## 比较器和运放

**问：比较器为什么会颤振？**  
输入在阈值附近受到噪声影响，输出反复翻转；可用迟滞、滤波或更合理的阈值解决。

**问：开漏/开集输出为什么需要上拉？**  
输出晶体管只能主动拉低，释放时需要上拉电阻建立高电平。

**问：虚短和虚断的前提是什么？**  
通常要求运放处于负反馈、线性工作区，并且输入共模和输出范围没有超限。

**问：差分放大器的电阻匹配为什么重要？**  
匹配误差会把共模信号转换成输出误差，降低共模抑制能力。

## 采样和电源

**问：低端采样和高端采样的主要差异是什么？**  
低端简单但会抬高负载地；高端不破坏负载地，但要求放大器处理更高共模电压。

**问：线性稳压器为什么发热？**  
输入输出压差的大部分功率在串联调整器件上消耗，近似为 `(Vin-Vout)×Iout`。

**问：BUCK 关断时为什么需要续流路径？**  
电感电流不能突变，必须通过二极管或同步管继续流动，否则开关节点会出现高压尖峰。

**问：电感饱和有什么后果？**  
电感量下降、电流快速增加，导致开关管、线圈和 PCB 回路过流发热。

**问：为什么电源调试要先限流上电？**  
可以降低接反、短路、器件击穿和焊接错误造成的损坏范围。

## 系统和 PCB

**问：循迹小车为什么要分模块调试？**  
传感器、阈值判断、控制逻辑、电机驱动和电源相互影响，分模块才能快速定位故障。

**问：电机启动会让单片机复位的常见原因是什么？**  
启动/堵转电流造成电源跌落、地线回流干扰、去耦不足或电源与逻辑共用路径不合理。

**问：为什么 DRC 通过仍不能认为 PCB 正确？**  
DRC 主要检查规则，无法完全发现封装引脚错误、回流路径、热设计、机械干涉和可制造性问题。

**问：高 di/dt 回路为什么要短而紧凑？**  
回路面积越大，寄生电感和电磁辐射通常越明显，开关尖峰和噪声也更难控制。
