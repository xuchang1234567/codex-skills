# 直线导轨与直线轴承：选型、精度与安装误差

## 核心区别

直线导轨通常由导轨、滑块和滚动体组成，能够同时承受径向、侧向和一定倾覆力矩，适合机床、自动化平台和需要高刚度、高精度的直线运动。直线轴承通常与圆柱光轴配合，结构简单、成本低、安装方便，适合载荷较小、精度要求一般、空间或预算受限的机构。

## 选型判断

先看载荷方向和力矩。若载荷偏心、存在明显倾覆力矩或需要预紧刚度，优先选择直线导轨；若主要是轻载轴向往复、导向长度短且允许一定间隙，直线轴承更经济。再看速度、加速度、寿命、污染、噪声和维护条件。导轨需要平行度和安装基准，轴承需要光轴直线度、表面硬度和同轴度。

## 精度与误差

系统精度不只由零件标称精度决定，还受到导轨平行度、安装面平面度、滑块预紧、光轴挠曲、轴承间隙和装配应力影响。两条导轨若不平行，会使滑块卡滞或局部负载集中；多根光轴若不平行，会造成直线轴承偏载和摩擦增加。安装时应先松装、校正基准、低速往复检查，再按对角和分段顺序逐步锁紧。

## 工程建议

高精度、高刚度、长寿命和承受力矩的场景优先直线导轨；轻载、低成本、结构简单且可接受较低刚度的场景可选直线轴承。粉尘、切屑和润滑条件恶劣时，应考虑密封、防护和维护便利性，不能只比较采购价格。最终选型应按额定载荷、静安全系数、寿命、安装误差和供应商样本校核。
