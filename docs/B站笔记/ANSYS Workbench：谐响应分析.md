# ANSYS Workbench：谐响应分析

> 覆盖说明：本笔记基于视频字幕、元数据和证据索引整理。字幕密度偏低，归档提示存在中等画面依赖风险；因此这里提炼的是分析逻辑、参数关系和结果判读框架，不替代对视频界面逐项操作的复现。

## 学完你应该获得什么

- 理解谐响应分析回答的工程问题，以及它与模态分析的关系。
- 能建立从材料、模型、接触、网格到模态和谐响应的基本流程。
- 知道模态阶数、有效质量和累计质量分数如何辅助截断判断。
- 能设置频率扫描范围和求解间隔，并定位响应峰值。
- 能区分变形频响、定向位移、总变形、等效应力和相位响应的用途。

## 一句话总论

谐响应分析用于计算线性结构在正弦周期载荷下的稳态受迫振动。实际工作流是“先用模态分析确定结构的动态基线，再在覆盖相关模态的频率范围内施加载荷和阻尼，最后从幅值、应力、相位和峰值频率判断共振风险、动刚度和疲劳隐患”。[1][2]

## 适用场景与前置知识

适用于电机、涡流、风载等近似正弦激励下的线性结构。前置条件包括：已定义材料的密度、弹性模量和泊松比；模型的边界、接触和网格能够代表实际结构；激励幅值、方向和频率单位明确。若存在明显材料非线性、接触状态变化或大变形，不能把本视频流程直接当作充分的物理模型。

## 知识地图

`材料与模型 -> 接触 -> 网格 -> 模态分析 -> 有效质量/累计质量 -> 频率扫描与载荷 -> 谐响应求解 -> 幅值/应力/相位 -> 设计判断`。

视频示例是一个液压系统：先把材料改为铝合金，检查活塞与缸体的接触，再生成网格；初次网格质量过低，通过提高分辨率改善到平均质量约 0.77，随后设置固定支撑并求解模态。[3][4]

## 核心概念卡

### 1. 模态是谐响应的基础

模态分析给出固有频率和振型。视频示例中低阶模态约为 28.976 Hz、29.54 Hz 等，不同阶次表现为 Y 向、X 向振动或扭转。谐响应的危险区通常发生在激励频率接近结构固有频率时，因此不能跳过模态分析直接猜扫描范围。[5]

### 2. 有效质量与累计质量

有效质量表示某阶模态在特定方向上对整体振动的贡献；累计质量分数表示已提取模态对总质量贡献的累积程度。视频采用累计质量分数达到约 90% 以上作为模态截断的实用检查，同时分别查看 X、Y、Z 方向，不能只看一个总百分比。[6][7]

这不是绝对充分条件：若关心局部响应、某一方向激励或高频局部模态，还应检查目标频带内是否包含相关模态，并结合响应收敛进行补充判断。

### 3. 频率范围与求解间隔

谐响应设置的核心是扫描最小频率、最大频率和频率步长。视频示例把模态分析的频率范围与谐响应扫描范围联系起来，并用约 1.5 倍的覆盖关系避免扫描区间遗漏参与计算的模态。[8][9]

求解间隔可采用两阶段策略：先用较粗步长扫描，找到响应峰值区域；再在峰值附近细化步长。步长过大可能错过窄峰，步长过小则会增加计算量。频率单位必须统一，且应检查软件警告中是否提示模态未覆盖当前扫描范围。

### 4. 阻尼

阻尼控制决定共振峰的高度和宽度。阻尼过小会形成尖锐高峰，阻尼过大可能掩盖结构真实的共振敏感性。阻尼比应来自试验、材料/结构资料或合理的工程假设，并进行敏感性分析；不能只因为模型能求解就认为阻尼输入正确。[10]

## 可复用操作流程

1. 在 Workbench 中建立模态系统，并把谐响应系统连接到模态结果。
2. 在工程数据中定义材料，检查密度和弹性参数是否适用于动态分析。
3. 导入几何，核对部件命名、材料分配和单位。
4. 检查自动识别的接触；对活塞/缸体等配合面确认绑定、无分离、无摩擦或粗糙等类型符合真实连接。[3]
5. 生成网格并检查质量；局部几何、接触区和载荷区需要比远场更严格的网格检查。
6. 设置固定支撑等边界，先求解足够阶数的模态，查看固有频率、振型、有效质量和累计质量。
7. 在谐响应中施加正弦载荷：明确位置、方向和幅值，例如视频示例中的 X 向 10 N。
8. 设置频率范围和初始步长，求解后在峰值附近重新细化。
9. 插入结果并统一查看同一峰值频率下的各类响应，避免使用软件默认的最大频率作为结果频点。[11]

## 结果怎么读

- **变形频率响应**：给出位移幅值随频率的曲线，用峰值定位共振频率和最大振幅。[12]
- **定向位移**：只看激励或关心方向的位移分量，可用于评估方向性动态刚度。视频示例在约 27.467 Hz 处得到方向响应峰值。[13]
- **总变形**：合成所有方向位移，用于定位最大振动区域和结构强化位置；示例总变形约 0.495 mm，但是否可接受必须回到间隙、功能和强度要求。[14]
- **等效应力**：将多轴应力转为综合强度指标，用于与材料许用应力、疲劳准则比较。必须在真正的峰值频率下评估，而不是直接读取默认频点；示例峰值处约为 3.04 MPa。[15]
- **相位响应**：描述位移相对于激励的相位差，可辅助判断阻尼、共振附近的响应变化和振动控制策略。[16]

最终设计判断至少包括：是否避开共振工作频率、峰值变形是否影响功能、应力是否满足强度/疲劳要求、阻尼变化是否改变结论，以及网格和模态截断是否已收敛。

## 常见误区

- 把谐响应当作静力分析：它关注稳态幅值和相位，不是简单叠加一个静态力。
- 只求六阶模态而不看有效质量：阶数只是输入数量，方向贡献和累计质量更关键。
- 频率扫描范围太窄：会漏掉参与响应的模态，软件也可能给出覆盖警告。[9]
- 用默认频点读应力：应力结果必须切换到目标峰值频率。[11][15]
- 只看最大云图，不看曲线和相位：云图告诉你“哪里大”，频响曲线才告诉你“在哪个频率大”。

## 自测题

1. 为什么谐响应前要先做模态分析？答：需要知道固有频率和振型，才能识别共振区并确定扫描范围。
2. 有效质量回答什么问题？答：某阶模态对指定方向整体振动的贡献有多大。
3. 为什么要先粗扫再细化？答：兼顾计算成本与窄峰捕捉能力。
4. 总变形和定向位移有何区别？答：前者是位移矢量合成量，后者是指定方向分量。
5. 相位响应可以辅助判断什么？答：阻尼效果、共振变化和振动控制。
