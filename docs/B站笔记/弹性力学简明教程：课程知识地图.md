# 弹性力学简明教程：课程知识地图

## 课程主线

`连续体建模 -> 基本方程与边界条件 -> 直角坐标解析解 -> 极坐标经典问题 -> 有限元离散与计算`。

本课程对应徐芝纶《弹性力学简明教程》第五版的部分章节，视频实际覆盖 47 个分 P；其中 P31 无可用字幕。教材第 5 章及更后续内容不应被误认为已由本视频完整覆盖。

## 模块 1：基本概念与建模（P01-P05）

- 研究对象是连续、线弹性、小变形条件下的弹性体。
- 基本量包括体力、面力、内力、应力、位移和应变。
- 建模必须先明确坐标系、研究对象、外载与约束；基本假定决定方程适用范围。

## 模块 2：平面问题的方程体系（P06-P19）

- 先判断采用平面应力还是平面应变，不能因网格是二维就任意替换。
- 平衡方程、几何方程、本构方程、相容方程与边界条件共同构成边值问题。
- 位移法：求位移后回代应变、应力；应力法：结合平衡和相容方程求应力。
- Airy 应力函数可自动满足无体力平衡，其后仍需满足双调和方程和边界条件。

## 模块 3：直角坐标解析方法（P20-P25）

- 逆解法与半逆解法用于构造满足控制方程的应力场或应力函数，再由边界条件确定其物理问题。
- 典型算例包括矩形梁纯弯曲、简支梁均布荷载与楔形体。
- 从应力恢复位移时，积分常数对应刚体平移和转动，必须由约束或参考点确定。

## 模块 4：极坐标与经典弹性问题（P26-P36）

- 圆环、圆筒、孔洞、隧洞和半平面等问题宜采用极坐标。
- 极坐标平衡与几何方程含 `1/r` 项，不能将笛卡尔公式简单换字母。
- 典型解包括厚圆筒 Lamé 解、孔口应力集中、半平面集中力和分布力解。

## 模块 5：有限元法基础（P37-P47）

- 将连续体离散成节点和单元，以 `u = N delta_e`、`epsilon = B delta_e`、`sigma = D B delta_e` 建立单元关系。
- 单元刚度由 `k_e = integral(B^T D B) dA` 得到，随后装配为全局方程 `K Delta = F`。
- 需要正确处理位移边界、等效结点荷载、网格收敛与后处理。
- 常应变三角形单元内应力为常量；云图中的节点应力往往来自外推或平滑，不能替代物理判断。

## 学习顺序建议

1. 先掌握模块 2 的四套方程和两类边界条件。
2. 用模块 3、4 的解析解训练“假设场 -> 满足方程 -> 施加边界 -> 验算”的闭环。
3. 最后进入模块 5，理解有限元不是脱离弹性力学的黑箱，而是上述方程的离散近似。
