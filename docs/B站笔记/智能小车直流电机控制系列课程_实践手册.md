# 智能小车直流电机控制系列：实践手册

## 目标

最终得到一个可控、可测、可闭环的双电机或麦轮底盘。每一步都先验证当前层，再进入下一层，避免把供电、接线、测速和 PID 问题混在一起。

## 阶段一：硬件上电前

准备：MG520 编码减速电机、TB6612、ESP32、12 V 电机电源、5 V 逻辑电源、万用表、示波器或逻辑分析仪。

检查：

- 根据电机标签确认额定电压、额定转速和减速比。
- 根据实物或厂家资料确认六线线序，不能按颜色想当然。
- 区分 `VM` 电机电源、`VCC` 逻辑电源和 `GND`。
- 确认 `STBY` 的有效电平；TB6612 通常需要拉高。
- 先不接电机，测逻辑电源和地之间的电压。
- 断电测量电源与地之间是否短路。

## 阶段二：单台电机开环

先只接一台电机，验证方向和 PWM：

1. `STBY` 置有效。
2. 方向输入设置为第一种组合。
3. 以较小 PWM 启动，确认电机方向。
4. 改变占空比，确认速度随输出变化。
5. 切换方向，确认没有异常发热和明显短路。
6. 测试停止、滑行和制动三种状态。

如果电机不转，按“电源 -> `STBY` -> 方向输入 -> PWM -> 电机输出 -> 机械负载”顺序排查，不要马上改 PID。

## 阶段三：编码器计数

把 A/B 两路接到支持外部中断的 ESP32 输入。中断函数只做三件事：读取必要电平、更新计数、更新方向。不要在中断里打印大量日志或执行 PID。

先手动转动电机，确认：

- A、B 都有变化；
- 两路相位约为四分之一周期；
- 正转和反转计数符号相反；
- 减速箱输出轴转一圈时，计数与编码器规格和减速比大致一致。

若方向反了，优先检查 A/B 线序或软件正负号，不要直接交换电源线后忽略编码器方向。

## 阶段四：定时器测速

用固定周期定时器读取一段时间内的脉冲增量。基本换算思路是：

`速度 = 脉冲增量 / 采样时间 / 每转脉冲数`

若要得到输出轴速度，还要正确处理减速比和边沿倍频。所有单位必须统一，例如 rad/s、r/min 或脉冲/采样周期不能混用。

建议记录一张表：目标速度、采样周期、脉冲增量、计算速度、实际转速。先确认测速稳定，再进入闭环。

## 阶段五：单电机 PID

控制循环每个周期执行：

1. 读取并清理上一周期编码器计数。
2. 计算实际速度。
3. 计算误差 `e = target - measured`。
4. 更新比例、积分、微分项。
5. 做输出限幅和积分抗饱和。
6. 输出方向与 PWM。

调参顺序：先关闭积分和微分，调比例；再逐步加入积分消除稳态误差；最后视需要加入微分抑制变化。目标速度为零时应有明确的停止策略，避免积分继续累积导致重新启动时猛冲。

## 阶段六：双电机

左右电机必须拥有独立的：

- 编码器计数；
- 目标速度和实际速度；
- PID 积分、上次误差和输出；
- 方向符号与 PWM 引脚。

双电机测试顺序：单独测试左轮、单独测试右轮、同速同向、同速反向、低速同步、负载变化。若两轮 PWM 相同但速度不同，这是正常现象，闭环应通过各自反馈修正。

## 阶段七：M/T 法测速

低速时，固定时间内可能只有少量脉冲，M 法容易产生较大量化误差；M/T 法同时利用脉冲数和脉冲周期，通常更适合跨速度范围测量。比较算法时至少测试：

- 低速恒速；
- 中速恒速；
- 加速过程；
- 减速过程；
- 突加负载。

同时观察响应延迟、速度抖动和 CPU 占用，不能只看某一个时刻的数值。

## 阶段八：麦轮与蓝牙

在四轮麦轮之前，先确认每个轮子的正方向和安装位置。然后分别测试前后、左右平移和原地旋转。蓝牙输入应经过：接收、解析、范围限制、超时判断、速度映射，再进入轮速分解。

必须实现失联保护：超过设定时间没有收到有效指令时，将所有目标速度置零，并执行安全停止。不要让上一条速度命令无限期保持。

## 故障排查表

| 现象 | 优先检查 |
|---|---|
| 电机完全不转 | `VM`、共地、`STBY`、PWM 和方向输入 |
| 只能单向转 | H 桥方向输入、驱动器输出、软件方向逻辑 |
| 电机转但无速度 | 编码器 `VCC/GND`、A/B 线序、中断引脚 |
| 速度方向相反 | A/B 方向判断或电机正负号约定 |
| 低速速度跳动 | 脉冲分辨率、采样周期、M/T 法和滤波 |
| PID 振荡 | 比例过大、采样周期不稳、积分饱和、反馈符号错误 |
| 左右轮不同步 | 两套 PID 是否独立、死区补偿、机械差异 |
| 麦轮方向错乱 | 轮子安装方向、轮速映射矩阵、坐标系约定 |
| 蓝牙断开后继续跑 | 指令超时和急停逻辑缺失 |

## 最终验收

- 断电检查无短路，供电电压符合器件规格。
- 单轮正反转和 PWM 调速正常。
- 两路编码器计数方向正确，速度单位一致。
- 单电机闭环能跟踪目标速度，输出不持续饱和。
- 双电机分别闭环，左右轮状态互不污染。
- 低速测速结果可解释，M/T 法延迟和噪声已记录。
- 蓝牙失联会停止，麦轮四个基本运动方向正确。
