- 能用“定义—原理—用途—例子”的结构回答基础问题。

这类“速成一页纸”最适合考前重建数电知识地图、准备复试口头问答；若目标是期末笔试或完整掌握设计题，还必须补充推导、真值表、时序图和习题训练。[1][2]

## 适用场景与边界

### 适合

- 已学过数电、但知识点遗忘较多，需要快速串联。
- 复试面试前需要建立术语和章节之间的联系。
- 想先判断自己缺哪些模块，再安排后续精学。

### 不适合

- 完全零基础且希望只看这一条视频就会做题。
- 需要系统训练卡诺图、触发器时序、计数器设计和综合题。
- 需要把视频中的每个公式、例题和板书完整抄录下来。

评论反馈也呈现出这个边界：有人认为视频逻辑清楚、适合快速入门或复试；也有人认为它更适合面试，不足以覆盖期末笔试所需的大量习题。[1][2]

## 数电复试知识地图

### 1. 数制与编码

复习目标是能在不同表示之间转换，并说明每种编码的用途：

- 二进制、十进制、十六进制之间的转换。
- 原码、反码、补码的基本思想及符号位处理。
- BCD 码与普通二进制的区别。
- 8421 码的有效范围。

8421 码用四位二进制表示一个十进制数字，只使用 0000 到 1001 表示 0 到 9；1010 到 1111 是无效码。评论中有人把“8421 能表示 16 种组合”和“8421 用来表示十进制数字时只使用 0 到 9”混在一起，复试回答时应区分“所有四位组合”和“合法 BCD 编码”。[3]

### 2. 逻辑代数与逻辑门

至少要能说清楚与、或、非、与非、或非、异或、同或的逻辑关系，并能写出真值表。不要只背门电路名称，要同时掌握：

- 输入为哪些组合时输出为 1。
- 哪些门具有“万能门”性质。
- 如何用与非门或或非门实现其他逻辑。
- 德摩根定律怎样用于逻辑式变换。

面试中常见追问不是“这个门叫什么”，而是“为什么能用它实现其他逻辑”。回答可从恒等式出发，再给出把某些输入并接或反相的构造方式。

### 3. 组合逻辑电路

组合逻辑的输出只由当前输入决定，没有记忆功能。复习顺序建议是：

1. 根据文字描述列输入和输出。
2. 写真值表。
3. 化简逻辑表达式或用卡诺图化简。
4. 选择门电路、译码器、编码器、多路选择器等实现。
5. 检查无关项、竞争冒险和输出是否符合题意。

高频模块包括半加器、全加器、译码器、编码器、数据选择器和比较器。回答模块题时，应说明“输入是什么、输出是什么、用途是什么”，而不是只报出名称。

### 4. 时序逻辑电路

时序逻辑的输出不仅与当前输入有关，还与过去状态有关。核心是触发器和状态变化：

- RS、JK、D、T 触发器的功能和特性。
- 建立时间、保持时间、时钟边沿等基本时序概念。
- 异步置位/复位与同步控制的区别。
- 状态表、状态图和状态方程之间的转换。

面试回答时可用一个固定句式：先说触发器保存状态，再说明时钟到来时输入如何决定下一状态，最后说明它在寄存器或计数器中的用途。

### 5. 寄存器与计数器

寄存器用于暂存和移位数据，计数器按时钟脉冲改变状态。应重点理解：

- 串入串出、串入并出、并入串出、并入并出。
- 异步计数器和同步计数器的区别。
- 模 N 计数器如何确定状态数量。
- 计数器如何通过清零、置数或反馈实现指定模值。
- 74160、74161 等常见计数器型号的功能定位。

评论中出现了“74160 是不是十进制、74161 是不是十六进制”的疑问。复试时不要把型号名称直接等同于进制概念，应查对应器件手册，结合其计数范围、清零/置数方式和级联功能回答。[4]

## 复试口头回答模板

遇到一个陌生数电概念，可以按四步组织：

1. **定义**：它属于组合逻辑、时序逻辑、编码还是存储器件？
2. **工作原理**：输入、状态或时钟怎样影响输出？
3. **典型用途**：它在系统中解决什么问题？
4. **一个对比或限制**：它与相近概念有什么不同，最容易错在哪里？

例如回答“组合逻辑和时序逻辑的区别”：组合逻辑输出只由当前输入决定；时序逻辑还依赖历史状态，通常需要触发器和时钟；前者常见于译码器和加法器，后者常见于寄存器和计数器；判断关键是有没有存储状态。

## 一页纸复习清单

- [ ] 能完成常见数制转换。
- [ ] 能解释补码和 BCD 码。
- [ ] 能写基本逻辑门真值表。
- [ ] 能使用德摩根定律变换逻辑式。
- [ ] 能从真值表得到组合逻辑表达式。
- [ ] 能说明译码器、选择器、加法器的用途。
- [ ] 能比较组合逻辑与时序逻辑。
- [ ] 能说明 D、JK、T 触发器的基本特性。
- [ ] 能解释寄存器和计数器的作用。
- [ ] 能区分异步/同步、清零/置数、有效/无效码。
- [ ] 能不用死记型号，依据数据手册解释 74160/74161。
- [ ] 能用一到两分钟完整回答一个基础问题。

## 复习安排

若只剩一天：先看知识地图，随后口述 10 个定义，再做数制转换、真值表和触发器状态题。

若还有一周：前两天补数制、逻辑代数和组合逻辑；第三至四天补触发器、寄存器和计数器；第五天开始做综合题和时序图；最后两天用问答形式模拟复试。

若目标是笔试：必须增加习题量，尤其是卡诺图、组合逻辑设计、触发器波形、计数器状态和时序分析。速成内容可以负责搭框架，但不能替代计算过程。

## 坑点与反例

- 把 BCD 码的 16 种四位组合误说成 16 个合法十进制数字。
- 把器件型号中的数字直接当成进制。
- 只背逻辑门名称，不会写真值表。
- 把组合逻辑和时序逻辑都当成“输入到输出”的即时映射。
- 只会说触发器名称，不会解释时钟和状态。
- 只背计数器结论，不会说明清零、置数和级联。
- 把“速成”理解成“无需基础和练习”。
