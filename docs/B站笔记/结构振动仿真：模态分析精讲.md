# 结构振动仿真：模态分析精讲

## 学完你应该获得什么

- 理解模态分析求解的物理量和数学问题。
- 能解释固有频率、模态振型、阻尼和有效质量。
- 区分自由模态、约束模态和预应力模态。
- 能判断模态结果能否用于共振规避和后续动力学分析。
- 能在 Workbench 中建立一个基本模态分析并检查结果。

## 一句话总论

模态分析是在给定质量矩阵和刚度矩阵下求解结构的固有频率与振型；频率告诉你结构容易在哪些频率附近振动，振型告诉你如何变形，约束、连接、预应力和材料参数会通过改变矩阵而改变结果。[1][2]

## 模态的数学本质

一般动力学方程为：

`M a + C v + K x = F`

无阻尼自由振动时，外力和阻尼忽略，得到：

`M a + K x = 0`

设结构按正弦形式振动 `x=phi sin(omega t)`，代入后得到广义特征值问题：

`(K - omega^2 M) phi = 0`

非零振型存在的条件是 `det(K-omega^2M)=0`。特征值 `lambda=omega^2` 给出固有圆频率，`f=omega/(2pi)` 给出普通频率；特征向量 `phi` 是对应的模态振型。[3][4]

振型是无量纲的比例形状。软件显示的变形数值通常经过质量归一化或最大值归一化，并不是实际工作载荷下的位移。因此不能把模态云图的“变形幅值”直接当成结构实际振幅；应关注节点相对运动、变形方向和节点之间的相位关系。[5]

## 模态分析能解决什么

第一，比较激励频率与固有频率，避开共振区。第二，通过振型识别结构刚度弱点和局部柔性区域，指导加筋、改厚度、改连接或改变质量分布。第三，为响应谱、随机振动、谐响应和瞬态动力学提供模态基础。第四，通过有效质量判断计算的模态阶数是否覆盖了关心方向的主要惯性贡献。[1][2]

模态结果不能单独给出疲劳寿命、实际响应幅值或阻尼后的峰值。它回答的是“结构自身倾向如何振动”，而不是“在某个具体载荷下最终振多大”。后续响应分析仍需载荷谱、阻尼、激励方向和边界工况。[6]

## 三类模态

### 自由模态

结构悬浮在空间中，没有外部约束。三维实体通常有 6 个刚体自由度，对应前六阶频率为零或接近零；这是刚体平移和转动，不是结构弹性变形。自由模态可用于检查模型连接、质量和刚度是否正确，也可与悬挂试验对应。[7]

### 约束模态

给结构施加实际支承和连接约束后求模态。约束会改变刚度矩阵，通常也会移除刚体模态，因此前六阶不再为零。约束方向、接触和共节点连接必须符合真实安装，否则固有频率和振型会系统偏差。[8]

### 预应力模态

先通过静力或稳态分析得到预应力，再把预应力状态传递给模态分析。拉紧的琴弦频率比松弛琴弦高，是预应力改变等效刚度的直观例子。预应力模态常用于受拉构件、旋转部件、制动盘和承压结构；应确认预应力工况、几何非线性和载荷传递设置一致。[9][10]

## 有效质量与模态截断

有效质量表示某个模态在指定平移或转动方向上能够参与整体惯性运动的程度。累计有效质量与总质量的比值可以帮助判断模态阶数是否足够。不同方向应分别查看，不能只看一个总和；如果关心 Y 方向而 Y 向累计有效质量仍低，就应增加模态阶数。[11]

低阶模态通常更容易被实际激励达到，也更容易产生共振，因此工程解释常先关注低阶频率。但高阶模态不能一概忽略：局部冲击、高频激励、短时载荷和局部结构响应可能需要更高阶模态覆盖。[12]

## Workbench 实操流程

1. 保存项目并导入几何，检查零部件之间的连接关系。
2. 判断是自由、约束还是预应力模态，先建立正确的分析系统。
3. 定义密度、弹性模量和泊松比；模态求解至少需要质量和刚度相关材料参数。[13]
4. 检查共节点、接触、连接释放和相对运动自由度；错误连接会直接改变刚度矩阵。
5. 设置网格并进行必要的局部加密，确保关键结构和连接被合理离散。
6. 约束模态中选择实际支承面和方向；预应力模态则将静力/稳态系统的结果传递到模态系统。
7. 设置提取阶数，求解固有频率和振型。
8. 查看频率列表、振型动画、有效质量和求解信息；用刚体模态、对称性和数量级检查结果。

## 阻尼与模态结果

经典无阻尼模态的特征值为实数，阻尼会改变动力学特征值的表达，可能出现复数或实部/虚部信息。工程软件中的阻尼通常以全局阻尼、材料阻尼、恒定阻尼比或质量/刚度比例阻尼输入。阻尼主要影响响应峰值和衰减，不应误解为改变无阻尼模态振型的唯一因素。[14][15]

对结构大变形或明显非线性问题，线性模态只是局部近似。预应力、接触状态和几何刚化/软化会改变切线刚度；应根据工况选择非线性静力预加载、线性化模态或更完整的瞬态分析。[10][16]

## 常见误区

- 把归一化振型的显示变形当实际位移。
- 自由模态出现六个零频率就误判为求解失败。
- 连接关系、接触释放或支撑方向设置错误，却只调整模态阶数。
- 只看第一阶频率，不看振型方向和有效质量。
- 用无预应力模态解释强预紧、旋转或承压结构。
- 认为模态分析已经包含真实载荷响应和阻尼峰值。
- 只看云图，不检查材料密度、刚度、单位和反映工况的边界条件。

## 自测题

1. 模态分析求解的特征值和特征向量分别代表什么？答：`omega²` 与模态振型。
2. 为什么自由模态前六阶接近零？答：三维结构有六个刚体自由度。
3. 约束模态为什么会改变固有频率？答：约束改变结构刚度矩阵并移除刚体运动。
4. 振型图的变形幅值有实际意义吗？答：通常没有，主要看相对形状和方向。
5. 有效质量的用途是什么？答：判断目标方向的模态参与是否充分。
6. 预应力模态为什么需要先做静力/稳态分析？答：预应力会改变等效刚度。
