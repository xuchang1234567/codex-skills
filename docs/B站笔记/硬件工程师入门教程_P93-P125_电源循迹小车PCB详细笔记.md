# 硬件工程师入门教程 P93-P125：电源、循迹小车与 PCB

## P93-P103：直流稳压与开关电源

### 线性稳压

直流稳压电源通过反馈检测输出电压并调节功率器件，使输出尽量稳定。线性稳压结构简单、噪声较低，但输入输出压差会转化为热量，设计时必须检查压差、负载电流、功耗和散热。

反馈型电源能根据输出变化自动修正，非反馈型结构更简单但负载和输入变化时输出稳定性较差。稳压器外围电容、接地、输入瞬态和输出负载都会影响稳定性，不能只看标称输出电压。

### BUCK

BUCK 通过开关管、储能电感、续流路径和输出电容把较高输入电压转换为较低输出电压。开关导通时向电感和负载传递能量，关断时电感通过续流路径继续供电。控制器通过占空比和反馈调节输出。

设计检查包括：开关管耐压和电流、二极管或同步管、 电感饱和电流、输出纹波、输入输出电容 ESR、开关频率和热损耗。高 di/dt 回路应尽量紧凑，功率地和信号地需要合理规划。

### 续流二极管

电感电流不能突变，开关关断时必须有连续路径。续流二极管为电感能量提供通路，防止开关节点产生过高感生电压。二极管的反向耐压、正向电流、恢复特性和损耗都要纳入选择。

### BOOST 与电荷泵

BOOST 通过电感储能和开关切换，把输入电压升高；电感电流路径、二极管方向、输出电容和开关节点极性必须画清。BOOST 的开关管、二极管和输出端可能承受高于输入的电压，耐压和布局不能按输入电压简单选择。

电感饱和会使电感量下降，电流快速增大，可能导致开关管和电感过热。选型要比较峰值电流、温升、直流电阻和实际工作频率，而不能只看标称电感值。

电荷泵利用电容周期性充放电和开关网络实现升压或负压，不需要电感，但输出电流、纹波、效率和负载能力有限。它适合轻负载或辅助电源，不应无条件替代电感型 DC-DC。

## P104-P115：纯硬件循迹小车

### 系统分层

纯硬件循迹小车可以拆成：黑线检测、比较/阈值、控制逻辑、电机驱动、电源和机械负载。调试时要按信号流向逐层验证，不要一上电就只看小车是否运动。

### 黑线检测与控制逻辑

传感器根据反射率差异输出信号，比较器或阈值电路把模拟变化转换为逻辑状态。多个传感器的组合表示车辆相对黑线的位置，控制逻辑再决定左右电机的方向和速度。

传感器阈值、环境光、地面颜色、安装高度和机械偏差都会影响可靠性。比较器颤振或阈值过于敏感时，需要迟滞、滤波或重新规划传感器布局。

### 电机驱动

电机启动和堵转电流可能远高于正常运行电流，驱动管和电源路径必须按峰值电流、压降和热量设计。电机端的储能电容、驱动回路和地线布局会影响电源噪声和逻辑稳定性。

### 电源设计

电池、稳压、逻辑电源和电机功率电源要明确分工。启动、转向和堵转时的电源瞬态可能造成单片机复位或比较器误动作。应合理放置储能电容、去耦电容，并验证电源在最大负载下的电压跌落。

### 系统验收

1. 单独验证每个黑线传感器输出。
2. 固定传感器输入，验证控制逻辑真值表。
3. 不接电机时检查驱动输出逻辑。
4. 接入电机后低速测试正反转和停止。
5. 在不同光照、速度和弯道下测试循迹稳定性。
6. 测试电机启动、急停和堵转时是否导致逻辑复位。

## P116-P125：原理图与 PCB

### 原理图封装

单部件和多部件封装需要明确引脚编号、引脚类型、单位关系和电气连接。原理图符号好看不等于封装正确；必须核对数据表、引脚编号和实际器件方向。

### PCB 封装

PCB 封装要同时包含焊盘、丝印、阻焊、装配方向和机械尺寸。连接器、开关、电机接口和大电容尤其要确认插拔方向、孔径、间距和高度。

### 从原理图到 PCB

导入 PCB 后先检查网络是否完整，再处理板框、安装孔、关键器件定位和布局。不要一开始就自动布线；先确定电源入口、电机接口、开关电源功率回路、模拟采样区和数字逻辑区。

### 布局与铺铜

大电流、快速开关和敏感模拟信号应分别规划回路。开关电源的高 di/dt 环路要短而紧凑，采样线尽量采用合理的 Kelvin 引出，运放反馈线避免绕过噪声源。铺铜不是“铺得越多越好”，要检查回流路径、分割、热扩散和信号参考平面。

### PCB 检查

设计规则检查只能发现一部分问题，不能替代工程复核。最终要检查：

- 封装引脚与数据表一致；
- 极性器件方向正确；
- 电源和地网络没有遗漏；
- 线宽、过孔和铜厚满足电流；
- 高压节点间距足够；
- 关键器件可焊接、可测量、可更换；
- 连接器和机械边框不会干涉；
- 丝印不会覆盖焊盘或遮挡方向标识；
- 生产文件、钻孔文件和装配信息完整。

## 全课程实战项目

建议最终完成一块纯硬件循迹小车控制板：先画电源、传感器、比较器、MOS/三极管驱动和电机接口原理图，再建立封装、定位功率器件、规划采样与逻辑区域，最后完成铺铜和 DRC。首版板子应预留测试点和调试跳线，不要为了追求紧凑而牺牲可测性。

## 完成后的工程能力

学完 P1-P125 后，目标不是熟记所有电路，而是能从需求出发完成：器件选型、参数计算、保护设计、反馈分析、模块组合、原理图、PCB 封装、布局、铺铜和验证。每个设计都要留下输入条件、计算依据、最大应力、失效模式和测试方法。
