# 什么是模态：从特征根到系统稳定性

## 学完你应该获得什么

- 能用“输出响应的最终走势”理解模态，而不是只背术语。
- 能从微分方程的特征根判断模态形式。
- 能从传递函数极点判断模态和系统稳定性。
- 能区分不同特征根、重根和共轭复根对应的响应。
- 能解释分子/零点影响模态比重，但不决定基本模态。

## 一句话总论

模态是系统输出中决定长期走势的指数、指数乘多项式或指数乘正弦/余弦项；它由微分方程的特征根或等价的传递函数极点决定，而极点位于左半平面、虚轴或右半平面分别对应衰减、临界和发散趋势。[1][2]

## 先从一阶系统看模态

对一阶衰减方程做拉氏变换，输出通常可写成：

`X(s)=A/(s+a)`

反变换得到：

`x(t)=A e^(-at)`

其中 `A` 由初始条件或输入决定，只改变幅值；指数项 `e^(-at)` 决定随时间衰减的走势，因此它就是模态。`a>0` 时响应衰减，`a<0` 时响应发散。[1]

这给出一个实用定义：模态不是输出中的任意一项，而是决定输出长期形状的动态项。常数系数会改变模态所占比重，但不会改变指数的衰减率或发散率。

## 多个不同特征根

若传递函数分母可以分解为 `(s+a)(s+b)`，部分分式展开后会得到：

`x(t)=A1 e^(-at)+A2 e^(-bt)`

两个指数项就是两个模态。初始条件、输入和分子决定 `A1、A2`，即每个模态参与响应的比重；分母的根决定模态本身。即使两个系统分子不同，只要分母相同，它们就有相同的模态集合。[3][4]

工程上常把衰减较慢的模态称为主导模态，因为它在长期响应中占据更大比例。一个快速衰减模态可能初始幅值很大，但很快消失；一个慢模态即使系数较小，也可能决定后期响应。

## 重根对应的多项式因子

当特征根重复时，响应不再只是单纯的指数。例如 `(s+a)^2` 对应：

`(A1 + A2 t)e^(-at)`

重根次数越高，指数前的多项式次数越高。三重根可能出现 `t² e^(-at)`。因此判断模态时不能只看到 `e^(-at)`，还要保留前面的 `t、t²` 等因子；它们会影响响应的瞬态形状。[5]

重根不一定意味着系统不稳定。若 `a>0`，即使乘上有限阶多项式，指数衰减最终仍会压过多项式增长；若根位于右半平面，指数发散则会放大重根造成的多项式项。

## 共轭复根对应振荡

若特征根为：

`s=-alpha ± j omega`

时域响应具有：

`e^(-alpha t) [A cos(omega t)+B sin(omega t)]`

指数项决定包络是衰减还是发散，正弦/余弦项决定振荡频率。`alpha>0` 时是衰减振荡，`alpha=0` 时是等幅振荡，`alpha<0` 时是振幅不断增大的发散振荡。[6][7]

这也是机械振动中“频率”和“稳定性”可以分开理解的原因：复根的虚部告诉你振荡快慢，实部告诉你振荡包络是否衰减。

## 微分方程与传递函数的等价性

对零初始条件下的线性微分方程做拉氏变换，可得到：

`G(s)=X(s)/R(s)=N(s)/D(s)`

微分方程的特征根来自齐次方程特征多项式 `D(s)=0`；传递函数的极点也正是分母 `D(s)` 为零的位置。因此两者描述的是同一组系统模态，只是一个从时域微分方程出发，一个从复频域传递函数出发。[8][9]

传递函数分母的根决定模态和稳定性，分子零点主要改变各模态的激励权重和响应曲线形状。零点可能削弱某个模态的可见程度，甚至在特定输入输出通道中抵消它，但不能把系统本身的极点集合误认为零点。

## 极点位置与稳定性

- **左半平面**：极点实部为负，模态含衰减指数，响应趋于稳定。
- **虚轴**：实部为零，理想无阻尼情况下可能持续振荡；重根落在虚轴上通常会带来不稳定增长。
- **右半平面**：极点实部为正，模态含增长指数，响应发散，系统不稳定。

对共轭复根，只需观察实部即可判断包络稳定性；虚部决定振荡频率。这个判断不依赖具体初始幅值，因此可以在没有完整时域仿真的情况下先做稳定性预判。[10]

## 实用分析流程

1. 写出系统微分方程或传递函数。
2. 取齐次特征方程，或直接令传递函数分母为零。
3. 求全部特征根/极点，检查是否有重根和复根。
4. 根据根的形式写出模态项：指数、多项式乘指数，或指数乘正弦/余弦。
5. 根据实部判断衰减、临界或发散，根据虚部判断振荡频率。
6. 再看分子和初始条件，判断各模态参与的比重与输出通道可见性。

## 常见误区

- 把模态理解成某一张振型图，忽略控制理论中的动态项。
- 只看分子或初始条件，不求分母根。
- 遇到重根仍只写一个指数项，漏掉 `t e^(-at)`。
- 看到复根只关注振荡频率，不看实部的衰减/发散。
- 把传递函数零点当成决定系统稳定性的量。
- 只根据单次有限时长仿真判断稳定性，忽略极点位置。

## 自测题

1. 一阶系统 `1/(s+2)` 的模态是什么？答：`e^(-2t)`。
2. 分母相同、分子不同的两个系统模态是否相同？答：相同，但模态权重可能不同。
3. 重根 `(s+a)^2` 对应什么额外因子？答：`t e^(-at)`。
4. 共轭复根的实部和虚部分别表示什么？答：实部决定包络衰减/发散，虚部决定振荡频率。
5. 极点位于右半平面意味着什么？答：存在发散模态，系统不稳定。

## 快速判别例题

### 例 1：两个不同实根

若 `G(s)=1/[(s+2)(s+5)]`，极点为 `-2、-5`，响应包含 `e^(-2t)` 和 `e^(-5t)`。前者衰减更慢，通常是长期响应的主导模态；分子若改变，只会改变两个模态的权重。

### 例 2：二重根

若分母为 `(s+3)^2`，响应形式包含 `(A+Bt)e^(-3t)`。虽然有 `t` 因子，因指数仍衰减，系统最终稳定；不能把“有 t”误判为不稳定。

### 例 3：稳定振荡

若极点为 `-0.5±j4`，响应是衰减振荡，包络由 `e^(-0.5t)` 控制，振荡角频率约为 4 rad/s。若实部变成 `+0.5`，振荡包络改为发散。

### 例 4：极点和零点

若 `G(s)=(s+1)/[(s+2)(s+5)]`，零点 `-1` 会改变响应形状和模态比重，但极点仍为 `-2、-5`，因此系统基本模态和稳定性不由零点决定。

## 模态形式速查

| 特征根/极点 | 时域模态 | 稳定性判断 |
|---|---|---|
| 单一实根 `-a` | `e^(-at)` | `a>0` 时衰减 |
| 不同实根 | 多个指数项叠加 | 最慢衰减项常主导长期响应 |
| 重根 `-a` | `t^k e^(-at)` | 指数衰减仍可压过多项式增长 |
| 复根 `-a±jω` | `e^(-at)sin/cos(ωt)` | `a>0` 时衰减振荡 |
| 虚轴根 `±jω` | 等幅正弦/余弦 | 理想无阻尼临界状态 |
| 右半平面根 | `e^(at)` 或增长振荡 | 发散，不稳定 |

## 与结构模态的联系

在线性多自由度结构中，模态分析得到的是广义特征值和振型；控制理论中的极点分析得到的是系统动态模态。两者都在寻找系统的自然动态成分，只是结构问题通常从 `Kφ=ω²Mφ` 出发，控制问题常从微分方程或传递函数分母出发。连接两者的关键是质量、刚度、阻尼和状态空间模型。
## 模态分析实践工作流

1. 明确输入输出：确定激励、观测点、关心方向和频率范围。
2. 建立线性模型：检查质量、刚度、边界和连接是否与实际工况一致。
3. 求取极点或特征值：区分实根、重根和共轭复根。
4. 写出对应模态项：保留指数、时间多项式及正弦/余弦因子。
5. 判断稳定性：检查每个极点实部，不能只看某一次有限时长响应。
6. 判断可观测性：结合输入输出分子、零点和初始条件，分析哪些模态会出现在目标通道。
7. 与结构仿真对照：将控制模型的动态模态与有限元固有频率、振型和有效质量对应起来。
