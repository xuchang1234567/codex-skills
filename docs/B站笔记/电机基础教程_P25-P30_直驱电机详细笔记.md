# 电机基础教程 P25-P30：直驱电机、直线电机与 DD 马达

## P25-P26：直驱电机的含义与优势

直驱是让电机直接连接负载，减少或取消减速机、皮带、齿轮等中间传动。它可以减少回差、磨损、噪声和传动误差，提高响应速度和结构紧凑性；但电机必须直接承担负载所需转矩，通常对电机尺寸、低速转矩、控制器、散热和机械刚度提出更高要求。

直驱并不意味着所有指标都更好。没有减速机构的转矩放大后，电机侧需要更大输出；负载惯量也更直接地作用于电机，机械振动和负载扰动可能更明显。选择直驱前要比较传动方案的成本、精度、速度、维护、刚度和安装空间。

## P27：三种常用直驱电机

直驱系统可能使用旋转型力矩电机、直线电机或 DD 马达等结构。旋转型直驱直接输出低速大转矩；直线电机把电磁力直接转换为直线运动；DD 马达适合转台、分度和高精度旋转平台。

不同结构的选型参数不同，但共同要关注：连续/峰值力或转矩、速度、加速度、负载惯量、位置反馈、冷却、安装刚度、运动范围和驱动器匹配。

## P28：直驱电机选型依据

选型流程仍然是负载分析、速度分析、加速度分析和热分析，但不能套用带减速机构的简单公式。需要直接计算负载侧所需力/转矩，并检查电机在目标速度下是否还能提供足够的连续和峰值能力。

对直线电机，要考虑移动部件质量、导轨摩擦、垂直方向重力、加速度、推力常数、最高速度和行程。对旋转 DD 马达，要考虑转台负载惯量、外部偏载、连续转矩、峰值转矩、角加速度和编码器分辨率。

## P29：直线电机选型例题

直线运动选型可按以下步骤进行：

1. 确定移动质量、目标速度和加速度。
2. 计算加速力、摩擦力、重力分量和外部工艺力。
3. 把各项力相加并加入安全系数。
4. 根据推力常数和电流计算所需电流。
5. 检查连续推力、峰值推力、速度-推力曲线和热。
6. 核对导轨、编码器、驱动器和安装刚度。

水平和垂直直线运动不能混用同一计算：垂直轴需要考虑重力、制动、断电下滑和保持方案；水平轴更关注摩擦、加速度和外部负载。

## P30：DD 马达选型例题

DD 马达直接驱动转盘或旋转机构。计算时需要确定负载惯量、角速度、角加速度、外部转矩、摩擦、连续运行时间和定位/速度要求。加速阶段需要额外转矩，连续运行则要检查热平衡和持续转矩。

若转台存在偏心负载、周期性扰动或工艺切削力，还要检查机械刚度、轴承、结构共振和编码器安装。高分辨率编码器不能自动消除机械回差、结构变形和安装误差。

## 直驱选型验收

- [ ] 已明确直驱取消了哪些传动环节。
- [ ] 已重新计算负载侧力/转矩，没有沿用减速机构公式。
- [ ] 已区分连续能力和峰值能力。
- [ ] 已计算惯量、加速度和外部扰动。
- [ ] 已核对驱动器、电流、编码器和冷却。
- [ ] 已检查机械刚度、轴承、共振和安装误差。
- [ ] 垂直轴已考虑重力、制动和断电安全。
- [ ] 已用工况曲线验证速度范围和热。

## 全课程复习框架

从步进到伺服、驱动电机和直驱，选型逻辑逐步从“能不能转”提升到“能否在完整工况下稳定、精确、可靠地运动”。最终判断应综合负载、速度、加速度、转矩/推力、惯量、反馈、驱动器、热、机械和安全。
