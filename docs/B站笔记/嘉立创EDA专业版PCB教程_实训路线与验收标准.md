# 嘉立创 EDA 专业版 PCB 教程：实训路线与验收标准

## 项目一：51 单片机核心板

### 阶段 1：需求与选型

确定输入电压、稳压输出、单片机型号、晶振、复位、下载接口、排针、LED、按键和扩展功能。每个器件记录型号、封装、供电范围、关键额定值和数据手册。

验收：能说明每个器件为什么选择，能画出电源和最小系统框图。

### 阶段 2：原理图

按电源、核心、时钟、复位、下载和外围功能分块绘制。完成后检查网络标签、引脚编号、未连接引脚、电源脚、上拉/下拉和极性。

验收：ERC 错误已解释；从任意一个接口都能沿网络追踪到核心器件或电源；原理图不依赖“看起来连上了”。

### 阶段 3：封装

逐个核对芯片、排针、晶振、电容、电阻、按键和接口封装。检查焊盘编号、Pin 1、器件方向、孔径、间距、丝印、板边和装配高度。

验收：符号引脚与封装焊盘一一对应；极性器件在丝印上有明确方向；连接器插拔方向符合机械结构。

### 阶段 4：布局与布线

先放板框、安装孔、连接器和核心芯片，再放电源、晶振、复位、下载接口和外围器件。电源和地优先，时钟/复位保持短而紧凑，普通 GPIO 再完成布线。

验收：所有网络连通；电源线宽满足电流；去耦靠近芯片；测试点可探测；丝印不压焊盘。

### 阶段 5：输出与测试

完成 DRC、丝印和生产文件导出。准备限流上电、稳压输出、复位、时钟、下载和 GPIO 测试。

交付物：原理图、PCB、BOM、Gerber、钻孔文件、坐标文件、装配图、测试记录和版本说明。

## 项目二：USB 拓展坞

### 阶段 1：接口与电源

明确上游/下游端口、速率、供电能力、限流、ESD、连接器方向和外壳机械约束。确定控制芯片、电源器件、保护器件和晶振/配置元件。

验收：能画出接口、电源、保护和控制芯片之间的信号流与功率流。

### 阶段 2：差分信号原理图

给差分网络统一命名，核对芯片引脚和连接器引脚。确认 USB 信号、电源、地、保护器件和未使用引脚处理方式。

验收：每对差分线有清晰起点和终点；没有把差分信号当普通 GPIO 处理；保护器件位置和回流路径明确。

### 阶段 3：PCB 布局布线

先放连接器和板框，再放 ESD 器件、核心芯片、电源、晶振和去耦。差分线保持连续参考平面、相近环境和合理长度；减少过孔、跨分割和靠近开关噪声源。

验收：差分规则、线宽、间距、阻抗目标和板厂能力一致；连接器插拔空间和外壳装配通过检查。

### 阶段 4：生产与首板测试

执行 DRC，核对 BOM 和生产文件版本。首板测试覆盖上电电流、端口电压、设备枚举、数据传输、多个端口带载、插拔和保护行为。

## 通用验收标准

- [ ] 原理图、PCB 和 BOM 版本一致。
- [ ] 符号、封装和数据手册引脚一致。
- [ ] 电源、地、保护和测试点明确。
- [ ] 功率、模拟、高速和普通信号分区合理。
- [ ] 线宽、孔径、间距、阻抗和铜厚满足要求。
- [ ] DRC/ERC 的每个例外都有理由。
- [ ] 机械结构、连接器方向和器件高度通过检查。
- [ ] 生产文件可被板厂和装配方直接使用。
- [ ] 首板测试有输入条件、仪器设置、波形、温升和结论。
