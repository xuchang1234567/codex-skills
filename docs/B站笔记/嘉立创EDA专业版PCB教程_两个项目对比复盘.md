# 嘉立创 EDA 专业版 PCB 教程：两个项目对比复盘

## 总体对比

| 项目 | 51 单片机核心板 | USB 拓展坞 |
|---|---|---|
| 主要训练 | 最小系统与完整 PCB 流程 | 接口、差分信号和电源保护 |
| 难点 | 电源、复位、晶振、外围功能 | USB 信号、ESD、连接器和机械约束 |
| 适合阶段 | PCB 初学项目 | 入门后提升项目 |
| 主要风险 | 封装引脚、供电、下载接口 | 差分布线、供电冲突、ESD 和插拔 |
| 首板验证 | 上电、复位、下载、GPIO | 枚举、数据传输、多端口带载 |

## 核心板复盘

核心板适合建立第一套完整工作流：从需求、器件选型、最小系统原理图，到符号/封装、布局、布线、DRC 和生产文件。复盘重点是能否解释每个电阻、电容、晶振、复位和接口的作用，而不是只会跟着视频点击。

首板应预留电源测试点、复位测试点、下载接口和关键 GPIO。第一次打板应优先保证可测、可改、可焊，避免过度追求尺寸和外观。

## USB 拓展坞复盘

USB 项目在核心板基础上加入接口保护、差分线、连接器机械约束和更严格的电源问题。复盘重点是：差分对是否保持连续参考平面，ESD 器件是否靠近接口，USB 供电与调试器供电是否可能反向，连接器方向和焊接空间是否真实可用。

首板测试不能只看“设备亮不亮”，还应测试枚举、数据传输、多个接口同时工作、插拔、异常供电和静电保护。出现问题时先分离供电、信号、焊接和软件枚举因素。

## 推荐实践顺序

1. 先完成核心板原理图和封装核对。
2. 打样并完成限流上电、下载、复位和 GPIO 测试。
3. 根据首板问题改进电源、丝印、测试点和布局。
4. 再开始 USB 拓展坞，先确认接口、芯片和保护方案。
5. 先验证低速/基础连接，再验证带载和数据传输。

## 复盘问题

- 哪些错误 DRC 能发现，哪些必须人工或实测发现？
- 哪些器件位置是由机械结构决定的？
- 哪些网络需要优先布线？为什么？
- 电源、信号和地回流在两个项目中有什么不同？
- 如果首板不能工作，最小化问题范围的第一步是什么？
