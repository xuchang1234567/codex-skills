# 硬件工程师入门教程 P1-P28：基础电路、电容、电感与滤波

## P1-P8：电路安全、电阻与欧姆定律

硬件分析先从安全开始：确认电源电压、极性、负载额定值和可能的短路路径，再接入示波器、万用表或实验电源。欧姆定律 `V = IR` 连接电压、电流和电阻；功率可写成 `P = VI`，也可结合欧姆定律写成 `I^2R` 或 `V^2/R`。计算结果必须和器件额定功率、电流、耐压以及散热条件比较。

电阻识别不只是读阻值，还要识别精度、功率和实际工作温升。限流电阻通过压降限制电流，分流电阻改变电流分配，分压电阻在高阻输入条件下提供比例电压。分压计算时不能忽略后级负载，否则实际输出会低于空载公式结果。设计电阻网络时同时检查功耗和输入输出阻抗。[P02@00:00-15:47][P03@00:00-17:29][P06@00:00-19:58][P08@00:00-12:14]

## P9-P15：电容、充放电与稳压滤波

电容储存电荷，关键动态特征是两端电压不能瞬间突变。电阻和电容组成的 RC 网络中，时间常数 `τ = RC` 决定充放电速度。上电延时可以利用电容电压逐渐变化触发阈值；断电延时则依赖储存电荷通过指定路径释放。实际设计必须考虑漏电、负载电流、阈值误差和电容容差。

电容可以用于去耦、滤波和稳压辅助，但不能把“大电容”简单当成万能稳压器。高频噪声、负载瞬态和电源线寄生参数决定电容放置位置、类型和数量。电容两端电压的动态变化要和后级阈值、复位、使能或比较器输入结合分析。[P09@00:00-14:29][P11@00:00-19:15][P13@00:00-17:55]

容抗随频率升高而降低，理想电容的容抗大小与频率和电容值有关。低通滤波通常让低频变化通过、衰减高频变化；设计时要说明信号源阻抗、负载阻抗、截止频率和幅频/相频变化。滤波不是只看一个电阻和一个电容的数值，还要看它是否被后级加载。[P16@00:00-06:02][P17-P18]

## P17-P20：低通与高通

低通电路的本质是利用器件对不同频率的阻抗差异，使低频和高频走不同的电流路径。高通电路则抑制低频或直流分量，让变化较快的成分通过。判断滤波器时，先画低频等效电路和高频等效电路，再检查输入、输出和地之间的信号路径。

常见误区：把截止频率当成“截止后完全没有信号”；忽略负载改变分压；只按公式算 RC 而不验证源阻抗；忘记电容初始电压和瞬态过程。实际调试应同时观察时域波形和频率响应。[P17-P20]

## P21-P28：电感与 LC 滤波

电感储存磁场能量，核心特征是限制电流的变化速度，而不是把电流最大值永久限制在某个值。电感电流不能瞬间突变，回路突然断开时可能产生较高感生电压，因此必须规划续流、钳位或吸收路径。

电感的感抗随频率升高而增大，因此可与电容组成低通或高通网络。LC 滤波具有较强的频率选择能力，但也可能产生谐振、过冲和自激。设计时要考虑电感直流电阻、饱和电流、额定电流、电容 ESR、负载阻抗和阻尼。[P21-P28]

电感回路中的电阻突然增大，磁场能量需要寻找新的释放路径；若路径不合理，电压会快速升高，可能击穿开关器件。实际电路中常见的续流二极管、TVS、RC 吸收和有源钳位，本质都是为感性能量提供受控路径。

## 基础计算与验收

1. 给定电源、电阻和负载，计算工作电流与电阻功耗。
2. 给定 RC 延时目标，选择时间常数并检查阈值误差。
3. 对一个滤波器分别画低频和高频等效电路。
4. 计算电容容抗和电感感抗，判断它们在目标频率下更像开路还是短路。
5. 对电感断电回路画出能量释放路径，指出最危险节点。
6. 检查滤波器是否被信号源或负载阻抗改变。
