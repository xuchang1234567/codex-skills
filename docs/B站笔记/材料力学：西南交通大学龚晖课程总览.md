# 材料力学：西南交通大学龚晖课程总览

## 课程主线

材料力学研究杆、梁、轴和桁架等构件在载荷下的内力、应力、变形与强度。课程的统一流程是：画受力图并求支反力 -> 截面法求内力 -> 绘制内力图 -> 由内力求应力 -> 用强度/刚度/稳定条件验算 -> 必要时处理超静定、温度和装配约束。[1]

## 学习地图

`基本假设 -> 轴向拉压 -> 扭转 -> 梁内力与弯曲 -> 梁位移 -> 超静定与温度装配应力`

### 1. 基础与轴向拉压

先明确材料力学的基本假设和构件理想化。轴向拉压问题用截面法求轴力 `N`，再用 `σ=N/A` 得到平均正应力。若截面、材料或载荷沿杆长变化，应分段计算并绘制轴力图。强度条件把应力与许用应力比较，刚度条件则检查伸长量。[2]

拉压变形遵循胡克定律，轴向变形可按 `δ=∫N/(EA) dx` 分段叠加。桁架结点位移可以利用杆件伸长和几何关系求出；横向变形由泊松比决定。材料试验部分要区分弹性阶段、屈服、强化、颈缩和断裂，并据此理解安全系数与许用应力的来源。[3]

### 2. 扭转

扭矩图由截面法得到，等直圆杆扭转时切应力沿半径近似线性增加，最大值出现在外表面。扭转强度验算关注最大切应力，刚度验算关注单位长度扭转角。圆轴的极惯性矩决定抗扭能力，因此材料远离中心布置通常更有效。非圆截面自由扭转不能简单套用圆轴公式，应使用适合截面形状的理论或数值模型。[4]

### 3. 梁内力与弯曲

梁问题先求剪力和弯矩。分布载荷 `q`、剪力 `Q` 和弯矩 `M` 之间存在微分关系，图形上的斜率、跳跃和极值位置可以互相校核。复杂梁、刚架和曲杆要明确截面方向、内力正号和局部坐标，不能把直梁公式直接搬到曲杆。[5]

纯弯曲时截面变形后仍近似保持平面，梁的弯曲正应力可写为 `σ=M y/I`。强度条件要检查截面最外缘；合理设计则通过截面形状和材料分布提高截面惯性矩。弯曲切应力与剪力、截面面积和截面几何有关，工字梁腹板和翼缘的受力分工不同，应分区域验算。[6]

### 4. 梁的位移

梁位移可以用叠加原理、外力平移和单位载荷思想求解，也可以从挠曲线微分方程积分。积分法需要分段处理集中力、支座和载荷突变，并用边界条件和连续条件确定积分常数。对称和反对称荷载可以减少计算量，但必须同时检查位移和转角的对称性。[7]

### 5. 超静定结构

超静定问题仅靠平衡方程不够，需要加入变形协调条件和材料/几何关系。基本解法是：选多余约束力 -> 释放约束得到基本静定结构 -> 求外载和单位多余力产生的位移 -> 用协调条件建立方程 -> 回代求全部反力和内力。[8]

三杆桁架、带刚性构件梁和工程支承是典型例子。关键不是增加更多平衡方程，而是识别哪个位移必须相等、哪个构件不能伸长或哪个支座变形受到约束。[9]

### 6. 温度、装配与超静定

自由构件受均匀温升只产生热伸长，不产生约束应力；当伸长被支座、刚性构件或装配误差限制时，才出现温度应力和装配应力。计算时先求自由热变形，再用协调条件求约束反力，最后由本构关系得到应力。[10]

扭转超静定和简单超静定梁沿用同一主线：平衡方程 + 变形协调 + 本构关系。温度、装配和外载可以在线性弹性范围内叠加，但必须确认材料仍未进入塑性，且几何和接触状态没有改变。[11]

## 通用解题模板

1. 画整体受力图，求支反力并标出载荷单位。
2. 用截面法求轴力、剪力、弯矩或扭矩，绘制内力图。
3. 根据构件类型选择正应力、切应力、弯曲或扭转公式。
4. 进行强度、刚度和稳定性检查，区分平均应力与局部峰值。
5. 超静定问题补充变形协调；温度/装配问题补充自由变形与约束关系。
6. 用边界、对称性、量纲和极值位置复核结果。

## 常见误区

- 轴力图、剪力图和弯矩图的正号未统一。
- 把许用应力当材料屈服强度，忘记安全系数。
- 把圆轴扭转公式用于非圆截面。
- 只验算最大正应力，忽略切应力、刚度和局部稳定。
- 超静定结构只列平衡，不列变形协调。
- 温度应力计算中忘记先求自由热变形。
- 梁位移积分时漏掉分段连续条件或边界条件。

## 自测题

1. 轴向拉压的基本未知量是什么？答：轴力、应力和轴向变形。
2. 为什么强度和刚度要分别验算？答：强度控制是否破坏，刚度控制变形是否超限。
3. 圆轴扭转最大切应力在哪里？答：外表面。
4. 分布载荷、剪力和弯矩有什么关系？答：它们通过微分关系互相联系，可用图形斜率和跳跃校核。
5. 超静定比静定多了什么方程？答：变形协调方程。
6. 自由热伸长被限制时会发生什么？答：产生约束反力和温度应力。
