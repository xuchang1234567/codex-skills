# 嘉立创 EDA 专业版 PCB 教程：评论问题与学习提醒

> 以下内容来自评论区，属于学习者问题和经验反馈，不等同于视频正文。

## 零基础边界

评论中反复出现“没有模电数电、单片机或电路基础能否学习”的问题。较稳妥的理解是：本课程可以让 PCB 绘图零基础者快速进入工具和项目流程，但不等于完全没有电路基础也能立刻理解所有原理图。至少应补齐电阻、电容、二极管、三极管、MOS、供电、复位和接口的基本作用。

## 数据手册与外围电路

学习者常问芯片外围电路如何确定。答案应回到数据手册：典型应用、推荐工作条件、引脚功能、去耦、复位、晶振、上拉/下拉和保护电路都需要查证。不要因为“别人这样画”就复制到自己的芯片上。

## 软件与封装

常见问题包括嘉立创 EDA 下载、元件属性/Value 显示、封装选择和原理图与 PCB 交互。建议先熟悉工程、库、属性面板和网络检查，再开始完整项目；符号引脚编号与封装焊盘编号必须逐一核对。

## 布局布线

评论提到自动泪滴、两层板模板、布局传递和原理图/PCB 交叉选择等工具操作。快捷操作能提高效率，但不能替代电源回流、差分信号、模拟采样和机械结构分析。自动布线或复制布局后仍要人工检查。

## 实物制作

USB 拓展坞项目的反馈集中在 Type-C 焊接、供电路径、USB 转 TTL 供电冲突、加热台、锡膏连锡和无线鼠标测试。实物调试时要先确认供电来源、开关是否真正切断、USB 数据线是否支持数据、接口焊接是否有桥连，再判断芯片和 PCB 是否损坏。

## 建议的评论区复习问题

1. 我是 PCB 绘图零基础，还是电路基础也零基础？缺口不同，补课顺序不同。
2. 每个芯片外围元件的依据是哪一页数据手册？
3. 这个封装的 Pin 1、焊盘编号和实物方向是否一致？
4. 电源开关、USB 供电和调试器供电是否存在反向供电？
5. DRC 通过后，是否还检查回流、热、机械和可焊性？
6. 首板故障是原理图、封装、焊接、供电还是测试条件导致的？
