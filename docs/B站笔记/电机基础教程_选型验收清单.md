# 电机基础教程：选型验收清单

## 需求

- [ ] 运动类型明确：旋转、直线、输送、丝杆、齿轮齿条或转台。
- [ ] 行程、速度、加速度、定位精度和运行周期已定义。
- [ ] 负载质量、转动惯量、摩擦、重力和外部工艺力已记录。
- [ ] 启动、加速、急停、堵转和最大负载工况已考虑。

## 计算

- [ ] 稳态转矩/推力已计算。
- [ ] 加速转矩/推力已计算。
- [ ] 传动效率、减速比、导程或节圆半径已纳入。
- [ ] 负载惯量已折算到电机侧。
- [ ] 连续能力、峰值能力和安全系数分别核对。
- [ ] 速度-转矩或速度-推力曲线覆盖全工况。

## 电机与驱动器

- [ ] 电机类型与反馈需求匹配。
- [ ] 驱动器额定电流、峰值电流和电源匹配。
- [ ] 编码器分辨率、接口和反馈方向已确认。
- [ ] 制动、急停、过流、过温和断电安全已设计。
- [ ] 安装尺寸、轴径、法兰、连接器和散热满足机械结构。

## 验证

- [ ] 空载低速方向和反馈正确。
- [ ] 加速、减速和反向没有异常振动或报警。
- [ ] 额定负载和最大负载下速度/位置误差满足要求。
- [ ] 连续运行温升在允许范围内。
- [ ] 机械刚度、联轴器、丝杆/皮带/齿轮间隙可接受。
- [ ] 记录实际电流、速度、温升、误差和故障码。
