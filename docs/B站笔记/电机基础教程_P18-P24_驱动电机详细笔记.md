# 电机基础教程 P18-P24：驱动电机分类与选型

## P18-P20：驱动电机分类

电机分类有多个维度，不能把它们混为一个互斥列表：按换向/控制方式可分有刷和无刷；按供电形式可分直流和交流；按旋转磁场与转子关系可分同步和异步；按相数可分单相和三相。

有刷直流电机结构直观、控制简单，但存在电刷和换向器磨损、火花、噪声和维护问题。无刷直流电机用电子换向，效率、寿命和高速性能通常更好，但需要位置检测、驱动器和换向控制。

三相电机适合工业功率和连续运行，单相电机常见于小功率交流场景。同步电机转速与旋转磁场同步，异步电机存在转差。选择时应结合电源、转矩、速度、控制方式、成本、维护和环境。

## P21：三相/单相/直流/交流与同步/异步

比较电机时先写清楚分类依据：电源是直流还是交流，绕组和磁场如何产生，转子是否需要励磁或永磁，换向由机械还是电子完成，转速是否与电源频率同步。这样可以避免把“无刷直流”和“永磁同步”等不同命名体系直接等同。

对于工程选型，真正重要的是供电和驱动器是否匹配、负载特性是否匹配、速度调节范围是否满足、启动和制动是否可控，以及维护和安全要求是否可接受。

## P22-P23：常用驱动电机与选型公式

常用驱动电机包括有刷直流、无刷直流、交流异步、永磁同步和其他专用电机。选型前要先确定电机种类，再根据负载计算转矩、转速、功率和惯量。

通用计算框架：

- 旋转功率与转矩、角速度相关；
- 线性负载需要通过半径、导程或传动效率换算到电机轴；
- 加速需要额外转矩；
- 摩擦、坡度、冲击和安全系数不能省略；
- 连续运行看额定能力，启动和加速看峰值能力。

电机额定功率不是唯一选型标准。速度-转矩曲线、热、启动电流、驱动器能力、编码器/制动和机械接口同样重要。

## P24：皮带输送机选型

皮带输送机需要根据负载质量、皮带和托辊阻力、输送速度、倾角、启动加速度、滚筒半径和效率计算驱动转矩。启动时的阻力和加速转矩可能高于稳定匀速运行，应分别核算。

选型还要检查皮带打滑、减速机构、滚筒轴承、张紧、连续热负载和急停工况。若使用减速机，需要把减速比、效率和输出转矩同时纳入电机侧计算。

## 驱动电机选择表

| 需求 | 重点候选 | 主要风险 |
|---|---|---|
| 低成本、简单调速 | 有刷直流 | 电刷磨损、噪声、维护 |
| 高寿命、高效率 | 无刷直流 | 驱动和换向复杂度 |
| 工业连续运行 | 交流异步/同步 | 电源和变频器匹配 |
| 高动态、高精度 | 伺服 | 成本、调试和惯量匹配 |
| 低速定位、成本敏感 | 步进 | 高速转矩下降、失步 |
