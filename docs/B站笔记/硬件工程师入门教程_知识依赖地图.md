# 硬件工程师入门教程：知识依赖地图

## 主链

```text
电压/电流/功率
        |
        v
电阻与分压 -- RC/RL 动态 -- 低通/高通滤波
        |
        v
二极管/整流/钳位 -- 三极管开关 -- MOS 功率开关
        |                         |
        v                         v
比较器阈值与迟滞 ------------ 运放反馈与放大
        |                         |
        +----------> 电流采样/恒流源
                              |
                              v
                 线性稳压 / BUCK / BOOST
                              |
                              v
              电机驱动 / 传感器 / 循迹小车
                              |
                              v
                     原理图 -> PCB -> 测试
```

## 依赖关系

### 电阻 -> 电容/电感

电阻基础支撑分压、限流和功耗计算；RC/RL 在此基础上引入时间响应、储能和频率特性。没有电压、电流和功率概念，就无法判断滤波器或延时电路是否安全。

### 二极管 -> 三极管/MOS

二极管提供单向导电、整流、钳位和续流路径；三极管与 MOS 把控制信号变成开关或功率输出。感性负载保护依赖对电感电流连续性的理解。

### 比较器 -> 运放 -> 反馈系统

比较器解决“是否超过阈值”，运放解决“按比例放大误差”，采样和恒流源进一步构成反馈控制。比较器迟滞、运放稳定性和反馈极性是同一条设计链上的不同层次。

### MOS/功率器件 -> 电源

MOS 的 `Vgs`、导通损耗、寄生电容和体二极管是 BUCK/BOOST 等开关电源的基础。电源设计还要加入电感饱和、续流、纹波、热和 PCB 回流路径。

### 电源/驱动 -> 系统/PCB

循迹小车把传感器、比较器、逻辑和电机驱动组合起来；PCB 则要求把原理图中的网络、功率路径、信号参考和机械结构变成可制造实体。

## 学习判断

- 不懂分压和功耗：先回 P1-P8。
- 不懂瞬态和滤波：先回 P9-P28。
- 开关管发热：重点复习 P38-P65。
- 比较器乱跳：重点复习 P66-P71。
- 运放输出饱和/振荡：重点复习 P72-P92。
- 电源纹波或炸管：重点复习 P93-P103。
- 小车复位或跑偏：重点复习 P104-P115。
- PCB 首板问题：重点复习 P116-P125，同时回看对应功率和回流路径。
