# 硬件工程师入门教程：公式与参数速查表

## 基础电路

- 欧姆定律：`V = IR`
- 功率：`P = VI = I^2R = V^2/R`
- 串联电阻：`R_total = R1 + R2 + ...`
- 并联电阻：`1/R_total = 1/R1 + 1/R2 + ...`
- 分压：`Vout = Vin × R_bottom / (R_top + R_bottom)`

分压公式默认后级负载不影响电阻网络；接入负载后，应把负载并入下臂重新计算。

## RC 与 RL

- RC 时间常数：`tau = RC`
- 电容电压不能瞬间突变
- RL 时间常数：`tau = L/R`
- 电感电流不能瞬间突变
- 电容容抗：`Xc = 1 / (2πfC)`
- 电感感抗：`Xl = 2πfL`

实际电路还要考虑初始条件、负载、器件 ESR、线圈直流电阻和寄生参数。

## 滤波

- 一阶 RC 截止频率：`fc = 1 / (2πRC)`
- 一阶 RL 截止频率可由 `R/L` 与角频率关系推导
- LC 网络需要检查谐振、阻尼、ESR 和负载影响

“低通/高通”只是频率响应分类，不能替代对输入阻抗、输出阻抗和瞬态波形的检查。

## 二极管与三极管

- 电阻限流近似：`R = (Vin - Vdevice - Vload) / I`
- 电阻功耗：`P = I^2R`
- 三极管基极电流设计要结合目标集电极电流和强制放大倍数
- 稳压二极管必须检查工作电流、反向耐压和功耗

感性负载必须提供续流或钳位路径，不能只按稳态电流选择开关器件。

## MOS 管

- 核心驱动量：`Vgs = Vg - Vs`
- 导通损耗近似：`Pcond = I^2 × Rds_on`
- 栅极电荷越大，所需驱动电流和开关时间通常越高
- 功率选型要看指定 `Vgs` 下的 `Rds_on`，不能只看阈值电压

检查：耐压、连续/脉冲电流、导通电阻、栅极电荷、体二极管、热阻、开关频率和布局回路。

## 运放与采样

- 同相放大：`Av ≈ 1 + Rf/Rg`
- 反相放大：`Av ≈ -Rf/Rin`
- 电流采样：`Vsense = I × Rsense`
- 采样功耗：`Psense = I^2 × Rsense`
- 跨阻放大：`Vout ≈ -Iin × Rf`

使用虚短/虚断前，检查负反馈、输入共模范围、输出摆幅、带宽、压摆率、失调和稳定性。

## 电源

- 输入功率：`Pin = Vin × Iin`
- 输出功率：`Pout = Vout × Iout`
- 效率：`eta = Pout / Pin`
- 线性稳压器热耗散近似：`Pheat = (Vin - Vout) × Iout`
- 理想 BUCK：`Vout ≈ D × Vin`
- 理想 BOOST：`Vout ≈ Vin / (1-D)`

实际开关电源还需计入开关损耗、导通损耗、二极管压降、电感 DCR、电容 ESR 和控制误差。

## PCB 设计检查

- 线宽按持续电流、峰值电流、铜厚和允许温升选择
- 高 di/dt 回路尽量短、紧凑、面积小
- 模拟采样线远离开关节点和大电流回流路径
- 去耦电容靠近器件电源引脚
- 连接器、极性器件和大功率器件必须有清晰方向标识
- DRC 通过不等于设计正确，还要人工复核封装、回流、散热和机械装配

## 安全原则

任何公式计算都必须与器件额定电压、电流、功耗、温升、瞬态和故障状态比较。涉及市电、高压、大电流、电池短路或大容量电容时，不应只依据理论笔记直接上电。
