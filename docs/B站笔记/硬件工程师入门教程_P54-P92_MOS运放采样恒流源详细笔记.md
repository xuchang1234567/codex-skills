# 硬件工程师入门教程 P54-P92：MOS、比较器、运放、采样与恒流源

## P54-P65：MOS 管与电源开关

MOS 管是电压控制型器件，导通程度与栅源电压 `Vgs` 有关。设计时不能只看栅极对地电压，必须看实际 `Vgs`；高端开关中的源极可能随电源或负载变化，导致同样的栅极电压无法得到同样的导通程度。

### 导通与损耗

数据表中的阈值电压只表示开始出现小电流，不代表适合功率导通。应查看指定 `Vgs` 下的导通电阻、连续/脉冲电流、功耗和热阻。导通损耗通常与电流平方和导通电阻有关；开关损耗则与电压、电流、上升/下降时间和开关频率相关。

### 寄生电容

MOS 的栅极、漏极和源极之间存在寄生电容。栅极驱动需要给这些电容充放电，驱动电阻过大时开关变慢，过小时可能带来振铃和 EMI。高频或大功率场景要考虑栅极电阻、下拉电阻、驱动电流、死区和 PCB 回路面积。

### 体二极管与双向电流

MOS 内部寄生体二极管会提供一个固定方向的电流路径。即使沟道关断，体二极管也可能导通，因此电源防反接、半桥、电机和感性负载设计不能忽略它。若要求双向阻断，常需要背靠背 MOS 或其他专门结构。

### NMOS/PMOS 选择

低端开关使用 NMOS 较容易得到足够的 `Vgs`；高端开关可用 PMOS 简化驱动，但导通电阻和速度可能不如合适的 NMOS。高端 NMOS 通常需要自举或专用栅极驱动器，使栅极电压高于源极。

## P66-P71：比较器

比较器把两个输入电压进行比较，输出表示哪一侧更高。开漏/开集电极输出只能主动拉低，释放时需要上拉电阻；上拉阻值影响上升时间、静态电流、噪声抗扰度和多个输出并联时的逻辑行为。

比较器输入必须处在允许的共模范围内，输出也有响应时间和饱和恢复时间。输入信号接近阈值时，噪声可能让输出反复跳变，这就是颤振。加入正反馈形成迟滞，或使用滤波、合理阈值和屏蔽，可以提高稳定性。

## P72-P78：运放基础与放大电路

理想运放分析常用两条近似：负反馈正常工作时输入端电压近似相等，即虚短；输入端电流近似为零，即虚断。它们不是无条件真理，输入共模范围、输出摆幅、增益带宽、压摆率、失调和稳定性都会限制实际电路。

### 常见结构

- 同相放大：输出与输入同相，增益由反馈电阻比决定；
- 反相放大：输出反相，输入电阻和反馈电阻决定增益；
- 单电源反相：需要设置合适的偏置基准，不能让信号低于器件允许范围；
- 差分放大：放大输入差值并抑制共模信号，电阻匹配直接影响共模抑制能力。

推导时先写节点电流关系，再使用虚短/虚断，最后检查实际电源范围。电阻误差、输入偏置电流和失调电压会在高增益或小信号测量中变得明显。

## P79-P84：电流采样与仪表放大器

低端采样把采样电阻放在负载与地之间，电路简单，但会抬高负载地电位；高端采样不破坏负载地，却需要输入共模范围更高、差分测量能力更强的电路。

采样电阻要同时考虑阻值、功耗、精度、温漂和压降。阻值太小，采样信号小、放大器误差占比大；阻值太大，功耗和对负载的影响增加。

仪表放大器适合放大微小差分信号并抑制共模电压。电阻匹配、输入保护、共模范围、输出摆幅和偏置电压都必须检查。带偏置的差分放大可以把双向电流映射到单电源 ADC 可测范围，但偏置误差会直接转化为测量零点误差。

跨阻放大器把输入电流转换成输出电压，反馈电阻决定转换比例；输入电容、反馈电容和运放带宽会影响稳定性。光电流、传感器电流和漏电流测量中尤其要重视噪声与反馈补偿。

## P85-P92：恒流源与反馈稳定性

恒流源通过检测输出电流、与参考值比较并调节功率器件，使负载变化时电流尽量保持稳定。关键是确定检测点、反馈极性、控制元件工作区和最大压差。

高端反馈恒流源要处理高侧电压、差分检测和控制器共模范围。恒流源不仅要“电流稳定”，还要检查最小压差、最大功耗、启动行为、短路状态和负载断开状态。

反馈回路可能自激振荡。原因包括环路增益过高、相位裕度不足、寄生电容、负载变化和布局回路过大。常见改善方式包括反馈补偿电容、串联栅极/基极电阻、调整带宽、增加阻尼和缩短高 di/dt 回路。

## 综合设计检查

- MOS 是否按实际 `Vgs` 选择，而不是只看阈值电压？
- 栅极是否有默认状态，关断时是否可靠？
- 体二极管是否形成了意外电流路径？
- 开关速度、寄生电容和驱动回路是否匹配？
- 比较器输入范围和输出上拉是否正确？
- 运放是否满足共模、输出摆幅、带宽和稳定性？
- 采样电阻的压降、功耗和误差是否可接受？
- 高端采样是否处理了共模电压？
- 恒流源是否考虑启动、短路、断载和散热？
- 反馈环路是否存在振荡和布局敏感点？
