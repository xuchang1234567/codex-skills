# 理解有限元分析的原理

## 学完你应该获得什么

- 知道有限元法解决的不是“画网格”，而是复杂连续体问题的近似求解。
- 能说清节点、单元、自由度、网格、刚度矩阵各自扮演的角色。
- 能从 `K u = f` 理解载荷、位移、刚度与约束如何共同决定结果。
- 能理解局部单元刚度如何装配成整个模型的全局刚度矩阵。
- 能建立基本有限元工作流，并知道哪些判断不能交给软件。

## 一句话总论

有限元分析把难以直接求解的连续体离散为由节点连接的单元，用每个单元的近似力学关系组装出全局方程，再在材料、载荷和边界条件下求节点位移，最后由位移恢复应变、应力等结果。[1][2]

## 为什么需要有限元

对简单梁，可由平衡关系直接计算剪力、弯矩和应力；但几何、载荷或材料一旦复杂，尤其进入二维或三维结构，连续场中处处满足平衡的解析求解就很困难。有限元法用数值近似替代解析解，适用于静力、动力、模态分析，也可扩展到流体、传热和电磁问题。[1]

这里以线性静力弹性分析为主：目标通常是获得位移、应变和应力。要注意，线性静力意味着材料本构、载荷关系和边界条件都做了特定简化；计算结果是否可信取决于这些假设是否适用于实际问题。

## 知识地图：从实体到结果

`定义问题 -> 离散化 -> 选单元 -> 单元方程 -> 全局装配 -> 施加载荷与约束 -> 求解位移 -> 后处理与验证`。

离散化把物体切分为有限个小单元，并在节点处连接；节点与单元的集合称为网格。这样不再需要直接处理整个连续体，而是在有限个自由度上满足平衡与连续性。[2]

单元形状与阶次必须匹配问题：薄板可使用表面单元，三维实体可用固体单元，杆和梁可用线单元；三角形更适合复杂边界，四边形在一些情况下性能更好；二阶单元借助中间节点提高近似能力，但并非自动优于合理网格的一阶单元。[3][4]

## 核心概念卡

### 自由度与节点位移

有限元通常先求节点位移 `u`，因为应变可由位移梯度导出，应力再由材料本构关系得到。每个可独立变化的平移或转角是一个自由度。二维梁节点可有两个平移和一个转角，三维梁节点常有六个自由度；固体单元节点一般只有三个平移自由度，转动由周围节点位移间接表示。[4][5]

### 单元刚度矩阵

单元的力-位移关系可写作 `f_e = K_e u_e`。`K_e` 是单元刚度矩阵，编码了单元在材料、几何和局部坐标下抵抗变形的方式；其阶数由单元自由度数决定。它类似弹簧的刚度关系，但能够同时表达轴向、弯曲、剪切和扭转等耦合行为。[5][6]

### 全局刚度矩阵与装配

按节点连接关系把所有 `K_e` 相加，得到结构级方程 `F = K U`。共享节点意味着相邻单元在该节点具有相同位移，这就是连续性条件，也是装配时相应矩阵项相加的原因。大型模型的 `K` 往往是稀疏、带状的；在线弹性问题中它通常还是对称矩阵。若单元不与全局坐标轴对齐，还须先做坐标变换。[7][8]

## 求解有限元模型的实际流程

1. **定义问题。** 给出几何、材料参数、接触或连接假设、载荷工况和目标量；先明确要回答的是强度、刚度、频率还是温度问题。
2. **选择理想化与网格。** 决定用梁、壳还是实体单元。不要因为软件能建实体就放弃更恰当的简化；单元类型、网格质量和局部加密都来自工程判断。[3][4]
3. **施加边界条件与载荷。** 边界条件本质是已知的节点位移，常见为固定某个自由度；载荷向量包含外力和力矩。缺约束会导致刚体运动和奇异矩阵，过约束则可能人为增大刚度。[9]
4. **组装并求解。** 软件组装全局 `K`，在约束条件下解出 `U`。概念上可写成求逆，但实际大规模稀疏系统通常使用直接或迭代求解器，例如共轭梯度法，而不是显式求逆。[9][10]
5. **后处理与验证。** 由节点位移计算单元内应变、应力和其他场变量。云图是结果呈现，不是验证本身；需检查平衡、变形形态、单位、网格收敛、边界敏感性，并与手算、试验或已知解交叉核验。[10][14]

## 刚度矩阵从哪里来

最直接的路线从控制单元行为的平衡微分方程出发，得到所谓强形式；它对简单单元可行，但对一般问题不够方便。更通用的弱形式把微分方程转为积分形式，允许用近似解满足平衡。[11]

两条常见路线是：最小势能原理，把平衡位移视为总势能最小的位移配置；以及 Galerkin 加权残差法，用试函数近似未知场，再令残差的加权积分为零。对于应力分析，两者会导出相同的单元刚度矩阵；后者适用范围更广。[12][13]

二者都依赖**形状函数**：它以节点值插值单元内部的位移场，通常选用多项式。形状函数决定了单元能表示何种变形，也解释了为何网格加密和提高单元阶次会改变近似精度。[13]

## 常见误区

- **“网格越密一定越正确”。** 网格只能减少离散误差，不能修复错误的材料、约束、载荷、连接或物理模型。
- **“应力云图最大值就是设计结论”。** 约束点、尖角或点载荷附近可能出现应力奇异性；应看热点的物理意义、网格收敛及规范定义的位置。
- **“软件负责正确性”。** 软件负责矩阵计算、装配和求解；工程师负责问题定义、单元和网格选择、结果解释与验证。[14]

## 自测题

1. 为什么要离散化？答：把连续体问题转为有限个自由度上的近似平衡问题。
2. `K U = F` 中最先求的未知量是什么？答：节点位移 `U`。
3. 共享节点为何能使单元矩阵装配？答：共享处位移连续，相应自由度的刚度贡献相加。
4. 何时不应把梁简化为实体？答：当局部三维应力、连接细节或厚度效应决定问题时。
5. 后处理后还必须做什么？答：验证假设、平衡、网格收敛和结果合理性。
