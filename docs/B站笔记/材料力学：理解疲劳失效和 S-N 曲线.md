# 材料力学：理解疲劳失效和 S-N 曲线

## 学完你应该获得什么

- 能解释为什么构件的工作应力远低于静强度也会断裂。
- 能按“萌生、扩展、瞬断”描述疲劳裂纹的演化。
- 能读懂 S-N 曲线的横轴、纵轴，以及它能回答的寿命问题。
- 能区分高循环疲劳、低循环疲劳和已有裂纹三种分析入口。
- 能把平均应力、复杂载荷和离散试验数据纳入疲劳设计判断。

## 一句话总论

疲劳不是“某一次载荷太大”，而是循环载荷使微小裂纹逐步长大；S-N 曲线把应力范围映射为破坏前循环次数，是无初始裂纹构件估算疲劳寿命的基础工具。[1][2]

## 知识地图

`循环载荷 -> 裂纹萌生 -> 稳定扩展 -> 剩余截面不足而断裂` 是失效机理。工程上先把实际载荷化为应力循环，再根据载荷特征选择模型：恒幅、全反复且主要弹性变形时用 S-N 曲线；应力超过屈服、塑性应变显著时转向应变-寿命方法；构件已有可测裂纹时转向断裂力学和裂纹扩展规律。[1][5][8]

## 核心概念卡

### 1. 疲劳失效

疲劳裂纹最常从自由表面、缺口、孔边等应力集中处萌生。随后裂纹在每个循环中扩展，直到有效截面不足以承载，发生最终断裂。因此，静力校核合格不等于疲劳安全；螺栓、自行车曲柄和承压管道都可能因随时间变化的载荷而失效。[1]

### 2. S-N 曲线

S-N 曲线以应力范围（或应力幅）为纵轴、破坏循环次数 N 为横轴，N 通常采用对数坐标。对给定应力范围，从曲线读出破坏循环数，再结合实际每分钟或每天的循环频率，就能换算预期寿命。例如，视频中的示例把 100 MPa 的应力范围对应到约 50 万次循环。[2]

对部分铁素体材料，曲线在高循环区域会趋于水平，形成耐久极限。若应力范围低于该限值，理想化模型认为可无限寿命；具体项目仍应以材料数据和规范为准。[3]

### 3. 应力范围、幅值与平均应力

应力范围 `Delta sigma = sigma_max - sigma_min`，应力幅 `sigma_a = Delta sigma / 2`，平均应力 `sigma_m = (sigma_max + sigma_min) / 2`。全反复循环的平均应力为零；实际载荷常带有非零平均应力，尤其拉伸平均应力通常会缩短疲劳寿命。[6]

## 如何做一次疲劳寿命判断

1. 明确关键部位及其循环应力历史，特别关注孔、螺纹、焊趾和截面突变。
2. 对恒幅载荷，确定应力范围和平均应力，选用与材料、表面状态、环境和可靠度相匹配的 S-N 数据。
3. 不要直接采用试验点的平均拟合线。疲劳寿命离散性很大，规范曲线通常会下移以降低失效概率；视频示例中下移两个标准差，可将破坏概率从约 50% 降至约 1%。[4]
4. 若平均应力不为零，用对应平均应力的 S-N 数据，或借助 Goodman 图修正耐久极限。Goodman 图的横轴为平均应力、纵轴为应力幅；位于边界线下方可用于判断无限寿命，但它本身不能给出有限寿命。[6][7]
5. 若载荷谱不规则，先用雨流计数把复杂历程分解为若干应力循环，再按 Miner 线性累积损伤求和：`D = sum(n_i / N_i)`。当 `D > 1` 时，按该准则判定达到疲劳失效。[7]

## 重要边界与常见误区

- **高循环不等于低风险。** 高循环疲劳通常循环次数超过 1 万次，单次应力较低且主要是弹性变形，但累计循环足以造成破坏。[5]
- **低循环不宜硬套 S-N 曲线。** 当循环应力超过屈服强度、塑性变形不可忽略时，应优先采用应变-寿命关系，例如 Coffin-Manson 关系。[5]
- **已有裂纹时别再只看 S-N。** S-N 方法适合无初始裂纹的寿命估算；若裂纹尺寸已知，应计算临界裂纹尺寸，并用线弹性断裂力学与裂纹扩展规律预测剩余寿命。[8]
- **Miner 准则是工程近似。** 它把不同应力级的损伤线性相加，适用于快速评估；实际的载荷顺序效应、平均应力和环境影响仍需在模型与试验中处理。

## 自测题

1. 为什么低于极限强度的应力也会造成断裂？答：循环载荷会萌生并扩展裂纹，最终导致剩余截面失效。
2. S-N 曲线的 N 表示什么？答：给定应力条件下破坏前的循环次数。
3. 为什么横轴常用对数坐标？答：疲劳寿命的循环次数跨度很大。
4. 拉伸平均应力通常如何影响寿命？答：通常会降低疲劳寿命。
5. 已知初始裂纹时最合适的分析入口是什么？答：断裂力学和裂纹扩展分析。
