# 弹性力学简明教程：从连续体方程到有限元

## 学完你应该获得什么

- 理解弹性力学研究对象、基本量和连续体假设。
- 掌握平面应力/平面应变的选择逻辑。
- 能组织平衡方程、几何方程、本构方程、相容条件与边界条件。
- 能在位移法、应力法和 Airy 应力函数法之间选择。

## 一句话总论

弹性力学的主线是：用连续体模型描述位移、应变和应力，用平衡、几何、本构与边界条件组成边值问题，再根据几何选择解析坐标与求解方法；当解析求解困难时，将同一套方程离散成有限元矩阵并数值求解。[1][2]

## 一、连续体模型与基本量

弹性力学研究弹性体在外力、约束和温度等作用下的位移 `u`、应变 `ε` 与应力 `σ`。体力作用于单位体积，面力作用于边界单位面积，内力通过应力描述。问题开始前要明确研究域、坐标、量纲、符号和载荷类型。[1]

常用基本假定是连续、均匀、各向同性、线弹性和小变形。小变形使我们可以用变形前几何写平衡方程；线弹性使本构关系线性；各向同性用少量材料常数 `E、ν、G` 描述。若材料塑性、几何大变形或各向异性显著，必须更换模型。[2]

## 二、平面问题的四套关系

### 平面应力与平面应变

薄板且厚度方向应力可以忽略时，使用平面应力：`σz=τxz=τyz=0`。长体或厚向变形受强约束时，使用平面应变：`εz=γxz=γyz=0`。二维网格并不自动决定这两者，选择取决于几何、载荷和约束。[3]

### 平衡方程

平面静力平衡为：

`∂σx/∂x + ∂τxy/∂y + fx = 0`

`∂τxy/∂x + ∂σy/∂y + fy = 0`

斜截面的牵引由应力张量和边界方向余弦变换得到。平衡是“力不产生净加速度”的条件，但单独满足平衡并不能保证存在连续位移场。[4]

### 几何方程与刚体运动

小变形下：

`εx=∂u/∂x，εy=∂v/∂y，γxy=∂u/∂y+∂v/∂x`

刚体平移和刚体转动不产生应变，却会改变位移值。因此位移解中的积分常数不能随意删除，必须由约束或位移基准确定。[5]

### 本构方程

各向同性线弹性材料用 Hooke 定律连接应力和应变。平面应力常写为 `εx=(σx-νσy)/E`、`εy=(σy-νσx)/E`、`γxy=τxy/G`；平面应变则常用 Lamé 常数表达三维约束产生的 `σz`。求解前要明确未知量是位移、应变还是应力，并选用相应形式。[6]

### 边界条件与 Saint-Venant 原理

位移边界给出 `u、v`；应力边界给出边界牵引 `t=σn`，自由边界即牵引为零。二维边界每一点需要两个标量条件。Saint-Venant 原理允许用静力等效的简化荷载替代复杂端部载荷，但只保证远离加载区的整体响应近似，不能用来掩盖局部应力集中。[7][8]

## 三、三条解析求解主线

### 位移法

将几何方程和本构方程代入平衡方程，以 `u、v` 为未知，得到位移型偏微分方程。解出位移后回代得到应变和应力。位移法适合边界位移已知或位移场较容易构造的问题。[9]

### 应力法与相容方程

以 `σx、σy、τxy` 为未知，使用平衡方程，再加上相容条件。相容条件保证应变场确实来自连续、单值的位移场，而不是三个看似满足平衡但无法由位移实现的应变函数。[10]

### Airy 应力函数

无体力平面问题可设：

`σx=φ,yy，σy=φ,xx，τxy=-φ,xy`

这样平衡方程自动满足，剩下的是让 `φ` 满足双调和方程 `∇⁴φ=0`，再由边界条件确定待定系数。应力函数的关键不是背公式，而是选择满足双调和方程的函数族、计算边界牵引并回代验证。[11]

## 四、直角坐标解析算例

逆解法先假定一个满足控制方程的应力场，再反算它对应的边界荷载；它适合构造精确解和校核公式。半逆解法则先利用几何、对称性和边界形式猜测部分函数，再代入相容或双调和方程确定系数。[12]

矩形梁纯弯曲可以从二维弹性解恢复材料力学中的 `σx=-My/I`，同时显示泊松效应、端部边界和细长梁假设何时会带来偏差。由应力回求位移时要走“应力 -> 应变 -> 积分位移”链条，积分常数包含刚体平移和转动。[13]

简支梁均布载荷的完整流程是：选择多项式 Airy 函数，验证双调和条件，施加边界条件求系数，得到应力，再由本构和几何关系求位移。楔形体含体力时必须把体力项纳入控制方程，不能直接套用无体力解。[14]

## 五、极坐标与经典问题

圆孔、圆筒、隧洞等问题的边界几何天然适合极坐标。极坐标基矢随位置变化，平衡和几何方程出现 `1/r` 项，不能把 `r、θ` 当成笛卡尔坐标直接替换。应力法仍可使用应力函数，但坐标变换必须保持分量定义一致。[15]

轴对称圆环或厚圆筒的压力解属于 Lamé 型：

`σr=A-B/r²，σθ=A+B/r²`

再利用内外壁径向应力边界求 `A、B`。压力隧洞和圆孔问题提醒我们：局部孔口应力集中取决于孔形、载荷和外边界距离，不能只用一个固定集中系数替代完整分析。[16]

半平面边界集中力的处理要把集中点附近小半圆的合力平衡与远处无载边界分开。分布荷载可视为连续微元集中力的叠加，对集中力解积分获得；位移还要选择远处基准，以排除刚体位移常数。[17]

## 六、有限元：把连续体方程变成矩阵

有限元将连续体离散为节点和单元。单元内位移用节点位移插值：

`u=Nδe`

由此得到：

`ε=Bδe，σ=Dε=DBδe`

其中 `N` 是形函数，`B` 是应变-位移矩阵，`D` 是材料矩阵。虚功或能量原理给出单元刚度：

`ke=∫ BᵀDB t dA`

体力、面力和集中力要转换成与位移模式一致的等效结点荷载。按共享自由度装配所有单元后，形成全局方程 `KΔ=F`。[18]

三节点常应变三角形（CST）每节点两个平移自由度，共六个自由度；形函数为坐标一次函数，单元内 `B、ε、σ` 为常量。CST 能表达刚体位移和常应变，但应力在单元内不变，节点应力和云图通常是外推或平滑结果，不能把平滑云图当成原始精确场。[19]

位移模式至少要包含刚体位移、能表达常应变，并保持单元内及单元间的位移连续。网格加密和自由度增加应使结果趋于稳定；若结果只是在云图上更平滑，却没有位移、应力或能量收敛，不能称为验证。[20]

## 七、可复用的解题与仿真流程

1. 根据几何和载荷选择平面/空间、笛卡尔/极坐标以及应力/应变假设。
2. 列出未知量、平衡、几何、本构、相容和边界条件，检查方程数与未知数是否匹配。
3. 解析题优先利用对称性和边界构造函数；有限元题则选择单元、材料矩阵和网格。
4. 求解后回代检查平衡、相容、边界牵引和单位；不要只检查一个最大应力数值。
5. 有限元结果还需检查刚体约束、网格质量、局部加密、网格收敛、反力平衡及材料界面后处理。

## 自测题

1. 平面应力和应变的选择依据是什么？答：几何厚度、约束和载荷，而不是二维网格本身。
2. 为什么平衡方程还不够？答：还需几何、本构、相容和边界条件。
3. Airy 应力函数的好处是什么？答：在无体力情况下自动满足平衡，把重点转为双调和与边界条件。
4. 为什么积分位移会出现常数？答：它们代表刚体平移和转动，需由约束确定。
5. CST 单元的应力为何是常量？答：线性位移插值导致 `B` 常量，进而应变和应力在单元内常量。

## 模块化深读

### 模块 A：从问题定义到方程闭合

做任何弹性力学题，先写四件事：研究域是什么、边界施加什么、材料采用什么本构、希望求什么量。之后把未知量和方程数量列成表。以二维位移法为例，未知量是 `u、v` 两个函数；平衡给两条方程，几何和本构负责把应力消去，边界提供每个边界点的位移或牵引条件。若方程数不闭合，通常不是“算不出来”，而是模型还没定义完整。

最容易犯的错是把“载荷已知”误当成“边界条件已知”。边界上既可能给力，也可能给位移；同一段边界不能同时任意指定两者。对于固定边，位移约束是主条件；对于自由边，牵引为零；对于接触边，则需要额外的接触兼容和接触力条件。

### 模块 B：位移法、应力法和函数法如何选择

位移法的优势是位移天然满足连续性，适合位移边界清楚、位移场易构造的问题；代价是代入后得到的偏微分方程可能较复杂。应力法直接处理工程上关心的应力，但必须额外满足相容条件，防止得到一个无法由连续位移产生的“假应力场”。Airy 应力函数进一步自动满足无体力平衡，适合规则边界、载荷形式可表达的平面问题。

可按以下决策：

1. 位移边界占主导，先尝试位移法。
2. 应力边界和自由边界明显，且几何规则，可尝试应力函数法。
3. 圆形、孔口和圆筒边界优先换到极坐标。
4. 几何、材料或载荷复杂到无法构造函数时，转入有限元，不要继续堆叠猜测项。

### 模块 C：解析解的验证闭环

一个候选解至少要经过四项检查：

- 在域内满足平衡方程。
- 由应力反算的应变满足本构关系。
- 应变能够积分出单值、连续的位移，即满足相容。
- 所有边界上的位移或牵引条件都满足。

只检查其中一项是不够的。例如 Airy 函数可以自动满足平衡，但函数系数仍可能无法满足真实边界；只满足边界的应力场也可能违反相容。解析解的价值正在于它同时满足这四项，因此可作为数值模型的基准解。

### 模块 D：极坐标问题的建模纪律

切换到极坐标不是把 `x` 改成 `r`、把 `y` 改成 `θ`。极坐标基矢随位置变化，导数、剪应变和应力变换都会出现几何项。写方程时先确认应力分量的下标顺序、正方向和单位，再使用轴对称条件消去与 `θ` 无关的项。

对圆筒和圆孔问题，应先写径向边界条件，再判断外边界是否足够远。把无限域近似为半平面或把孔口近似为孤立孔，都有适用距离；若外边界太近，经典解的应力集中系数就不能直接使用。

### 模块 E：从连续方程到有限元矩阵

有限元不是独立于弹性力学的黑箱。其每一步都能在连续理论中找到对应物：

| 连续体概念 | 有限元对应 |
|---|---|
| 位移场 | 节点位移与形函数 `u=Nδ` |
| 几何方程 | `ε=Bδ` |
| 本构方程 | `σ=Dε` |
| 虚功/能量平衡 | 单元刚度 `ke` |
| 边界牵引 | 等效结点荷载 |
| 全域平衡 | `KΔ=F` |

因此调试有限元模型时，应沿这条链反查，而不是只盯着最后的云图。若位移方向错误，先查自由度编号和坐标；若应力数量级错误，查单位、材料矩阵和厚度；若矩阵奇异，查刚体约束；若局部峰值随网格加密持续上升，查尖角、点载荷或接触奇异性。

### 模块 F：网格与后处理的可信度

网格收敛至少应观察一个物理目标量，例如远场位移、总反力、某截面平均应力或应变能。单个节点的最大应力在尖角、点载荷和材料界面处可能没有收敛意义。CST 单元的单元应力为常量，节点云图通常经过外推和平滑；跨材料界面时不能把不同材料的应力任意平均。

验证顺序建议为：先检查反力是否平衡，再检查边界位移是否符合预期，再检查粗细网格的目标量变化，最后才解释局部应力峰值。解析解、手算梁理论、对称性和量纲分析都是低成本但高价值的交叉校核。

## 章节练习题

## 有限元结果验收模板

**模型定义**：记录几何单位、材料 `E、ν`、厚度、单元类型、坐标系和载荷单位。检查材料常数是否与工程单位一致。

**边界与载荷**：列出每个约束自由度，确认没有多余约束；列出外力、面力和体力，核对它们的合力与力矩。

**方程与求解**：确认全局矩阵没有未约束刚体模态；检查求解器残差、反力平衡和能量或功的数量级。

**网格研究**：至少建立粗、中、细三组网格，比较目标位移、反力、平均应力或应变能；对于尖角和点载荷，单点峰值不作为唯一收敛指标。

**后处理**：区分单元值、积分点值、节点外推值和节点平滑值；对材料界面、接触边和奇异区域单独标注。

**物理校核**：用对称性、量纲、手算梁理论、解析解或试验数据做交叉检查；任何“看起来合理”的云图都不能替代这些检查。

## 解析解与有限元的迁移案例

### 案例 1：均布拉伸薄板

解析思路是先判断薄板是否可视为平面应力，再写远端拉力和自由边条件。若应力场可取常量，平衡和边界都容易满足，位移则由本构和几何关系积分得到。有限元中，等价问题需要建立薄板几何、选择平面应力单元、在两端施加一致的面力或位移约束，并通过反力与理论平均应力校核。此案例适合验证材料矩阵、厚度和单位。

### 案例 2：带圆孔的拉伸板

解析上，圆孔改变了局部应力场，孔边切向应力出现集中，远场应力不能直接代表孔边峰值。极坐标或复变函数方法可给出理想化孤立孔解；有限元则需要在孔边局部加密，并比较孔边应力随网格变化的趋势。若孔距外边界不够远，孤立孔解析解只能作为定性参考，不能直接当作数值基准。

### 案例 3：厚圆筒内外压

解析解以径向平衡和轴对称条件为起点，得到 `σr=A-B/r²`、`σθ=A+B/r²`，再由内外壁压力求常数。有限元中应使用轴对称单元或足够合理的二维实体模型，施加径向压力和轴向约束。比较内壁和外壁的径向、环向应力，是检查极坐标实现、压力方向和约束设置的有效方法。

### 案例 4：简支梁均布荷载

解析解可以用 Airy 函数构造，先满足双调和方程，再施加简支和均布荷载边界。有限元中可选择梁单元或二维平面单元；梁单元效率高，但无法给出支座附近的二维局部应力；平面单元更接近连续体，却需要更细的网格和更严谨的边界建模。两种模型应比较全局挠度和支反力，而不是要求局部应力完全一致。

## 从错误结果反推错误来源

若整体位移数量级错一个数量级，优先查单位、厚度、弹性模量和载荷单位；若位移方向反了，查坐标正方向、结点编号和载荷符号；若模型无法求解，查刚体自由度和重复约束；若反力不平衡，查遗漏载荷、面力积分或约束反力输出；若只有尖角应力发散，查奇异性而不是盲目继续加密；若粗细网格结果振荡，查单元畸变、单元阶次和边界层网格。

## 公式推导与符号检查

平衡方程来自微小矩形的力平衡：相邻面的应力差、剪应力差和体力之和为零，得到 `σx,x+τxy,y+fx=0` 及其 y 方向对应式。体力单位是力/体积，面力不能直接代入域内方程。

小变形几何关系中，`εx=u,x`、`εy=v,y`，工程剪应变为 `γxy=u,y+v,x`；若改用张量剪应变，则 `εxy=γxy/2`。工程应变和张量应变混用会造成剪切项差一个 2。

平面应力中厚度方向应力近似为零，但厚度方向应变可能由泊松效应产生；平面应变中厚度方向应变为零，但厚度方向应力通常不为零。两者材料矩阵不同，不能只根据二维网格名称选择。

Airy 应力函数 `σx=φ,yy`、`σy=φ,xx`、`τxy=-φ,xy` 代入无体力平衡式后，混合偏导项相互抵消，因此自动满足平衡；双调和条件来自相容和本构，仍需用边界条件确定函数系数。

有限元中，虚功等式结合 `ε=Bδ`、`σ=Dε` 得到 `ke=∫BᵀDB dA`。这说明材料刚度、几何尺寸、厚度和单元形函数共同决定刚度矩阵。二维模型使用实际厚度还是单位厚度，必须在结果解释中明确。

单位自检表应同时列出几何、材料、载荷、厚度和输出单位：应力是力/面积，位移是长度，刚度矩阵条目是力/位移。单位一致性是有限元结果验收的第一道检查。

## 常见错误诊断表

| 现象 | 优先检查 | 典型原因 |
|---|---|---|
| 方程无唯一解 | 刚体自由度、约束秩 | 固定不足或约束方向错误 |
| 反力总和不等于外载荷 | 荷载积分、单位、约束反力输出 | 面力漏乘厚度或漏掉边界载荷 |
| 位移比手算大/小很多 | `E`、厚度、长度和单位 | MPa/Pa、mm/m 或单位厚度混用 |
| 应力方向反号 | 坐标和正号约定 | 法向方向、剪应力下标或载荷方向错误 |
| 孔边应力随加密持续增大 | 几何奇异性和载荷施加方式 | 尖角、点载荷、刚性约束边缘 |
| 网格加密后结果振荡 | 单元质量、单元阶次、边界层 | 畸变单元或不适合的单元类型 |

## 第一模块深度讲解：为什么需要弹性力学

材料力学通常把杆、梁、轴等结构抽象成一维构件，用截面内力和少量截面参数估算平均应力、挠度或扭转角。这种抽象非常高效，但它把截面上的应力分布、孔边集中、连接区域和复杂边界细节隐藏起来。弹性力学则回到连续体本身，在区域内求解随位置变化的位移、应变和应力场。

结构力学关注结构整体的受力和变形，弹性力学关注连续体内部的局部场变量；有限元则把弹性力学的连续方程离散成可计算的代数系统。三者不是相互替代，而是从简化模型到精细模型的不同层级。工程分析应先用低成本模型建立数量级和受力路径，再用弹性力学或有限元处理局部细节。

弹性力学问题必须同时满足三类约束。第一类是力学约束：内部每个微元都要满足平衡。第二类是几何约束：应变必须由位移梯度产生，不能出现互相矛盾的独立伸长。第三类是物理约束：应力与应变必须符合材料本构。边界条件把这些局部关系与外部支承、载荷和环境连接起来。

一个很实用的建模动作是先画“问题边界图”，再画“未知量图”。边界图标出固定边、自由边、受力边、对称边和接触边；未知量图标出每个区域内要求的位移、应变、应力或反力。最后再决定使用解析法、梁壳模型还是实体有限元。这样可以避免一上来打开软件，却不知道要验证什么。

小变形假设还包含一个经常被忽略的后果：平衡方程通常在初始几何上写，载荷方向和边界位置不随变形更新。若结构发生大转动、屈曲前后几何显著变化或接触状态改变，就必须使用几何非线性分析；线性弹性方程不能靠加密网格解决这些物理变化。

各向同性假设也需要主动检查。钢材在宏观上常可近似各向同性，但复合材料、轧制板、木材、晶体和增材制造材料可能具有方向相关的刚度。此时 `D` 矩阵不再由两个或三个常数简单决定，坐标方向、铺层和材料主轴必须进入模型。

## 第二模块深度讲解：平面模型如何选择

平面应力的典型对象是薄板。薄板在自身平面内承受拉伸、剪切或面内弯曲时，厚度方向的应力相对较小，可以把 `σz、τxz、τyz` 近似设为零。但这并不意味着厚度方向没有变形；泊松效应仍可能使板在厚度方向收缩或膨胀。

平面应变的典型对象是沿 z 方向很长、端部效应可以忽略的结构，例如无限长隧洞截面或长坝体截面。由于 z 方向被周围材料限制，常把 `εz、γxz、γyz` 设为零，但为了维持这个约束，`σz` 一般不为零。

选择二维模型时应问三个问题：厚度相对于平面尺寸是否足够小？结构是否沿一个方向足够长？厚度方向是否存在强约束或显著载荷？如果三个问题都无法明确回答，先做平面应力和应变两种敏感性分析，或直接建立三维局部模型比较。

边界条件也要和模型维度一致。对称边不是“把一条边固定”，而是禁止法向位移、允许切向位移；自由边的牵引为零；施加面力时要确认力是总力、单位长度力还是单位面积压力。对于二维实体，压力和面力是否需要乘以厚度取决于软件对载荷的定义。

圣维南原理在这里提供了一个重要的工程判断：如果两种端部载荷的合力和力矩相同，离开加载区足够远后，整体应力场可能接近。但支座附近、孔边和焊趾等局部区域正是应力集中发生的位置，不能用远场等效原则替代局部模型。

## 第三模块深度讲解：三条解析求解路线

位移法把 `u、v` 作为主要未知量。先由位移梯度得到应变，再由本构得到应力，最后代入平衡方程。这条路线的优点是位移连续性天然满足，尤其适合位移边界清晰的问题；缺点是消元后得到的偏微分方程可能包含高阶导数，边界条件的分类也会变得不直观。

应力法直接把 `σx、σy、τxy` 作为未知量。平衡方程负责保证力学平衡，本构关系把应力转成应变，相容方程则保证这些应变能够由一个连续位移场产生。相容条件不是可有可无的附加公式，而是从位移连续性消去 `u、v` 后留下的可积性条件。

Airy 应力函数是应力法的一种结构化写法。它通过二阶偏导构造三个应力分量，使无体力平衡自动成立。剩下的任务是选择满足双调和方程的函数，并利用边界牵引确定系数。函数的形式通常来自边界形状、载荷多项式阶次、对称性和变量可分离性，而不是凭空猜一条公式。

验证解析解时应沿逆向路径检查：由函数得到应力，由应力得到牵引，确认边界；再由本构得到应变，检查相容；最后确认域内平衡。若只检查函数满足双调和方程，却没有检查边界和相容，得到的只是一个候选场，不是完整解。

逆解法适合构造标准答案或验证数值软件：先选一个已知满足控制方程的场，再计算它所对应的边界载荷。半逆解法更接近工程推导：先根据对称性和边界猜测一部分形式，再用控制方程缩小未知系数，最后进行完整边界检验。两者都要求最后回代，而不能停在“方程形式看起来合理”。
## 第四模块深度讲解：极坐标经典问题

极坐标的核心价值不是公式更短，而是坐标线与边界几何一致。对于圆孔、圆筒和隧洞，径向边界可以直接写成 `r=a` 或 `r=b`，压力也天然表现为径向牵引。若坚持使用笛卡尔坐标，圆边界会变成复杂曲线，边界条件和函数形式都更难处理。

极坐标中，径向位移和环向位移分别记为 `u_r、u_θ`。环向应变包含 `u_r/r`，剪切应变包含位移对 `θ` 的导数和 `-u_θ/r` 项。这些项体现了极坐标基矢随位置旋转的事实，也是把笛卡尔公式直接搬到极坐标时最容易漏掉的部分。

轴对称问题利用几何和载荷对称性：物理量不随 `θ` 变化，许多偏导项消失，径向和环向应力成为 `r` 的函数。厚圆筒内外压问题的 Lamé 解说明，径向应力满足边界压力，而环向应力通常在内壁更危险；工程设计不能只看平均压力。

圆孔应力集中说明“远场应力”和“局部峰值”是两个不同问题。孔边几何改变力流路径，局部切向应力可能远高于远场拉应力。孔距边界足够远时可以使用孤立孔近似；孔距、孔形、载荷方向或边界条件改变后，必须重新建模。

半平面集中力问题是分布荷载叠加的重要基础。把边界分布力分解为 `q(s)ds` 的微元集中力，对每个微元的响应积分，就能构造复杂边界荷载的近似解。这也连接到有限元中的等效结点荷载：两者都要求在选定的位移模式或响应空间中保持虚功等效。

## 第五模块深度讲解：有限元离散的力学含义

有限元离散的第一步不是划网格，而是选定单元内允许出现的位移模式。三节点线性三角形把每个节点的两个平移自由度插值到单元内部，形函数满足本节点为 1、其他节点为 0，并且所有形函数之和为 1。后一个性质保证常量位移可以被精确表示，从而能够表示刚体平移。

形函数对坐标求导形成 `B` 矩阵。对于线性三角形，形函数是一阶函数，导数为常量，因此 `B`、应变和在均匀材料下的应力在单元内都是常量。这带来计算简单和装配高效的优点，也带来局部应力分辨率有限、节点云图需后处理的限制。

单元刚度矩阵的每个条目都有明确的物理解释：第 `j` 个自由度施加单位位移时，在第 `i` 个自由度上产生的结点力就是 `k_ij`。对称性来自互易功关系，正定性则与约束是否消除刚体运动有关。若模型刚体自由度未约束，刚度矩阵会出现零特征值或求解奇异。

结点荷载不能只按静力合力随意分配。对分布力，必须使离散位移模式下的外力虚功与连续分布力一致，这就是一致等效结点荷载。集中力作用在单元内部时，也应按形函数在载荷点的取值分配到各节点，而不是凭距离平均分配。

网格收敛是近似解可信度的核心证据。网格加密并不保证所有局部应力都收敛，尤其是尖角和点载荷奇异性；应选择有明确物理意义的位移、反力、平均应力或应变能作为目标量。若目标量趋于稳定，才能认为离散误差已受控。

## 第六模块深度讲解：从建模到结果验证

有限元建模应按“问题定义、理想化、离散、材料、载荷、约束、求解、后处理、验证”的顺序进行。问题定义决定要输出什么；理想化决定使用梁、壳、平面还是实体；离散决定单元和网格；材料和载荷决定方程中的物理参数；约束决定方程是否有唯一解。

边界条件是最敏感的建模环节之一。固定支座不是“把附近所有节点全部锁死”，而是根据实际连接决定哪些平移和转动自由度被限制。对称约束应只限制法向位移；若把切向位移也锁住，模型会人为变硬。载荷施加在点、线还是面上，也会改变局部应力分布和奇异性。

求解后先检查整体平衡：所有外力与支反力的合力、力矩应在数值误差范围内闭合。然后检查刚体方向是否被约束、变形形状是否符合对称性、位移数量级是否符合手算估计。之后再做网格收敛和局部应力分析。

后处理时要区分四种数据：节点位移、积分点应力、单元平均/常量应力、节点外推或平滑应力。设计判断应尽量使用与规范对应的原始或积分点结果；云图适合展示分布趋势，不适合单独证明局部峰值的真实性。

当结构包含材料界面、接触、尖角或孔口时，结果解释必须包含区域标签。材料界面两侧的应力分量可能不连续；接触压力只在接触区有物理意义；尖角应力可能是数学奇异性；孔口应力需要沿孔边路径提取并结合网格研究。把这些结果全部平滑成一张“漂亮云图”，反而会隐藏建模风险。

最终报告至少应包含模型假设、材料和单位、单元与网格、边界和载荷、求解器状态、反力平衡、网格收敛、关键结果、局限性和下一步验证计划。这样别人才能判断结果是否可复现、可解释、可用于决策。

## 综合案例：同一问题的三层模型

以带孔薄板受拉为例，第一层可用材料力学估算净截面平均应力和整体伸长，快速判断载荷数量级。第二层用弹性力学的孔口解析解描述孔边应力集中，回答“局部峰值大约如何随孔形和载荷方向变化”。第三层用有限元处理有限宽度、多个孔、非规则连接或接触，回答真实几何中的局部响应。

三层模型的结果不必完全相同，但应在共同可比较的量上相互校核：远离孔边的平均应力、整体位移、总反力和对称性。若三层结果差异很大，先检查几何、边界、单位和载荷定义，再判断是否是模型层级差异。有限元不是用来替代低阶模型，而是用来处理低阶模型无法表达的细节。

这个思路也适用于圆筒、梁、隧洞和连接件。先用简单模型建立数量级，再用解析解理解参数影响，最后用有限元处理真实细节。反过来，若有限元结果无法解释，应回到解析解或手算模型寻找原因，而不是继续增加网格和输出图表。

## 课后实践项目

### 项目一：均布拉伸薄板

先用平面应力假设手算平均应力和伸长，再建立二维有限元模型。验收指标是反力与外载荷平衡，远离边界的平均应力接近手算值，网格加密后整体伸长趋于稳定。

### 项目二：圆孔板应力集中

建立不同孔径、孔距和载荷方向的模型，沿孔边提取切向应力。验收指标是远场应力保持合理、孔边峰值随网格变化可解释，并能区分真实应力集中与尖角奇异性。

### 项目三：厚圆筒内外压

分别使用轴对称有限元和二维解析 Lamé 解。验收指标是内外壁径向应力满足压力边界，环向应力分布趋势一致，轴对称模型不出现非物理切向变化。
## 核心术语表

- **体力 `f`**：作用于单位体积的力，如重力。
- **面力/牵引 `t`**：作用于边界单位面积的力，满足 `t=σn`。
- **应力 `σ`**：连续体内部单位面积上的内力表征。
- **应变 `ε`**：位移梯度的对称部分，描述局部变形。
- **相容**：应变能够由单值、连续的位移场产生。
- **Airy 应力函数 `φ`**：用于构造平面应力场的标量函数。
- **形函数 `N`**：把节点自由度插值到单元内部。
- **`B` 矩阵**：把节点位移转换为单元应变。
- **`D` 矩阵**：材料本构矩阵，把应变转换为应力。
- **`k` 与 `K`**：单元刚度矩阵与全局刚度矩阵。
- **积分点应力**：单元内部数值积分位置的原始应力结果。
- **节点外推应力**：由单元结果外推到节点、用于云图显示的结果。
## 课程 FAQ

**问：为什么不能只列平衡方程？** 因为平衡只限制力，不能保证应变来自连续位移，也不能说明材料如何变形；还需要几何、本构和边界条件。

**问：为什么二维模型有时结果差很多？** 平面应力和平面应变代表不同的厚度方向约束，材料矩阵、应力分量和变形模式都不同。

**问：为什么解析解和有限元结果不完全一样？** 解析解常使用无限域和理想边界，有限元还包含有限尺寸、网格离散、边界截断和后处理误差，应比较共同的远场量和整体量。

**问：网格越细是否一定越准确？** 网格细化只降低离散误差，不能修复错误边界、材料或物理假设；尖角和点载荷奇异性也不会消失。

**问：为什么同一节点应力会有多个值？** 相邻单元的积分点应力可能不同，软件再外推或平滑才形成云图；材料界面不应盲目平均。

**问：软件收敛是否代表结果正确？** 数值收敛只说明离散方程求解完成，还必须检查反力、单位、边界、网格、对称性和解析或试验对比。
## 章节依赖关系

基础量和基本假定是所有章节的前置；没有应力、应变、位移的统一符号，后面的平衡和本构容易发生概念混乱。平面问题章节建立了四类方程与边界条件，是解析解和有限元的共同理论底座。

解析解章节依赖平面问题中的平衡、相容和应力函数；极坐标章节进一步要求理解坐标变换和几何项。有限元章节并不是另起炉灶，而是把前面已经学过的位移、应变、本构和平衡关系离散为形函数、矩阵和结点方程。

推荐的知识迁移路径是：

`P1-P5 基础量 -> P6-P19 边值问题 -> P20-P25 解析构造 -> P26-P36 坐标适配 -> P37-P47 矩阵离散`

每次跨模块时，都应把上一模块的核心变量带入下一模块：位移进入几何方程，应变进入本构，应力进入平衡，边界条件进入解析系数或有限元约束，最终结果回到位移、应力和反力的工程解释。

## 平面弹性问题通用算法

1. 画出二维研究域和全部边界，标注坐标、法向、载荷和支承。
2. 根据厚度、长度和约束选择平面应力或平面应变，并写出材料参数。
3. 选择未知量：位移法使用 `u、v`，应力法使用 `σx、σy、τxy`，函数法使用 `φ`。
4. 列出域内方程：平衡、几何、本构和相容中哪些直接使用，哪些用于消元。
5. 在每条边界写出两个标量条件，检查是否与未知量数量匹配。
6. 根据几何和边界选函数形式；先验证控制方程，再解待定系数。
7. 回代得到全部场变量，检查边界、平衡、相容和单位。
8. 识别加载区、孔边、尖角和支座附近的局部效应，不把远场近似外推到奇异区域。
9. 若函数形式难以构造，转为有限元并保留同一套变量和边界定义。

## P1-P5 深度学习笔记：弹性力学的入口

### P1：弹性力学的内容

学习目标是回答“弹性力学到底求什么”。给定一个连续弹性体、外力和约束，主要未知量是域内每一点的位移、应变和应力。位移描述几何位置变化，应变描述局部变形，应力描述内部力传递。三者不是三个独立答案，而是通过几何和本构关系串联。

常见误区是把弹性力学等同于“求应力”。实际设计还要关心位移限值、支反力、接触压力、能量和稳定性；应力只是在材料模型下由位移或应变得到的一类输出。

### P2：体力、面力、内力

体力作用在体积内，重力是典型例子；面力作用在边界面上，压力和摩擦力属于这一类；内力是把连续体切开后相邻部分对截面的作用，局部化后用应力表示。建立方程前必须明确载荷属于域内源项还是边界项，否则会把单位和积分维度写错。

### P3：应力

一点的应力状态不是一个数，而是依赖截面方向的二阶张量。正应力描述法向作用，剪应力描述切向作用；改变截面方向后，同一点上的应力分量会按张量变换。工程上看到的最大主应力、最大剪应力和等效应力，都是对这一应力状态的不同后处理。

### P4：应变与位移

位移是场变量，位移梯度包含刚体运动和真实变形；应变取其对称部分，去掉纯刚体转动。小变形公式只保留一阶项，因此适用于位移梯度和转角都较小的情况。若大转动、屈曲或接触状态变化不可忽略，就要采用几何非线性描述。

### P5：基本假定

连续、均匀、各向同性、线弹性和小变形假定共同决定了后续方程的形式。每个假定都应有适用边界：复合材料需要各向异性本构，塑性变形需要非线性本构，大变形需要更新几何，裂纹和接触需要专门的界面或断裂模型。把这些假定写在分析报告开头，是结果可解释性的基础。
## P6-P12 深度学习笔记：平面问题基本方程

平面应力适用于薄板面内受力，平面应变适用于沿一个方向足够长且该方向变形受到约束的结构。二者差别在于被置零的是厚度方向应力还是厚度方向应变，选择错误会直接改变材料矩阵和应力结果。

对微小矩形列力平衡，得到两个偏微分方程，表达域内任意微元不能产生未平衡合力。体力作为域内源项出现；没有体力时可令源项为零，但不能省略边界牵引。

同一点切取不同方向的微元，会得到不同的正应力和剪应力分量，但它们由同一个应力张量变换而来。应力变换用于转换到结构坐标、材料主轴或规范要求的方向。

小变形几何方程把位移梯度转换为应变。积分应变回位移时出现刚体平移和转动常数，必须由支承、对称条件或位移基准确定；有限元中对应必须约束刚体自由度。

本构方程把应力与应变联系起来。平面应力和应变状态虽然都使用 `E、ν、G`，但消去厚度方向变量的方式不同。写材料矩阵前应确认工程剪应变或张量剪应变约定，否则剪切刚度会出现因子错误。

## P13-P19 深度学习笔记：边界条件与应力函数

位移边界直接给出 `u、v`；应力边界给出边界牵引 `t=σn`。对每条二维边界，都要把向量条件拆成两个标量条件，并根据边界法向方向正确组合正应力和剪应力。自由边不是“没有边界条件”，而是牵引为零的自然边界。

Saint-Venant 原理允许在远离加载区时用静力等效载荷替代复杂端部细节，但不能用于支座、孔口、尖角和载荷施加点的局部应力设计。局部区域若是设计关键，必须保留真实几何和真实接触/载荷分布。

位移法先把应力消去，求解位移型偏微分方程；应力法则把位移消去，求解应力并加入相容方程。选择方法时应看边界条件和几何，而不是习惯性地只用一种方法。

相容方程的物理意义是：应变场积分后必须得到连续、单值的位移。如果一个应力场满足平衡却不能满足相容，它在数学上看似“平衡”，物理上却无法由连续材料实现。

Airy 应力函数通过 `σx=φ,yy`、`σy=φ,xx`、`τxy=-φ,xy` 自动满足无体力平衡。函数仍需满足双调和方程，并用边界条件定出系数。完整流程是：选函数族、检查双调和、求边界牵引、解系数、回代检查应力和位移。

## P20-P25 深度学习笔记：直角坐标解析解

逆解法先给出一个满足控制方程的应力场，再反推出该应力场对应的载荷和边界。它特别适合验证已有公式：如果反算出的边界与目标问题一致，说明场的物理解释成立；如果不一致，只能把它视为另一个问题的解。

半逆解法利用对称性、载荷阶次和边界形状猜测函数的一部分形式，再将其代入双调和或相容方程。猜测不能代替验证，最终仍须检查域内方程、所有边界和位移单值性。

矩形梁纯弯曲案例把二维弹性力学与材料力学连接起来。远离端部时，经典 `σx=-My/I` 能很好描述主弯曲应力；靠近端部、支座或载荷引入区时，横向应力、泊松效应和局部边界层会变得重要。

由应力回求位移要先通过本构得到应变，再对位移梯度积分。积分常数不是随意删除的冗余项，而是刚体平移和刚体转动；若要获得绝对位移，必须给出参考点或位移约束。

简支梁均布荷载的 Airy 函数解展示了完整闭环：根据荷载多项式选择函数，验证双调和，施加简支与自由边条件，求出系数，再回代应力和位移。楔形体案例提醒我们，体力存在时控制方程和应力函数必须包含相应非齐次项。
## P26-P36 深度学习笔记：极坐标与经典解

极坐标平衡方程中的 `1/r` 项来自基矢随位置变化。径向、环向和剪切应力必须使用正确的分量定义；不能只把笛卡尔坐标中的下标替换成 `r、θ`。几何方程同样包含 `u_r/r` 和 `-u_θ/r` 等项。

极坐标下按应力求解仍遵循同一流程：选择应力函数、满足双调和条件、施加径向或环向边界、求出常数、进行坐标变换和验证。坐标变换的目的不是改变物理量，而是把同一应力张量用另一组基底表示。

轴对称问题中所有场量与 `θ` 无关，问题可大幅简化。P31 视频字幕缺失，因此只保留标题和目录信息，不对该课的具体推导做无证据扩写。

圆环和厚圆筒内外压问题的 Lamé 型解具有 `1/r²` 分布，内外壁径向应力由压力边界确定。环向应力通常在内壁更高，设计时应同时检查径向、环向和轴向约束应力。

压力隧洞可视为圆形边界与内压载荷的组合问题；当外边界、衬砌和地层相互作用显著时，孤立圆筒解只能作为初步估算。孔口应力集中需要明确孔形、远场载荷和外边界距离，不能使用固定常数替代完整分析。

半平面集中力问题要求在集中点附近建立局部平衡，半平面分布力则通过集中力微元积分叠加。求位移时要确定远场参考基准，因为二维半平面解常包含刚体平移或相对位移常数。

## P37-P47 深度学习笔记：有限元完整流程

P37 先把位移、应变、应力和结点力排列成列阵，为矩阵运算做准备。P38 将连续体离散为节点和单元，有限元的近似不在于“把图形切碎”，而在于限制每个单元内位移场的函数空间。

P39 的位移模式决定单元能表达什么变形。三节点线性三角形每节点有两个平移自由度，共六个自由度；形函数是一阶函数，满足节点插值性质和分片连续。P40 进一步说明收敛条件：位移模式应包含刚体位移、能表达常应变，并且单元之间保持位移连续。

P41 把位移插值代入几何方程，得到 `ε=Bδ`；再通过材料矩阵得到 `σ=Dε`。P42 的结点力列阵描述单元内部应力对节点的合力和力矩。P43 的单元刚度矩阵 `k` 由内力虚功得到，`k_ij` 的物理含义是第 j 自由度单位位移在第 i 自由度产生的节点力。

P44 处理等效结点荷载。分布力不能只按合力平均分配，必须在既定位移模式下满足虚功等效；内部集中力也要按载荷点处形函数值分配。P45 把所有单元按共享自由度装配成全局结点平衡方程 `KΔ=F`。

P46 是工程实现流程：划分网格、编号节点和单元、输入材料、转换荷载、组装矩阵、施加位移边界、求解结点位移。P47 则负责把位移回代成应变和应力，并整理节点位移、单元应力、反力和云图。

这 11 个分 P 可以压缩为一个可执行算法：选单元 -> 写 `N` -> 求 `B` -> 写 `D` -> 积分得 `k` -> 形成等效荷载 -> 装配 `K` 与 `F` -> 施加约束 -> 求 `Δ` -> 回代 `ε、σ` -> 做平衡、单位和网格收敛检查。

## 三个简化推导案例

### 平面应力单元的材料矩阵

从三维各向同性 Hooke 定律出发，令 `σz=0`，可得到面内应变与应力的关系。将 `εx、εy、γxy` 排成列阵，把 `σx、σy、τxy` 排成列阵，就得到平面应力材料矩阵。这个过程的关键不是背矩阵，而是理解厚度方向应力条件已经被代入；如果问题实际是厚向受限，就应改用平面应变矩阵。

### 厚圆筒的常数求解

设内半径为 `a`、外半径为 `b`，内外压力分别为 `p_i、p_o`。径向应力取 `σr=A-B/r²`，环向应力取 `σθ=A+B/r²`。将 `σr(a)=-p_i`、`σr(b)=-p_o` 代入，可得到关于 `A、B` 的两个线性方程。求出常数后，在 `r=a` 和 `r=b` 比较环向应力，通常内壁是设计控制位置。有限元结果应在相同位置提取并与该解比较。

### 三节点三角形单元

三节点三角形的形函数是坐标的一次函数。已知三个节点坐标后，可由面积坐标构造 `N1、N2、N3`；对它们求导即可得到常量 `B`。若材料和厚度在单元内恒定，`BᵀDB` 也是常量，刚度积分可化为该矩阵乘以单元面积和厚度。这个结果解释了 CST 单元计算快、但只能产生常应变的原因。

## 公式使用前的四项检查

1. 变量是否使用同一坐标系，极坐标分量是否需要转换。
2. 频率、长度、压力、厚度和材料参数是否使用一致单位。
3. 剪应变采用工程定义还是张量定义，矩阵是否与之匹配。
4. 公式的边界条件、对称性和体力假设是否与当前问题一致。

任何公式都只能在它的假设范围内使用；把一个经典解析解直接复制到不同几何或不同边界，往往比没有公式更危险。

## 工程分析报告模板

### 1. 问题定义

写明结构用途、失效判据、载荷工况、设计寿命和需要输出的量。不要只写“进行强度分析”，应说明是峰值应力、平均应力、位移限值、反力还是固有响应。

### 2. 假设与理想化

列出连续体、线弹性、小变形、平面应力/应变、轴对称、忽略接触或忽略局部细节等假设，并说明每一项的依据和潜在影响。

### 3. 方程和边界

给出控制方程、材料参数、坐标系、边界条件、载荷单位和正方向。对有限元模型，附节点自由度约束、压力面、厚度和单元类型。

### 4. 数值设置

记录网格尺寸、局部加密区域、单元质量阈值、积分设置和求解器容差。若采用对称模型，应说明对称边的约束方式及载荷是否按对称面缩放。

### 5. 验证证据

至少报告外载荷与反力平衡、关键位移与手算比较、网格收敛、对称性检查和局部峰值解释。若有解析解或试验，应明确比较位置、变量和误差定义。

### 6. 结论与局限

结论只针对已验证的工况和区域。对于尖角奇异性、未建模接触、材料非线性、疲劳、温度或动态效应，应单独列为局限和后续工作，而不是隐藏在一句“结果可靠”中。

## 一页式速查

**看到薄板**：先问厚度方向应力是否可忽略，可能是平面应力。

**看到长体或隧洞截面**：先问厚度方向应变是否受限，可能是平面应变。

**看到圆孔、圆筒、隧洞**：优先考虑极坐标和轴对称简化。

**看到规则边界与简单载荷**：尝试位移法、应力法或 Airy 函数。

**看到复杂几何或载荷**：转入有限元，但保留同样的平衡、几何、本构和边界检查。

**看到奇异峰值**：先判断是否为尖角、点载荷、接触边缘或材料界面，不要直接把最大云图值当设计值。

**看到求解失败**：先查刚体自由度、重复约束、单位和材料矩阵。

**看到结果异常**：按“单位 -> 载荷 -> 约束 -> 单元 -> 网格 -> 后处理”的顺序排查。

**准备交付报告**：必须提供假设、输入、验证、收敛、结论和局限。
