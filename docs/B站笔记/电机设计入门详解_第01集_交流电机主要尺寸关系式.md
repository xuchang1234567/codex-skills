# 电机设计入门详解：第 1 分 P 交流电机主要尺寸关系式

## 本集主题

本集从电机有效材料和客户给定的技术指标出发，推导交流电机主要尺寸关系式。核心结果是：电枢铁芯直径 `D`、有效铁芯长度 `Lef` 与计算功率、转速、线负荷和气隙磁密之间存在确定关系。

## 1. 电机材料分类

电机材料可以分为：

- 导电材料：定子或转子绕组，承担电流传导；
- 导磁材料：定子、转子铁芯，提供磁通路径；
- 绝缘材料：槽绝缘、层间绝缘和导线漆膜；
- 散热材料：风扇、散热片等；
- 支撑与保护材料：机壳、机座和机械支撑件。

直接参与机电能量转换的是绕组和铁芯，称为有效材料。有效材料尺寸主要靠电磁计算确定；绝缘、散热、支撑和保护材料则需要结合热计算、机械强度和制造工艺确定。[P01@00:00-03:20]

## 2. 设计输入与迭代思想

电机设计通常从客户给出的五类运行规范和五类技术要求出发。运行规范包括额定功率、额定电压、额定转速、频率和相数；技术要求包括效率、功率因数、最大转矩、启动转矩和启动电流。[P01@03:20-05:20]

设计不是一次代入公式就结束。常见流程是：

1. 用已知指标表示未知参数；
2. 预选某些参数的合理范围；
3. 计算其他尺寸和电磁量；
4. 将计算结果与预选值、工程误差和性能要求比较；
5. 不满足时重新调整预选参数，迭代计算。

## 3. 主要尺寸的物理含义

机电能量转换主要通过定、转子之间的气隙完成，因此主要尺寸必须与气隙相关。主要尺寸通常指电枢铁芯直径和有效铁芯长度，有时也把气隙长度作为重要几何参数。

- 交流电机的电枢在定子，电枢直径通常取定子内径；
- 直流电机的电枢在转子，电枢直径通常取转子外径。

本集以交流感应电机为例，后续关系式中的 `D` 应理解为电枢铁芯直径，即定子内径。[P01@05:20-08:00]

## 4. 从电磁功率推导

电磁功率在设计计算中用计算功率 `P'` 表示。相数为 `m` 时：

`P' = m E I`

其中 `E` 和 `I` 是每相电动势与每相电流的有效值或相应相量计算值。

交流电机相电动势基本形式为：

`E = 4.44 kNm f N Kdp Phi`

其中：

- `f`：电源频率；
- `N`：每相串联匝数；
- `Kdp`：绕组系数；
- `Phi`：每极磁通；
- `kNm`：气隙磁场波形系数，正弦分布时与常用 4.44 形式对应。

频率与同步转速、极对数的关系为：

`f = p n / 60`

其中 `p` 为极对数，`n` 为转速，单位为 r/min。

每极磁通可写成气隙磁密与有效面积的乘积。引入计算极弧系数 `alpha_p` 后，平均气隙磁密与气隙磁密最大值 `B_delta` 建立关系：

`B_delta,av = alpha_p B_delta`

每极有效面积与直径 `D`、有效长度 `Lef` 和极距有关。极距近似为：

`tau = pi D / (2p)`

因此磁通表达式中会出现 `D Lef B_delta` 的组合。[P01@08:00-15:30]

## 5. 线负荷

线负荷 `A` 表示电枢圆周单位长度上的安培导体数，单位通常为 A/m。若每相串联匝数为 `N`、相电流为 `I`、相数为 `m`，总导体数与匝数之间还要考虑每匝两根导体，因此可以得到形如：

`A = 2 m N I / (pi D)`

从而可用 `A` 表示相电流：

`I = pi D A / (2 m N)`

这一步的意义是把电流和匝数等局部参数，转换为与电枢圆周和电磁负荷有关的宏观设计参数。[P01@15:30-18:30]

## 6. 主要尺寸关系式

把频率、磁通、线负荷和极距表达式代入 `P' = mEI`，并约去极对数、相数和每相串联匝数等中间量，得到交流电机主要尺寸关系式的结构：

`D^2 Lef n / P' = 6.1 / (alpha_p kNm Kdp A B_delta)`

等价地，主要尺寸乘积可以表示为：

`D^2 Lef = [6.1 P'] / [alpha_p kNm Kdp A B_delta n]`

具体常数和符号应以视频板书、教材定义和单位制为准。它表达的工程含义是：在计算功率和转速确定时，提高允许的线负荷或气隙磁密可以减小所需的主要尺寸，但同时会带来温升、饱和、绝缘、噪声、损耗和制造约束，因此不能单纯追求更高负荷。[P01@18:30-24:00]

## 7. 单位检查

视频强调的主要单位包括：

- `D`、`Lef`：m；
- `n`：r/min；
- `P'`：VA 或按设计定义的计算功率单位；
- `A`：A/m；
- `B_delta`：T；
- `Phi`：Wb；
- `E`：V。

使用公式前必须统一线值/相值、峰值/有效值、机械转速/同步转速和米/毫米。自动字幕对公式符号存在近音识别风险，最终计算应对照视频画面或教材。

## 学习任务

1. 写出交流电机和直流电机电枢直径的不同定义。
2. 解释为什么主要尺寸关系式中出现 `D^2 Lef`。
3. 说明线负荷和气隙磁密提高后，分别可能引起哪些副作用。
4. 给定 `P'`、`n`、`A`、`B_delta` 和若干系数，按统一 SI 单位计算 `D^2 Lef`。
5. 将计算结果与预选尺寸比较，说明下一轮迭代应该调整哪些参数。
