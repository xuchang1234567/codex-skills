## P25：USB 拓展坞元件选型

USB 拓展坞的选型要从接口需求出发：USB 连接器类型、数量、速率、电源能力、上游/下游关系、保护器件和机械结构都要先明确。选型时检查控制芯片供电范围、接口信号定义、ESD 保护要求、外部晶振/电容、状态指示和连接器封装。

USB 相关器件不能只看能否连通，还要检查差分信号、阻抗、走线长度、参考平面、连接器方向和插拔可靠性。电源路径则要考虑端口限流、保险或保护、压降和多个负载同时工作时的电流。

## P26：USB 拓展坞原理图设计

原理图应按功能分块：USB 上游接口、控制/集线器芯片、下游接口、电源、ESD/过流保护、指示灯和调试/配置部分。每个接口的电源、地、差分信号和屏蔽/外壳连接要明确。

设计顺序建议是：先放核心芯片和电源，再连接上游/下游接口，随后加入保护、指示和配置元件。对每个芯片引脚逐项对照数据手册，确认未使用引脚、使能脚、复位脚、上拉/下拉和默认状态。

USB 差分信号不能按普通 GPIO 处理。原理图阶段就要为差分对命名、方向、匹配和布局留出清晰约束；电源和信号地的连接方式也应与芯片和接口推荐设计一致。

## P27：USB 拓展坞 PCB 布局

布局先确定连接器和板框，因为接口位置由机械结构决定；再放核心芯片、晶振、去耦、电源器件和保护器件。USB 连接器到控制芯片之间的差分路径应短、连续、少过孔，避免被电源开关节点或大电流回流切断参考平面。

ESD 保护器件应尽量靠近连接器，使静电能量在进入核心电路前被引导；去耦电容应靠近芯片电源脚；电源入口和限流/保护器件要形成清晰功率路径。

布局还要考虑连接器插拔空间、外壳、安装孔、板边距离、指示灯可见性和板厚。接口位置错误会使电气正确的 PCB 仍然无法装配使用。

## P28：USB 拓展坞 PCB 布线

差分线要按芯片和 PCB 工艺要求控制线宽、间距、长度和参考平面。两条线应保持相近长度和连续环境，避免急剧改变方向、过多过孔、跨分割和靠近噪声源。具体阻抗目标必须依据 USB 速率、器件数据手册和板厂能力确定，不能凭经验写死。

电源线和地回流要有足够宽度，连接器保护回路要短。高频信号、开关电源、晶振和指示灯等区域要避免互相干扰。完成布线后检查差分对规则、网络连通、过孔、铜皮和板边间距。

## P29：DRC 与生产文件

USB 拓展坞导出前，执行 DRC 并处理短路、未连接、线宽、间距、孔径、板框和丝印错误。对差分规则、特殊网络和设计例外要逐项确认，不要简单全部忽略。

生产输出前再次核对：原理图与 PCB 版本、BOM、封装、连接器方向、极性、板框、安装孔和生产层文件。首板测试应包括上电电流、空载/带载 USB 电压、枚举、数据传输、多个端口同时工作、插拔和静电保护验证。

## USB 项目验收清单

- [ ] 上游/下游接口定义清楚。
- [ ] 控制芯片电源、复位、时钟和配置正确。
- [ ] ESD、过流和电源保护路径明确。
- [ ] USB 差分对具有正确网络和规则。
- [ ] 差分线参考平面连续，长度/阻抗符合目标。
- [ ] 连接器位置、方向和机械空间正确。
- [ ] 去耦电容靠近电源脚，保护器件靠近接口。
- [ ] DRC/ERC 错误已解释并处理。
- [ ] Gerber、钻孔、BOM 和装配文件版本一致。
- [ ] 首板测试覆盖枚举、传输、插拔、带载和异常状态。

## 全课程项目对比

51 单片机核心板重点训练最小系统、外围功能、普通数字/模拟信号和完整 PCB 流程；USB 拓展坞进一步训练高速差分、接口保护、连接器机械约束和电源/信号完整性。前者适合作为基础 PCB 项目，后者更适合作为规则和布局能力的提升项目。
