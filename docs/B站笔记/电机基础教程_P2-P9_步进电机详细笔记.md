# 电机基础教程 P2-P9：步进电机与选型计算

## P2-P3：步进电机基础与特性

步进电机通过控制绕组通电顺序，使转子按离散步距运动。它的优势是控制结构直观、低速定位方便、成本相对可控；局限是高速转矩下降、振动噪声、共振和失步风险。

选用步进电机时不能只看保持转矩。实际运动中要看目标转速下的动态转矩、加减速过程、负载变化、驱动器电流、细分和供电电压。保持转矩是静止或特定条件下的指标，不等于高速运行时可用转矩。

## P4：步进电机参数

常见参数包括步距角、相数、额定电流、相电阻、相电感、保持转矩、转子惯量、允许转速、轴尺寸和安装尺寸。驱动器参数还会影响电流建立速度、细分、振动、发热和高速性能。

参数必须结合工况理解：电流影响转矩和发热，电感影响电流变化速度，转子惯量影响加速能力，步距角影响基础分辨率，细分改变控制分辨率但不等于真实定位精度同比提升。

## P5：电机选型流程

标准流程是：明确机构运动方式 -> 确定负载力/转矩 -> 计算速度和加速度 -> 考虑传动效率 -> 折算电机侧需求 -> 核对动态转矩曲线 -> 检查惯量、驱动器、电源和热。

选型应覆盖连续工况、启动、加速、减速、急停和最大负载。安全系数要有依据，不能简单把额定转矩乘一个很大的数字掩盖计算不确定性。

## P6-P8：选型计算

旋转负载要区分静态转矩、加速转矩、摩擦转矩和外部扰动。线性机构要把力通过丝杆、皮带或齿轮折算为电机转矩；同时考虑传动效率、导程、半径、减速比和机械损失。

加速所需转矩与惯量和角加速度有关，负载惯量经过传动比折算到电机侧时会受到传动比平方关系影响。速度计算也要统一单位，避免把线速度、转速、角速度和丝杆导程混用。

## P9：惯量介绍与计算

惯量表示物体抵抗角速度变化的能力。圆柱、盘、轴、齿轮和皮带轮的惯量计算方式不同；多个部件需要按传动关系折算到电机轴。惯量匹配影响加速响应、跟随误差、振动和失步风险。

步进电机通常不具备闭环反馈，实际加速曲线必须留出转矩裕量。若负载惯量过大，不能只提高脉冲频率，应从减速比、加速度、机构摩擦、驱动电流和电机规格一起调整。

## 选型验收清单

- [ ] 已明确负载、行程、速度和加速度。
- [ ] 已分别计算静态、摩擦和加速转矩。
- [ ] 已考虑丝杆/皮带/齿轮效率。
- [ ] 已把负载惯量折算到电机侧。
- [ ] 已查看目标转速下的动态转矩，而非只看保持转矩。
- [ ] 已核对驱动器电流、电源电压、细分和散热。
- [ ] 已验证启动、加速、最大负载和急停工况。
