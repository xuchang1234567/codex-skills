# 模态分析精讲：从固有频率到工程仿真

## 学完你应该获得什么

- 理解“模态”不是一张变形云图，而是结构的固有频率与对应振型。
- 能从无阻尼动力学方程解释特征值、固有频率和特征向量的关系。
- 知道自由模态、约束模态和预应力模态分别代表什么工况。
- 能用有效质量、刚体模态和低阶频率判断结果是否足够可信。
- 能把模态分析结果用于避开共振，并作为谐响应、响应谱、随机振动等分析的基础。

## 一句话总论

模态分析是在理想化边界和材料条件下求结构的特征值问题：固有频率由质量矩阵与刚度矩阵共同决定，特征向量给出振型；工程重点不是“变形了多少”，而是判断哪些频率和振型会在真实激励下被有效激发。[1][2]

## 模态是什么

结构的广义动力学方程可写为 `M a + C v + K x = F`。若先忽略阻尼和外部激励，得到 `M a + K x = 0`。假设结构以正弦形式振动，将位移写成某个振型乘以时间函数，就会得到广义特征值问题：

`K phi = omega^2 M phi`

其中 `omega^2` 是特征值，`omega` 是圆固有频率，`f = omega/(2 pi)` 是普通频率，`phi` 是特征向量，即振型。[1][3]

因此，一阶、二阶模态不是“第几张图片”，而是按频率排序的一组独立振动模式。每个模态具有固有频率、振型，若考虑耗能还具有阻尼特征。振型用于判断结构的相对刚弱和变形模式，但软件显示的振型幅值通常经过归一化，数值大小没有直接物理意义；应观察形状、节点和相对运动。[4]

## 为什么必须做模态分析

模态分析主要解决三个问题：

1. 找到结构容易发生共振的频率区间，并与发动机、转子、脉动载荷、路谱或设备激励频率避开。
2. 通过振型判断刚度薄弱位置、连接相对运动和结构优化方向。
3. 为谐响应、响应谱、随机振动和瞬态动力学提供模态基底或频率范围。[5]

低阶模态通常更容易被实际基座或外部激励激发，因此工程上先关注低阶频率和对应有效质量；高阶模态并非没有意义，但要看激励频带、局部响应和目标量。

## 三类模态与物理含义

### 自由模态

自由模态不施加支撑约束，结构在空间中自由振动。三维刚体结构理论上有六个刚体自由度，因此前六阶频率应为零或接近零；实验中常用柔软悬挂和锤击法近似实现自由边界。[6]

自由模态适合研究结构本体特性，但不是大多数设备的真实工况。若模型存在未连接的零件、错误接触或漏约束，也会出现额外的零频模态，应先排查连接和刚体运动。

### 约束模态

约束模态在指定支撑、固定面或连接约束下求解。约束改变了结构的可运动自由度，主要体现为刚度矩阵的变化，质量矩阵通常不因单纯固定边界而改变。[7]

约束必须代表真实装配。过约束会人为抬高刚度与频率，欠约束则产生虚假的刚体模态；连接关系中的相对转动或滑移释放也会改变刚度矩阵，从而改变整个频率谱。[8]

### 预应力模态

预应力模态先求一个静态或稳态载荷状态，再在线性化后的状态附近求模态。直观例子是琴弦：绷紧后横向等效刚度增加，固有频率升高；压缩、拉伸、离心力或接触状态都可能改变切线刚度。[9]

在 Workbench 类流程中，通常先建立静态结构分析，施加预紧力或工作载荷，再把该状态传递给模态系统。预应力模态结果要确认载荷步、接触状态和大变形设置确实被传递；若预应力引起明显几何非线性，不能只把结果当作普通线性模态。[10]

## 结果如何判断

### 有效质量与参与系数

模态振型是否能代表目标方向的整体运动，要看参与系数和有效质量。有效质量与总质量之比反映该模态集合在某个平移或转动方向上捕捉了多少惯性贡献。若关注 Y 向激励，却只计算到累计有效质量约 0.6，就应继续增加模态阶数；常见工程目标是让相关方向的累计比例达到项目要求，而不是机械地只取固定阶数。[11]

### 刚体模态与频率排序

自由模态前六阶接近零是合理现象；约束模态则不应保留这些刚体运动。看到非预期的零频或极低频，先检查约束、接触、共节点和零件是否真正连接。低阶频率也应与手算量级、试验或相似结构经验对比。

### 振型图怎么读

振型显示的是归一化后的相对位移。重点看节点线、反节点、零件之间的相对运动、局部柔弱区和连接处是否出现非物理穿透。不要把动画幅值当成实际振幅，也不要用不同模态的云图颜色直接比较响应大小。[4]

## 一套可复用的仿真流程

1. **定义工况。** 明确是自由、约束还是预应力模态，确定目标激励方向和频带。
2. **检查几何与连接。** 合并或共享正确节点，核对接触、连接释放和零件抑制状态。
3. **定义材料。** 模态至少需要密度和弹性模量；通常还需要泊松比。缺少质量或刚度参数无法形成有效的 `M`、`K`。[12]
4. **生成网格。** 在连接、薄弱区和几何突变处检查网格质量；用网格收敛而非云图平滑程度判断可靠性。
5. **设置求解阶数。** 初步提取足够多阶次，按低阶频率、目标方向有效质量和激励频带筛选。
6. **求解并检查。** 检查零频模态、频率量级、参与质量、振型连续性和连接处运动。
7. **传递到后续分析。** 用选定模态开展谐响应、响应谱、随机振动或瞬态分析，同时保持频带和边界假设一致。

## 阻尼与模态分析

基础模态通常先按无阻尼特征值问题求解。若需要考虑阻尼，可以设置全局阻尼、常阻尼比，或通过质量、刚度比例系数形成等效阻尼。此时特征值可能为复数，实部和虚部的解释与无阻尼频率不同；应依据求解器理论和阻尼模型判断稳定性，不要把复特征值直接当作普通 `omega^2`。[13]

## 常见误区

- 把振型图上的最大变形数值当成实际位移。
- 只计算少量阶次，却没有用有效质量确认覆盖了目标方向。
- 为了“让结果好看”随意增加固定约束，导致频率虚高。
- 忽略零件连接、接触和自由度释放，错误改变刚度矩阵。
- 把自由模态当作设备工作状态，或把预应力结果当作普通模态。
- 只看频率，不看振型、有效质量和激励方向。

## 自由度与建模排错

模态结果首先受自由度模型支配。实体单元通常以节点平移自由度表达变形，梁、壳单元还可能包含转角；连接、接触和远程约束会改变哪些自由度可以相对运动。一个频率异常时，应按“几何是否完整 -> 材料是否有密度与弹性参数 -> 零件是否真正连接 -> 约束是否过多或过少 -> 网格是否足够”的顺序排查，而不是先修改求解器选项。

模型验证至少包括三类对比：与简化单自由度或梁模型比较频率量级；与试验、历史产品或设计经验比较低阶振型；在关键区域加密网格，观察频率和目标响应是否趋于稳定。若频率稳定但振型发生突变，常提示连接、局部刚度或模态排序发生了变化。对于预应力模型，还要单独比较预应力前后的刚度变化，区分真实物理效应与边界设置错误。

最终报告应同时记录模型假设、约束、材料、网格、提取阶数、有效质量和验证对比，保证结果可复现、可解释。

## 自测题

1. 模态分析求解的核心数学问题是什么？答：广义特征值问题 `K phi = omega^2 M phi`。
2. 固有频率主要由什么决定？答：结构的质量矩阵与刚度矩阵。
3. 自由三维结构为何有六个接近零的模态？答：对应三个平移和三个转动刚体自由度。
4. 约束会优先改变什么？答：可运动自由度和刚度矩阵。
5. 预应力为什么能改变频率？答：它改变了线性化状态附近的切线刚度。
6. 为什么不能比较不同振型的动画幅值？答：振型通常被归一化，显示幅值不是实际响应幅值。
7. 有效质量不足时怎么办？答：增加提取阶数，直到目标方向达到项目规定的质量参与比例。
