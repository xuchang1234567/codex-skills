# 电机基础教程 P10-P17：伺服电机与机构选型

## P10-P11：伺服电机与执行电机对比

伺服系统通常包含电机、编码器、驱动器和控制器，通过反馈实现位置、速度或转矩控制。它的优势是响应快、精度高、能检测误差并补偿；成本、参数配置、调试复杂度和系统匹配要求也更高。

步进、伺服和普通驱动电机不能只按“精度高低”简单排序。选择取决于是否需要反馈、动态响应、负载变化、运行速度、定位要求、成本和故障后果。开环步进适合部分低速定位，伺服适合高动态或高可靠场景，普通驱动电机适合连续旋转和成本敏感应用。

## P12：伺服参数

选型和调试要关注额定/峰值转矩、额定/最高转速、编码器分辨率、惯量、过载能力、额定功率、制动、驱动器电源、控制接口和安装尺寸。峰值转矩用于短时加速或扰动，不能当作连续运行能力。

编码器反馈质量、机械刚度、联轴器间隙和负载惯量都会影响最终定位效果。电机参数、驱动器参数和控制器参数必须形成匹配，不能只更换电机而忽略驱动器和机械部分。

## P13-P14：控制方式与 PID

常见伺服控制方式包括位置控制、速度控制和转矩控制。位置控制关注目标位置与实际位置误差，速度控制关注速度跟踪，转矩控制直接调节输出力矩。实际系统可能采用位置环、速度环和电流/转矩环的级联结构。

PID 通过比例、积分、微分处理误差。比例提高响应，积分消除稳态误差，微分抑制快速变化但会放大噪声。调试时必须考虑采样周期、反馈滤波、输出限幅、积分饱和、机械共振和负载变化。

## P14-P17：惯量匹配与机构选型

负载惯量需要通过丝杆、齿轮、皮带等传动关系折算到电机轴。惯量比过大时，系统加速困难、响应变慢、跟随误差和振动增加；惯量比过小时也不一定自动最好，还要看刚度、带宽、减速比、负载变化和控制参数。

### 丝杆机构

丝杆把旋转运动转换为直线运动。选型需要根据轴向负载、导程、速度、加速度、效率、支撑方式、定位精度和自锁条件计算。电机转速与直线速度由导程关联，转矩与轴向力和效率相关。

### 齿轮齿条机构

齿轮齿条适合较长行程和连续直线运动。需要考虑齿轮节圆半径、线速度、加速度、齿面力、传动间隙、啮合精度和润滑。齿条运动中的冲击和反向间隙会直接影响伺服定位。

### 综合选型

选型应同时满足连续转矩、峰值转矩、速度、功率、惯量和热。还要核对驱动器容量、反馈接口、制动需求、急停和机械安全。计算得到的最小规格不能直接作为最终型号，需结合动态曲线和安全裕量复核。

## 选型流程

1. 明确运动轨迹和定位/速度要求。
2. 计算负载力、摩擦、加速和扰动转矩。
3. 通过机构参数换算电机速度和转矩。
4. 折算负载惯量并检查惯量匹配。
5. 选择电机、驱动器、编码器和制动方案。
6. 用速度-转矩曲线验证全工况。
7. 调整 PID 和机械刚度，验证超调、跟随误差和温升。
