# 模态是什么：从微分方程特征根到系统稳定性

## 学完你应该获得什么

- 能用响应的时间走势解释“模态”而不是死记名词。
- 理解微分方程特征根与传递函数极点为什么决定模态。
- 能处理不同特征根：非重根、重根和共轭复根。
- 能从复平面极点位置判断衰减、振荡和不稳定发散。
- 能区分极点决定的动态模式与零点/分子决定的模态权重。

## 一句话总论

模态是系统输出响应中决定长期走势的基本动态项；对于微分方程，它由特征根决定；对于传递函数，它由极点决定。极点实部决定衰减还是发散，虚部决定振荡频率，分子只改变各模态在响应中的权重。[1][2]

## 从一个一阶例子理解模态

考虑一个线性微分方程，经拉普拉斯变换并整理后得到 `X(s) = X0 / (s + 0.5)`。反变换为 `x(t) = X0 e^(-0.5t)`。其中 `X0` 是由初始条件决定的常数，只改变响应大小；指数项 `e^(-0.5t)`决定响应如何随时间衰减，因此它就是模态。[3]

这给出一个实用判据：先把响应写成时间域中的基本项，忽略纯常数系数，观察会随时间长期保留、衰减、振荡或增长的部分。不要把“幅值大”与“模态本身”混为一谈。

## 特征根如何出现

当系统模型是微分方程时，对零初始条件下的方程做拉普拉斯变换，会得到关于 `s` 的代数方程。将分母置零得到特征方程，其根就是特征根。若响应的分母是 `(s+3)(s+7)`，反变换后会出现 `A1 e^(-3t) + A2 e^(-7t)`。两个指数项就是两个模态，而 `A1`、`A2`只决定各自比重。[4][5]

因此，同一组特征根可以出现在不同初始条件或不同输入下；分子改变，响应权重改变，但系统允许的基本走势没有改变。[6]

## 三种特征根情况

### 非重实根

若特征根互不相同，响应由若干 `e^(lambda_i t)` 叠加而成。实部为负时该项衰减，实部为正时该项增长；绝对值较大的负实部衰减更快，长期响应通常由衰减最慢的模态主导。[7]

### 重根

若 `(s+a)` 出现重根，反变换不只是 `e^(-at)`，还会出现 `t e^(-at)`、`t^2 e^(-at)` 等多项式乘指数项。重根阶数越高，前面的 `t` 次数越高。例如三重根会产生最高到 `t^2 e^(-at)` 的项。[8]

这说明“模态只看指数”在重根情形不完整：多项式因子也会影响过渡过程的走势，但最终稳定性仍由指数的实部控制。

### 共轭复根

当特征根为 `lambda = -alpha +/- j omega` 时，时域响应包含 `e^(-alpha t) sin(omega t)` 或 `e^(-alpha t) cos(omega t)`。`alpha`决定包络衰减，`omega`决定振荡角频率，分子决定正弦与余弦的组合和相位。[9]

所以共轭复根对应的是衰减振荡或发散振荡。机械系统的固有振动、控制系统的欠阻尼响应，通常都可以用这种模式理解。

## 从传递函数看模态

传递函数 `G(s)=X(s)/R(s)` 的极点是分母为零的位置。把微分方程变换后，分母可以因式分解为 `K(s-p1)(s-p2)...`，这些 `p_i`正是特征根；将输入项移到另一侧得到的商就是传递函数，因此二者描述的是同一组系统动态。[10][11]

传递函数的**极点**决定模态，**零点**和分子主要决定各模态的权重、相位和响应形状。一个模态即使存在，若输入和输出配置使其权重接近零，实际响应中也可能几乎看不到它；这不代表该模态从系统中消失。[12]

脉冲输入是理解这一点的方便工具：输入 `r(t)=delta(t)` 的拉普拉斯变换为 1，输出的拉普拉斯变换直接等于传递函数；再反变换即可观察系统全部模态的组合。[13]

## 极点位置与稳定性

把极点画在复平面上：

- 左半平面：实部为负，模态含衰减指数，响应最终收敛。
- 右半平面：实部为正，模态含增长指数，响应发散，系统不稳定。
- 虚轴附近：衰减很慢，系统对扰动和噪声更敏感；纯虚轴还可能形成不衰减振荡。

共轭复极点在左半平面时表现为逐渐减弱的振荡，在右半平面时表现为振幅不断增大的振荡。[14][15]

这是为什么工程上常用极点判断稳定性：不需要先画完整响应，只要检查极点实部符号，就能判断长期趋势。对于重复极点，还要注意多项式乘指数可能放大过渡过程，即使指数最终衰减，也可能产生较长或较大的瞬态。

## 可复用的分析流程

1. 写出系统微分方程或传递函数。
2. 对微分方程做拉普拉斯变换，整理出 `X(s)` 或 `G(s)`。
3. 因式分解分母，得到特征根/极点；不要被分子系数分散注意力。
4. 按非重根、重根、复根选择对应的反变换形式。
5. 读取极点实部和虚部，判断衰减速度、振荡频率和稳定性。
6. 再看分子、零点和初始条件，判断每个模态在具体输入输出中的权重。

## 常见误区

- 把模态当作某一张动画图：这里讨论的是系统响应的动态基本项。
- 认为分子决定系统模态：分子主要改变权重，极点决定基本走势。
- 忽略重根前面的 `t`、`t^2` 因子。
- 只看极点虚部，不看实部：虚部给振荡，实部决定稳定性。
- 看到输出里没有某个模态就断言系统没有该模态；它可能只是被输入/输出配置抑制。

## 一个快速判读模板

拿到任意传递函数时，可以先完全忽略分子，做三步：第一，把分母因式分解并列出全部极点；第二，按极点的实部、虚部和重数写出对应的指数、正弦/余弦或时间多项式；第三，再回头看分子，判断哪些模态被放大、削弱或发生相位变化。这样既能快速判断稳定性，也能避免把响应幅值误当成系统固有特性。

例如，极点为 `-2` 和 `-5` 时，响应由两个衰减指数叠加，长期走势主要由 `e^(-2t)`控制；极点为 `-1 +/- j3` 时，响应是包络为 `e^(-t)`、角频率为 3 的衰减振荡；极点为 `+1` 时，即使分子很小，只要该模态没有被精确抵消，系统仍包含发散趋势。工程上所谓“快模态”和“慢模态”，本质就是不同实部大小带来的衰减时间尺度差异。

对于控制或结构振动问题，这个模板可以连接到更复杂的矩阵模型：多自由度系统的矩阵特征值提供一组极点/模态，模态参与系数相当于输入与输出对这些基本模式的权重。先掌握单输入单输出的极点直觉，再迁移到矩阵特征值问题，会比直接背公式更稳。

还要注意模型边界：极点判断反映的是线性、时不变模型的长期性质；非线性、时变参数、饱和、摩擦和接触可能使固定极点的解释失效。遇到这类问题，应先说明线性化工作点，再讨论局部模态和稳定性。

读图时也要区分极点图与响应图：极点图描述系统允许的动态模式，响应图还叠加了输入、初始条件和零点的影响。两者不是同一张信息的不同画法。

因此，模态分析的第一问应始终是“分母是什么”，第二问才是“分子如何激发它”。

这个顺序也适用于阅读仿真软件的频响曲线：先定位共振峰对应的极点，再追溯载荷方向和测点位置。

这样既能解释现象，也能避免把偶然的响应峰误判成新的系统模态。

## 自测题

1. 模态由什么决定？答：微分方程的特征根或传递函数的极点。
2. `X0` 会改变一阶响应的模态吗？答：不会，它只改变幅值。
3. 重根为什么会出现 `t e^(-at)`？答：重复极点的拉普拉斯反变换包含多项式时间因子。
4. 共轭复根的实部和虚部分别表示什么？答：实部决定衰减/发散，虚部决定振荡频率。
5. 极点在右半平面意味着什么？答：至少有一个发散模态，系统不稳定。
