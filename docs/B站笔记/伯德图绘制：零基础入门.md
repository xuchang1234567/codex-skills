# 伯德图绘制：零基础入门

> 覆盖说明：本笔记基于视频字幕、公式和元数据整理。视频大量步骤依赖板书/绘图画面；本次未进行关键帧视觉复核，因此图线位置、手写示意和个别数值以字幕为准，适合作为方法导览，不替代逐帧看图。

## 学完你应该获得什么

- 知道伯德图的横纵轴、单位和对数刻度含义。
- 能把开环传递函数拆成低频段、一阶环节和二阶环节。
- 能求转折频率、低频渐近线和各段斜率。
- 能用“分子加、分母减”的规则叠加幅频渐近线。
- 能识别二阶环节的共振鼓包与阻尼修正。
- 能用几何法或零分贝交点估算截止频率。

## 一句话总论

伯德图不是重新求一个复杂函数，而是把传递函数拆成典型环节，在对数频率轴上分别画出幅值渐近线，再将各环节贡献相加；最后仅对阻尼较小的二阶环节做真实曲线修正。[1][2]

## 1. 伯德图的基本对象

对开环传递函数 `G(s)`，令 `s = j omega` 得到频率特性 `G(j omega)`。幅值图使用

`L(omega) = 20 log10 |G(j omega)|`

单位为 dB；横轴是 `omega` 的对数刻度，常标为 `0.1、1、10、100`。对数轴意味着十倍频率间距相等，数值 5 不在 1 和 10 的正中间，而位于更靠近 10 的位置，因为 `log10(5)=0.699`。[3][4]

视频把任意传递函数看成三类“积木”：低频幂次项、一阶环节和二阶环节。完整幅频图等于各积木的 dB 曲线叠加。[2]

## 2. 先把传递函数写成标准型

### 低频段

低频因子写成 `K / s^v`。代入 `s=j omega` 后，幅值 dB 为

`20 log10 K - 20 v log10 omega`。

因此它在对数频率图上是一条直线，斜率为 `-20v dB/dec`，并且在 `omega=1` 处通过 `20log10K`。低频幂次项本身没有转折频率。[5][6]

### 一阶环节

统一写成 `1 + sT` 或 `1 + s/omega_b`。转折频率为 `omega_b=1/T`。分子中的一阶零点在转折后使斜率增加 `+20 dB/dec`；分母中的一阶极点使斜率减少 `-20 dB/dec`。[7]

### 二阶环节

统一写成

`s^2 + 2 zeta omega_n s + omega_n^2`

或除以 `omega_n^2` 的无量纲形式。`omega_n` 是固有频率，`zeta` 是阻尼比。二阶极点在转折后贡献 `-40 dB/dec`，二阶零点贡献 `+40 dB/dec`。[8]

## 3. 五步画幅频渐近线

1. **标准化。** 把每个因子改写为低频幂次、一阶标准型或二阶标准型，并保留总增益 `K`。
2. **列转折频率。** 只有一阶、二阶因子有转折频率；按从小到大排列。
3. **画低频基准线。** 先标 `omega=1` 时的 `20log10K`，再按 `-20v` 的斜率向两侧延伸。[5]
4. **逐个叠加。** 经过零点转折，斜率加 20 或 40；经过极点转折，斜率减 20 或 40。分子是“加”，分母是“减”。每经过一个转折，更新当前斜率。[9][10]
5. **修正二阶真实曲线。** 渐近线只适合快速绘图；二阶环节还要根据阻尼比判断是否出现共振鼓包。[11]

## 4. 二阶阻尼修正

当 `zeta < 1/sqrt(2)≈0.707` 时，真实幅频曲线会在转折附近鼓起；阻尼越小，峰值越明显。视频示例中 `omega_n=100、zeta=0.8`，曲线贴近渐近线；另一个例子中 `omega_n=8、zeta=0.2`，需要画出共振鼓包。[11][12]

对二阶环节，峰值修正常写成与 `20log10(1/(2zeta))` 相关的 dB 量；实际计算时要确认所用传递函数是极点形式、零点形式还是其倒数，避免符号弄反。[11]

## 5. 截止频率的估算

截止频率 `omega_c` 定义为幅值等于 1，即 `L(omega_c)=0 dB`。有两种实用方法：

- **几何法。** 在某一段渐近线上取两点 `(log omega_1,L_1)`、`(log omega_2,L_2)`，斜率为 `(L_1-L_2)/(log omega_1-log omega_2)`；再用相似三角形求与 0 dB 的交点。纵向高度等于横向对数距离乘以斜率绝对值。[13][14]
- **低频交点法。** 若低频段为 `K/omega^v` 且第一个转折前仍适用，令低频渐近线与 0 dB 相交，可得 `K=omega_0^v`。当 `v=1` 时，低频零分贝交点直接给出 `omega_0=K`；但必须检查交点是否确实位于第一转折频率之前。[15]

## 例题式流程

对类似 `G(s)=0.75(1+s/5)/(s(1+s/10)^2)` 的形式：低频 `K=0.75、v=1`，初始斜率为 `-20 dB/dec`；在 `omega=5` 处经过一个分子一阶零点，斜率变为 0；在 `omega=10` 处经过分母二阶极点，斜率再减 40，变为 `-40 dB/dec`。先画渐近线，再根据二阶阻尼比修正。[10][12]

## 常见误区与检查清单

- 把线性频率轴当普通等距轴：伯德横轴是 `log omega`。
- 忘记分子/分母符号：零点加斜率，极点减斜率。
- 给低频幂次项强行设置转折频率：转折只来自 `1+sT` 或二阶标准因子。
- 直接把渐近线当真实曲线：二阶低阻尼时峰值会明显偏离。
- 求 `omega_c` 前不检查所在斜率段：几何法必须使用交点所在的那一段直线。
- 频率单位混用：统一使用 rad/s 或 Hz，并在需要时换算。

## 一页纸操作模板

拿到题目先写：`G(s)=K/s^v × 一阶因子 × 二阶因子`。然后在草稿上列三行：转折频率从小到大、初始斜率 `-20v`、每个转折后的斜率变化。横轴先标十倍刻度，纵轴标 dB；先画经过 `omega=1`、`20logK` 的低频线，再逐个跨过转折点更新斜率。最后只检查二阶阻尼、0 dB 交点和单位。若题目要求精确曲线，必须回到二阶幅值公式计算修正，而不能只延长渐近线。[9][11]

本次未抽取关键帧，因此无法验证视频板书中曲线的具体像素位置、手绘拐点和所有中间算术；这些应以原视频画面为准。
建议练习时逐步复算每个转折点。

## 自测题

1. `K/s^2` 的低频斜率是多少？答：`-40 dB/dec`。
2. 分母一阶极点经过转折频率后改变多少？答：斜率减少 20 dB/dec。
3. 为什么 5 在对数轴上更靠近 10？答：位置由 `log10(omega)` 决定。
4. 阻尼比小于约 0.707 会出现什么？答：二阶真实曲线可能出现共振鼓包。
5. 截止频率对应的 dB 值是多少？答：0 dB。
