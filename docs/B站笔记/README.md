## 目录约定

- 根目录：面向阅读的课程总览、专题笔记、复习卡片和实训模板。
- `原始材料/`：按 BVID 保存的来源说明、字幕全文、证据索引和评论材料。
- `tmp_*`：历史提取流程生成的 Markdown 报告，仅用于追溯，不作为学习入口。

## 使用建议

长课程先读“导航”或“总笔记”，再按模块进入详细笔记；涉及工程选型时，必须同时查看原始材料中的证据索引和厂商数据手册。课程字幕、经验参数和评论不能替代实际测量、安全评审或正式设计计算。
