# 陷波滤波器：从共振抑制到控制系统应用

## 一句话总论

陷波滤波器在目标频率附近形成很深的衰减带，用来压制结构共振、机械振动或电源干扰，同时尽量保留其他频率的信号。它不是简单的低通滤波器：低通会整体削弱高频，陷波则只针对一个窄频段。

## 基本结构

典型二阶陷波器同时包含一对零点和一对极点。零点决定目标频率附近的衰减深度，极点决定带宽和过渡形状；阻尼参数越小，陷波越尖锐，对频率漂移越敏感。实际设计时要区分中心频率、陷波深度、带宽、品质因数 Q 和相位变化，不能只看幅频曲线最低点。

## 为什么能抑制共振

机械结构在固有频率附近会放大输入，控制器或传感器若在该频率注入能量，闭环可能出现明显振荡。把陷波器中心频率对准共振峰，可在环路增益中削弱该频率的能量传递，降低峰值响应。代价是中心频率附近的相位会发生变化，交越频率若靠近陷波带，系统稳定裕度可能下降。

## 参数调整流程

先通过扫频、模态分析或实验确定真实共振频率；再选择中心频率和带宽，初始带宽只覆盖需要抑制的峰值；随后检查陷波深度是否足够，避免过深导致数值噪声和相位突变；最后重新评估伯德图、增益裕度、相位裕度、闭环带宽和时域响应。结构频率随载荷、温度或装配变化时，应留出频率偏差裕度，不能把中心频率设置得过于理想化。

## 常见误区

陷波器不能修复错误的结构刚度、传感器安装或控制器符号；不能用一个滤波器掩盖多个相距很远的共振峰；不能只看开环幅值下降而忽略相位损失；不能在采样频率不足时直接部署高频陷波器。实现时还要检查离散化、系数量化、计算延迟和初始化瞬态。
