## Description

疲劳失效是一种由于在重复的循环应力加载下产生和扩展裂纹而导致断裂的失效机制。它可能导致材料屈服强度或极限强度远低于应力水平的失效。在这个视频中，我讨论了断裂的机制，以及如何使用S-N曲线来预测零件的疲劳寿命。我还讨论了拉伸平均应力的影响，以及我们如何使用Rainflow计数方法和Miner准则来考虑复杂的循环应力。
