## Description

我们将介绍将被分析的物体分割成小元素以及可以使用的不同元素类型的原因。我们还将介绍有限元法的关键概念，即刚度矩阵，包括如何推导元素刚度矩阵，使用加勒金方法和加权残差法等技术。一旦我们获得了元素刚度矩阵，我们就可以将它们组装成一个描述在一组施加载荷和边界条件下物体如何位移的全局刚度矩阵。
