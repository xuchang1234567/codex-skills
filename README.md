# Codex Skills

This repository is a versioned backup of the Codex skills installed in this directory.

## Contents

- `.system/`: system-provided Codex skills.

The skills retain their included license files. The Apache License 2.0 terms in
each bundled skill apply to that skill's contents.

## Notes

- Do not commit API keys, tokens, private keys, or local configuration files.
- This repository tracks the installed files as a backup. Codex may update the
  `.system` directory during upgrades.
- To use a skill in Codex, keep its folder under the local Codex skills
  directory and restart or refresh Codex after changes.
